# Lesson 053 — 동사 carry — 들고·넣고·메고 "가지고 다니다"

> 📺 **원본 영상**: [[ 생활 기초 영어어휘 ] carry - 가지고 다니다 / 들고 다니다 / 넣고 다니다](https://www.youtube.com/watch?v=UGgpU3heTXc)
> ⏱ 2분 49초 · 👁 95K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **carry** = 몸에 소지하고 이동하는 모든 것 — 손에 들고, 주머니에 넣고, 가방에 메고
> 2. `hold`(쥐고 있다)와 구별: 이동이 시작되는 순간 `carry`
> 3. **I usually carry my phone in my pocket** — "어디에 넣고 다니다" 표현 방식
> 4. 목적어 + 장소 순서: `carry + 물건 + in/on + 장소`

---

## ✨ 상황 1 — 평소에 뭘 어떻게 가지고 다니나

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I usually carry my phone in my pocket.** | **저는 보통 주머니에 핸드폰을 넣고 다녀요.** | **← 오늘 #1** |
| 2 | 🆕 **Sometimes I carry it in my hand.** | **손에 들고 다닐 때도 있어요.** | **← 오늘 #1 (대명사 it 활용)** |
| 3 | 🆕 **When I go out, I usually carry a backpack.** | **나갈 때는 보통 가방을 메고 다녀요.** | **← 오늘 #1** |

---

## ✨ 상황 2 — 이유까지 이어서 말하기

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | Because I don't like to carry many things in my pocket | 주머니에 물건들을 넣고 다니는 걸 좋아하지 않아서 | 도입 |
| 2 | 🆕 **So I carry everything I need in my backpack.** | **그래서 필요한 모든 것을 가방 안에 넣고 다녀요.** | **← 오늘 #1** |

### 🧩 문맥으로 이어 말하기
> **When I go out, I usually carry a backpack because I don't like to carry many things in my pocket, so I carry everything I need in my backpack.**
> (나갈 때는 주로 가방을 메고 다녀요. 주머니에 물건 많이 넣고 다니는 걸 안 좋아해서, 필요한 건 다 가방에 넣고 다녀요.)

---

## 🔑 오늘의 패턴 정리

```
[carry 기본 패턴]
carry + 물건 + in my pocket   →  주머니에 넣고 다니다
carry + 물건 + in my hand     →  손에 들고 다니다
carry + 물건 (가방)           →  (가방을) 메고 다니다

[응용]
I don't like to carry things in my pocket.
→  "carry + 물건 + 장소" 구조 그대로 to부정사 안에
I carry everything I need in my backpack.
→  everything I need = 내가 필요한 모든 것 (관계절)
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **carry** | v. | 가지고 다니다, 휴대하다 | 이동을 포함한 소지 → `hold`(그냥 쥠)와 구별 |
| **pocket** | n. | 주머니 | `in my pocket` |
| **backpack** | n. | 백팩, 가방 | `carry a backpack` = 가방을 메고 다니다 |
| **everything I need** | 표현 | 내가 필요한 모든 것 | `everything (that) I need` |

## 📝 파생 연습 (4문제)

1. **영작**: "저는 보통 지갑을 주머니에 넣고 다녀요."
2. **영작**: "그녀는 항상 우산을 가지고 다녀요."
3. **변환**: `I have my keys in my bag.` → carry 동사로 바꿔 말하기
4. **조합**: "주머니에 물건 많이 넣고 다니는 걸 안 좋아해서 가방을 들고 다녀요." 한 문장으로.

```

2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요. 그 후엔 제 질문에 답하면서 추가 예문/유사 표현을 계속 만들어 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 동사 carry — 들고·넣고·메고 "가지고 다니다"
- 오늘의 핵심:
  1. carry = 몸에 소지하고 이동하는 모든 것 (손에, 주머니에, 가방에)
  2. hold(쥐고 있음)와 구별: 이동이 시작되면 carry
  3. carry + 물건 + in/on + 장소 구조

  · I carry everything I need in my backpack. (필요한 모든 것을 가방에 넣고 다녀요)
- 핵심 어휘: carry, pocket, backpack, everything I need
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 carry 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 평소에 어떤 가방을 들고 다니나요?
2) 주머니에 주로 무엇을 넣고 다니나요?
3) 손에 직접 들고 다니는 물건이 있나요?
4) 가방 안에 꼭 넣고 다니는 물건은 무엇인가요?
5) 주머니에 물건 넣고 다니는 걸 좋아하나요, 싫어하나요?

제 답변 5개를 받은 다음, carry 패턴으로 영어 문장을 만들어 주세요.
각 영어 문장은 한국어 의역을 같이 붙여 주세요. 마지막에 제가 따라 말할 수 있도록 5문장을 표로 정리해 주세요.
```
