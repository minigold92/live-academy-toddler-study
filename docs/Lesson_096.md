# Lesson 096 — 걱정 표현 완전 정복: **I'm worried / I'm worried I might / What if / I don't want ~ to**

> 📺 **원본 영상**: [💡영어로 걱정 표현하기 ⟪ 기초 영어회화 / 영어 말하기 연습 ⟫](https://www.youtube.com/watch?v=6F2lQkvELTg)
> ⏱ 19분 09초 · 👁 144K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **누군가/무언가가 걱정되다** → `I'm worried about + 대상` (`about`은 "~에 대해서"가 아니라 "~이 걱정된다"는 뜻으로 받아들이기)
> 2. **~할까봐 걱정된다** → `I'm worried I might + 동사` (I'm worried that 생략 가능)
> 3. **~을 못할까봐 걱정된다** → `I'm worried I might not be able to + 동사` (`might not` 뒤에 반드시 `be able to`)
> 4. **~이면 어떡하지?** → `What if + 현재형 문장` (`might` 불필요, `What if` 자체에 불확실성 내포)
> 5. **~하지 않았으면 좋겠다** → `I don't want + 목적어 + to + 동사`

## ✨ 상황 1 — 누군가가 걱정된다

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I'm worried about her.** | **그녀가 걱정돼요.** | **← 오늘 #1** |
| 2 | Are you worried? | 걱정돼요? | 의문형 |
| 3 | Yes, I am. | 네, 걱정돼요. | 대답 (Yes I do ❌ → Yes I am ✅) |
| 4 | No, I'm not. | 아니요, 안 걱정돼요. | 부정 대답 |
| 5 | What are you worried about? | 뭐가 걱정되는데요? | 대상 묻기 |
| 6 | Why are you worried? | 왜 걱정되는데요? | 이유 묻기 |
| 7 | Why are you worried about that? | 그게 왜 걱정되는데요? | 특정 대상 이유 묻기 |

## 🔑 오늘의 변환/응용 패턴 정리

```
[걱정 대상 표현]
I'm worried about her.                  그녀가 걱정돼요.
Are you worried?                        걱정돼요?   → 대답: Yes, I am. / No, I'm not.
What are you worried about?             뭐가 걱정되는데요?
Why are you worried about that?         그게 왜 걱정되는데요?

[~할까봐 걱정됨]
I'm worried I might + 동사              ~할까봐 걱정돼요
I'm worried I might not + 동사          ~안 할까봐 걱정돼요 (의지)
I'm worried I might not be able to + 동사  ~못 할까봐 걱정돼요 (능력·기회)

[이러면 어떡하지?]
What if + 주어 + 현재형 동사?           ~이면 어떡해?
※ What if 뒤에 might 불필요
※ 3인칭 단수 주어는 동사에 -(e)s 붙이기 (What if he asks?)

[~하지 않았으면 좋겠다]
I don't want + 목적어 + to + 동사       ~가 ~하지 않았으면 좋겠다
I don't want them to find out.         걔네들이 알아내지 않았으면 좋겠어.

[주의 사항]
gain weight  (O)  ←  get weight (X)
get upset at my kids  (O)  ← get upset to my kids (X)
pay the bill  (O, 청구서를 내다)  ←  pay for the bill  (X, 그 청구서를 사다)
be too late  (O)  ←  to be late  (X)
lose one's temper  = 화를 참지 못하고 내다
get better  = 컨디션/건강이 좋아지다
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **worried about** | adj.+prep. | ~이 걱정되다 | `about`은 "대해서"가 아니라 "이 걱정돼"로 통째로 받아들이기 |
| **might** | modal v. | ~할 수도 있다 | 불확실한 가능성; `might not` = 안 할 수도 |
| **be able to** | v.phrase | ~할 수 있다 | `might not be able to` = 못 할까봐 |
| **lose one's temper** | v.phrase | 화를 참지 못하고 내다 | ↔ `bear` (내부에서 참다) |
| **gain weight** | v.phrase | 살찌다 | `get weight` 사용 불가 |
| **get better** | v.phrase | (컨디션·건강이) 좋아지다 | `recover my condition`보다 자연스러움 |
| **miss out** | v.phrase | 놓치다 | `I'm worried I might miss out.` |
| **find out** | v.phrase | 알아내다, 알아채다 | `I don't want them to find out.` |
| **turn into** | v.phrase | ~으로 바뀌다, 번지다 | `It might turn into an argument.` |
| **get a fine** | v.phrase | 벌금/딱지를 끊기다 | `get a fine for speeding` |

---

## 🗣️ 발음 팁

1. **What are you worried about?** — `What are you` 를 "와루유" 처럼 이어서. `worried` 강세는 첫 음절 "WOR-ried". 마지막 `about`을 약하게 흘리면 자연스러움.
2. **I'm worried I might** — `worried`와 `I` 사이에 끊지 말고 한 호흡으로. "아임 워리드 아이 마잇"처럼.
3. **might not be able to** — 빠르게 말할 때 "마잇낫비에이블투" 처럼 한 덩어리로. `not`을 강하게 발음.

## 📝 파생 연습 (5문제)

1. **영작**: "내가 영어 시험을 통과하지 못할까봐 걱정돼요." (`might not be able to` 활용)
2. **변환**: 위 문장을 `What if` 표현으로 바꿔 보세요.
3. **완성**: "뭔가 잘못 말할까봐 걱정돼요." 한 문장으로.
4. **오류 수정**: `I'm worried I might to forget.` → 올바르게 고치세요.
5. **세트 작성**: "걔네들이 코로나 걸리는 게 걱정돼요"를 세 가지 표현으로 (I'm worried / What if / I don't want).

<details>

2. `What if I can't pass the English test?`

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요. (제가 잘못 이해한 부분이 없는지 확인)
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요. 그 후엔 제 질문에 답하면서 추가 예문/유사 표현을 계속 만들어 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 걱정 표현 — I'm worried about / I'm worried I might / What if / I don't want ~ to
- 오늘의 핵심:
  1. 걱정 대상: I'm worried about + 대상 (about = 걱정의 대상 표시)
  2. ~할까봐: I'm worried I might + 동사원형
  3. ~못할까봐: I'm worried I might not be able to + 동사원형
  4. ~이면 어떡해?: What if + 현재형 (might 불필요)

- 핵심 예문:
  · I'm worried about her. (그녀가 걱정돼요)
  · What are you worried about? (뭐가 걱정되는데요?)

  · I'm worried I might say something wrong. (무슨 말을 잘못할까봐 걱정돼요)

  · I don't want them to find out. (걔네들이 알아내지 않았으면 좋겠어)
- 주의 사항: gain weight (O) / get weight (X); pay the bill (O) / pay for the bill (X); be too late (O) / to be late (X)
- 핵심 어휘: worried about, might not be able to, lose one's temper, gain weight, get better, miss out, find out, turn into, get a fine
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

1) 한 번에 한국어 문장 1개만 제시.

처음 5문제는 레슨에 나온 표현 그대로, 나머지 5문제는 같은 문법 포인트를 다른 단어/상황으로 응용해 주세요.

- 바람: I don't want + 목적어 + to + 동사원형 (예: I don't want them to find out.)
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 걱정 표현 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 요즘 걱정되는 사람이나 상황이 있는지
2) 내가 직접 뭔가를 잘못할까봐 걱정되는 것
3) 내가 어떤 것을 못하게 될까봐 걱정되는 것 (능력·기회)
4) "이러면 어떡하지?" 하고 상상하는 상황
5) 어떤 사람이나 것에 대해 "그러지 않았으면 좋겠다"는 바람

제 답변 5개를 받은 다음, 그것을 재료로 다음을 만들어 주세요:
- I'm worried about [대상].
- I'm worried I might [동사].
- I'm worried I might not be able to [동사].
- What if [상황]?
- I don't want [목적어] to [동사].

각 영어 문장은 한국어 의역, 발음 팁(필요하면)을 같이 붙여 주세요. 마지막에 제가 따라 말할 수 있도록 5문장을 표로 정리해 주세요.
```

- 중간에 **"좀 더 천천히"**, **"한국어로 설명해 줘"** 같은 보조 지시를 자유롭게 끼워 넣으세요.
- 레슨 진도가 쌓이면 여러 레슨의 자료 블록을 합쳐 붙여넣어 누적 복습도 가능.
