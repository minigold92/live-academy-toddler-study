# Lesson 077 — I want to be + 형용사/명사/전치사구/분사 (되고 싶다, 있고 싶다, 대해지고 싶다)

> 📺 **원본 영상**: [(don't) want to be + 4가지 패턴](https://www.youtube.com/watch?v=sIJAyQv4reg)
> ⏱ 25분 09초 · 👁 152K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. `I want to be + 형용사` — 어떤 상태이고 싶다/그렇게 행동하고 싶다
> 2. `I want to be + 명사` — 어떤 존재가 되고 싶다 (행동까지 내포)
> 3. `I want to be + 전치사구` — 어딘가에/누군가와 있고 싶다
> 4. `I want to be + 분사(과거분사)` — 어떤 대우를 받고 싶다 (수동의 뉘앙스)
> 5. 3인칭 연습 필수 → `He wants / She wants / Everyone wants` (단수 주어 주의)

> 💡 강사의 한마디:
> "모든 주어를 다 I로만 하면은 이 3인칭 활용에 대한 연습이 안 돼요. 기회가 될 때마다 3인칭으로 바꿔서 하는 연습을 꼭 챙겨서 하셔야 돼요."

---

## ✨ 상황 1 — want to be + 형용사

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I want to be careful.** | **나 조심하고 싶어. (조심스럽게 행동하고 싶어.)** | **← 오늘 #1** |
| 2 | 🆕 **He doesn't want to be alone.** | **걔는 혼자 있고 싶어 하지 않아.** | **← 오늘 #1 (3인칭)** |
| 3 | 🆕 **He just doesn't want to be alone.** | **그냥 혼자 있기 싫어서 그런 거야.** | **← 오늘 #1** |
| 4 | 🆕 **She just wants to be helpful.** | **그냥 도움을 주고 싶은 거야.** | **← 오늘 #1 (3인칭)** |
| 5 | 🆕 **I didn't want to be rude.** | **나는 무례하게 굴고 싶지 않았어.** | **← 오늘 #1 (과거형)** |
| 6 | 🆕 **I just wanna be good at my job.** | **난 그냥 내 일을 잘 하고 싶은 것뿐이야.** | **← 오늘 #1** |
| 7 | 🆕 **I didn't want to be too full.** | **너무 배부르고 싶지 않았어.** | **← 오늘 #1 (과거형)** |

## ✨ 상황 2 — want to be + 명사

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I just wanna be a good father.** | **나는 그냥 좋은 아빠가 되고 싶어.** | **← 오늘 #2** |
| 2 | 🆕 **I don't wanna be a disappointment to my parents.** | **부모님께 실망스러운 존재가 되고 싶지 않아.** | **← 오늘 #2** |
| 3 | 🆕 **She wants to be a singer when she grows up.** | **커서 가수가 되고 싶어 한대.** | **← 오늘 #2 (3인칭)** |
| 4 | 🆕 **I don't wanna be a burden to my family.** | **가족에게 짐이 되고 싶지 않아.** | **← 오늘 #2** |
| 5 | 🆕 **I really wanna be a part of that team.** | **그 팀의 일원이 정말로 되고 싶어.** | **← 오늘 #2** |

## ✨ 상황 3 — want to be + 전치사구

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I just wanna be with my family.** | **그냥 가족이랑 같이 있고 싶은 것뿐이야.** | **← 오늘 #3** |
| 2 | 🆕 **I didn't wanna be in that kind of relationship.** | **그런 관계 속에 있고 싶지 않았어.** | **← 오늘 #3** |
| 3 | 🆕 **I wanna be on the road before rush hour.** | **차가 막히기 전에 이동 중이고 싶어. (출발하고 싶어.)** | **← 오늘 #3** |
| 4 | 🆕 **I didn't wanna be in my room all day.** | **하루 종일 내 방에 있고 싶지 않았어.** | **← 오늘 #3** |
| 5 | 🆕 **She wants to be with someone she has more in common with.** | **더 많은 공통점이 있는 사람과 함께 있고 싶어 한대.** | **← 오늘 #3** |

## ✨ 상황 4 — want to be + 분사 (수동, 대우받고 싶다)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **Everyone wants to be respected.** | **누구나 다 존중을 받고 싶어 해.** | **← 오늘 #4 (everyone = 단수 → wants)** |
| 2 | 🆕 **Nobody wants to be disrespected.** | **아무도 무시당하고 싶어 하지 않아.** | **← 오늘 #4 (nobody = 단수 → wants)** |
| 3 | 🆕 **I didn't want to be labeled as a coward.** | **겁쟁이로 낙인 찍히고 싶지 않았어.** | **← 오늘 #4** |
| 4 | 🆕 **We just want to be treated fairly.** | **우리는 그냥 공평하게 대우받고 싶은 것뿐이에요.** | **← 오늘 #4** |
| 5 | 🆕 **I don't want to be trapped.** | **갇혀 있고 싶지 않아.** | **← 오늘 #4** |

### 🧩 문맥으로 이어 말하기
> **I just wanna be good at my job. That's all.**
> (난 그냥 내 일을 잘 하고 싶은 것뿐이야. 그게 다야.)

> **We just want to be treated fairly. We're not asking for a lot.**
> (우리는 그냥 공평하게 대우받고 싶은 것뿐이에요. 많은 걸 바라는 게 아니에요.)

## 🔑 오늘의 패턴 정리

```
[want to be 뒤에 오는 4가지 패턴]

① 형용사  → I want to be careful.           (조심하고 싶다)
            He doesn't want to be alone.    (혼자 있고 싶지 않다)
            I didn't want to be too full.   (너무 배부르고 싶지 않았다)

② 명사    → I wanna be a good father.       (좋은 아빠가 되고 싶다)
            I don't wanna be a burden.      (짐이 되고 싶지 않다)
            She wants to be a singer.       (가수가 되고 싶다)

③ 전치사구 → I wanna be with my family.     (가족과 함께 있고 싶다)
             I wanna be on the road.        (이동 중이고 싶다)
             I didn't wanna be in my room.  (방에 있고 싶지 않았다)

④ 분사(과거분사) → Everyone wants to be respected.     (존중받고 싶다)
                   I didn't want to be labeled as ~.   (낙인 찍히고 싶지 않았다)
                   We want to be treated fairly.        (공평하게 대우받고 싶다)

[주의: 주어 변화]
I want  / He wants / She wants / Everyone wants / Nobody wants
I don't / He doesn't / Nobody wants (이미 부정 → don't 불필요)
I wanted (과거) / I didn't want (과거 부정)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **rude** | adj. | 무례한 | I didn't want to be rude. |
| **helpful** | adj. | 도움이 되는 | She just wants to be helpful. |
| **burden** | n. | 짐 (부담보다 짐에 가까운 어감) | be a burden to = ~에게 짐이 되다 |
| **disappointment** | n. | 실망스러운 존재/실망 | be a disappointment to = ~에게 실망을 주다 |
| **a part of** | 표현 | ~의 일원, ~의 일부 | be a part of that team |
| **rush hour** | n. | 출퇴근 러시 시간 | before rush hour = 차 막히기 전에 |
| **label (as)** | v. | ~로 낙인 찍다, 라벨 붙이다 | labeled as a coward |
| **treat** | v. | 대우하다, 대하다 | be treated fairly = 공평하게 대우받다 |
| **trap** | v. | 가두다 | be trapped = 갇히다 |
| **in common** | 표현 | 공통점이 있는 | have more in common with |
| **coward** | n. | 겁쟁이 | |

## 🗣️ 발음 팁

1. **wanna** — `want to`의 구어 발음. "워너"처럼. `I just wanna be ~`를 한 호흡으로.
2. **everyone wants** — everyone은 단수. `everyone want`가 아니라 `everyone wants`.
3. **be treated fairly** — `treated`의 d와 `fairly`의 f가 이어지며 빠르게. "트리릿 페얼리"처럼.

## 📝 파생 연습 (5문제)

1. **영작**: "그녀는 혼자 있고 싶어 하지 않아." (3인칭, alone)
2. **영작**: "난 그냥 좋은 엄마가 되고 싶어."
3. **영작**: "모두가 사랑받고 싶어 해." (loved, everyone)
4. **영작**: "나는 그런 상황에 있고 싶지 않았어." (in that situation, 과거)
5. **구분**: `I wanna be a good father`와 `I wanna become a good father`의 느낌 차이를 우리말로 설명하면?

<details>
<summary>정답 보기</summary>

1. `She doesn't want to be alone.`
2. `I just wanna be a good mother.`
3. `Everyone wants to be loved.`
4. `I didn't wanna be in that situation.`

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 "I want to be ~" 4가지 패턴을 제 실제 바람/가치관으로 만들고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 어떤 사람이 되고 싶나요? (직업, 성격, 역할)
2) 어떤 상태이고 싶지 않나요? (하고 싶지 않은 것)
3) 누구와 어디에 있고 싶나요?
4) 어떤 대우를 받고 싶나요? (존중, 공평한 대우 등)

제 답변을 받은 다음, 아래를 만들어 주세요:
- want to be + 형용사 문장 1개
- want to be + 명사 문장 1개
- want to be + 전치사구 문장 1개
- want to be + 분사 문장 1개
- 부정형 (don't want to be) 문장 1개

각 문장에 한국어 의역과 발음 팁(필요하면)을 붙여 주세요. 마지막에 5문장 표로 정리해 주세요.
```
