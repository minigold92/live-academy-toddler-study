# Lesson 013 — **There is / There are**로 있다·없다·많다·별로 없다 표현하기

> 📺 **원본 영상**: [[LESSON #13 - PART 1] "무엇이 있다/없다" - there is(are) ~ there is(are) no ~](https://www.youtube.com/watch?v=WSUM2yUsub8)
> ⏱ 12분 56초 · 👁 567K회

> 🎯 **오늘의 핵심 포인트**
> 1. **있다** → `There is + 단수/셀 수 없는 명사` / `There are + 복수`
> 2. **없다** → `There is no + 명사` / `There are no + 복수명사`
> 3. **많다** → `There are many ~` / `There are a lot of ~` (셀 수 있는 것) / `There is a lot of ~` (셀 수 없는 것)
> 4. **별로 없다** → `There aren't many ~` / `There isn't a lot of ~`
> 5. **셀 수 있는 명사(car, person)** → `are` / **셀 수 없는 명사(traffic, time)** → `is`

---

## ✨ 상황 1 — 사람이 있다 / 많다

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **There is a person.** | **사람이 (한 명) 있습니다.** | **← 오늘 #1 (단수)** |
| 2 | 🆕 **There are people.** | **사람들이 있습니다.** | **← 오늘 #1 (복수)** |
| 3 | 🆕 **There are many people.** | **사람들이 많습니다.** | **← 오늘 #3 (many)** |
| 4 | 🆕 **There are a lot of people.** | **사람들이 많습니다.** | **← 오늘 #3 (a lot of)** |

---

## ✨ 상황 2 — 교통량 / 시간 (셀 수 없는 명사)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | There was a lot of traffic. | 차가 많이 막혔어. | 복습 (Lesson 10) |
| 2 | 🆕 **There is a lot of traffic.** | **차가 많이 막혀.** | **← 오늘 #5 (traffic = 셀 수 없는 명사 → is)** |
| 3 | 🆕 **There are a lot of cars.** | **차가 많아.** | **← 오늘 #5 (car = 셀 수 있는 명사 → are)** |
| 4 | 🆕 **There is a lot of time.** | **시간이 많아.** | **← 오늘 #5 (time = 셀 수 없는 명사 → is)** |

---

## ✨ 상황 3 — 없다

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **There is no bathroom.** | **화장실이 없어요.** | **← 오늘 #2 (단수로 없다)** |
| 2 | 🆕 **There are no bathrooms.** | **화장실이 없어요.** | **← 오늘 #2 (복수로 없다)** |
| 3 | 🆕 **There is no time.** | **시간이 없어요.** | **← 오늘 #2 (셀 수 없는 명사)** |

---

## ✨ 상황 4 — 이 근처에 좋은 식당이 많다 / 없다

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **There are many good restaurants near here.** | **이 근처에 좋은 식당이 많아요.** | **← 오늘 #3** |
| 2 | 🆕 **There are many good restaurants nearby.** | **이 근처에 좋은 식당이 많아요.** | **← 오늘 #3 (nearby)** |
| 3 | 🆕 **There are no good restaurants near here.** | **이 근처에 좋은 식당이 없어요.** | **← 오늘 #2** |
| 4 | 🆕 **There are no good restaurants nearby.** | **이 근처에 좋은 식당이 없어요.** | **← 오늘 #2** |

---

## ✨ 상황 5 — 별로 없다

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **There aren't many people.** | **사람이 별로 없어요.** | **← 오늘 #4** |
| 2 | 🆕 **There aren't a lot of people.** | **사람이 별로 없어요.** | **← 오늘 #4** |
| 3 | 🆕 **There isn't a lot of traffic.** | **교통량이 별로 많지 않아요.** | **← 오늘 #4 (셀 수 없는 명사)** |

---

## 🔑 오늘의 패턴 정리

```
[있다]
단수/셀 수 없는:  There is a person.           사람이 있어.
복수:             There are people.             사람들이 있어.

[많다]
셀 수 있는:       There are many/a lot of cars.  차가 많아.
셀 수 없는:       There is a lot of traffic.     교통이 많아.

[없다]
                  There is no bathroom.          화장실이 없어.
                  There are no bathrooms.        화장실이 없어.

[별로 없다 = 많다의 부정]
                  There aren't many people.      사람이 별로 없어.
                  There isn't a lot of traffic.  교통량이 별로 없어.

[near here vs. nearby]
near here = 여기 근처에  (기준점을 "여기"로 지정)
nearby    = 이 근처에    (기준이 이미 내포된 부사 → nearby here라고 하면 안 됨)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **traffic** | n. | 교통량 | 셀 수 없는 명사 → one traffic ❌ |
| **nearby** | adv. | 이 근처에 | nearby here라고 하면 안 됨 |
| **near here** | 부사구 | 여기 근처에 | "여기"를 기준으로 삼을 때 |
| **bathroom** | n. | 화장실 | 영어권에서 가장 일반적인 표현 |

---

## 🗣️ 발음 팁

1. **There is** — 연음 처리하여 빠르게 "데어리즈" 또는 "데어즈"처럼 발음하는 것이 자연스러워요.

---

## 📝 파생 연습 (4문제)

1. **영작**: "문제가 하나 있어요."
2. **영작**: "문제가 여러 개 있어요."
3. **변환 (없다)**: `There are many cars.` → "차가 없어요."
4. **변환 (별로 없다)**: `There are many restaurants.` → "식당이 별로 없어요."

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요. (제가 잘못 이해한 부분이 없는지 확인)
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요. 그 후엔 제 질문에 답하면서 추가 예문/유사 표현을 계속 만들어 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: There is / There are 로 있다·없다·많다·별로 없다 표현하기
- 오늘의 핵심:
  1. 있다: There is (단수/셀 수 없는) / There are (복수)
  2. 없다: There is no ~ / There are no ~
  3. 많다: There are many ~ / There are a lot of ~ / There is a lot of ~(셀 수 없는 것)
  4. 별로 없다: There aren't many ~ / There isn't a lot of ~
  5. 셀 수 있는 명사(car, person) → are / 셀 수 없는 명사(traffic, time) → is
- 핵심 예문:
  · There is a person. / There are people. (사람이 있다/사람들이 있다)
  · There is a lot of traffic. (교통량이 많아) ↔ There are a lot of cars. (차가 많아)
  · There is no time. (시간이 없어)
  · There are many good restaurants nearby. (이 근처에 좋은 식당이 많아)
  · There aren't many people. (사람이 별로 없어)
- 핵심 어휘: traffic, nearby, near here, bathroom
- 발음 팁: There is → 연음 처리 "데어즈"
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ 부분정답 / ❌). 잘못된 부분은 어떻게 고쳐야 하는지 짧게 설명.
3) 같은 패턴으로 난이도/응용을 조금씩 바꿔가며 총 10문제.
4) 마지막에 제가 자주 틀린 패턴 1~2가지를 짧은 복습 노트로 정리.

처음 5문제는 레슨에 나온 표현 그대로, 나머지 5문제는 같은 문법 포인트를 다른 단어/상황으로 응용해 주세요.

[레슨 핵심 패턴]
- There is + 단수/셀 수 없는 명사 (있다)
- There are + 복수명사 (있다)
- There is no ~ / There are no ~ (없다)
- There are many ~ / There are a lot of ~ (많다, 셀 수 있는 것)
- There is a lot of ~ (많다, 셀 수 없는 것: traffic, time)
- There aren't many ~ / There isn't a lot of ~ (별로 없다)
- 활용 어휘: traffic, time, cars, people, restaurants, bathroom, problem

자, 첫 문제 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 지금 내가 있는 곳 또는 자주 가는 곳 (동네/카페/학교 등)
2) 그곳에 많이 있는 것
3) 그곳에 없는 것 (불편한 점)
4) 그곳 근처에 있는 것 (식당, 카페, 가게 등)
5) 그곳에서 별로 없어서 아쉬운 것

제 답변 5개를 받은 다음, 오늘 배운 There is/are 패턴으로 각각 영어 문장을 만들어 주세요.
각 영어 문장은 한국어 의역을 같이 붙여 주세요. 마지막에 5문장을 표로 정리해 주세요.
```
