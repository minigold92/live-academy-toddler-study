# Lesson 088 — 시간·비용 표현: **it takes / it cost / it took + 구체적인 숫자**

> 📺 **원본 영상**: [⏰ 시간이 걸린다 / 💴 돈이 든다 ➠ 표현 방식이 같으니까 같이 연습하세요](https://www.youtube.com/watch?v=mwTOT6HDeLA)
> ⏱ 5분 40초 · 👁 38K회

> 🎯 **오늘의 핵심 포인트**
> 1. **시간이 걸리다** → `It takes long` / `It takes a long time` / `It takes a lot of time`
> 2. **구체적 시간** → `It took about two and a half hours.` (과거형 `took`)
> 3. **비용이 들다** → `It cost about 130,000원.` (cost의 과거형도 `cost`, 변화 없음)
> 4. **take와 cost는 구조가 매우 유사** — 시간/돈을 각각 넣으면 됨
> 5. **주의**: `It takes a lot.`만으로는 시간이 오래 걸린다는 말이 안 됨 → 반드시 `a lot of time`

---

## ✨ 상황 1 — 시간이 오래 걸린다

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It takes long.** | **오래 걸려요.** | **← 오늘 #1** |
| 2 | 🆕 **It takes a long time.** | **시간이 오래 걸려요.** | **← 오늘 #1 (time 추가)** |
| 3 | 🆕 **It takes a lot of time.** | **시간이 많이 걸려요.** | **← 오늘 #1 (lot of time)** |
| 4 | ❌ ~~It takes a lot.~~ | (시간에 대해서는 사용 불가) | 주의: `a lot`만으로는 시간 의미 안 됨 |

---

## ✨ 상황 2 — 구체적으로 얼마나 걸렸는지 (과거)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It took about two and a half hours.** | **두 시간 반 정도 걸렸어요.** | **← 오늘 #2 (took = take 과거)** |
| 2 | It took about two hours and a half. | 두 시간 반 정도 걸렸어요. | 도입 (어순 변형, 같은 의미) |

---

## ✨ 상황 3 — 비용이 얼마나 들었는지

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It cost about 130,000원.** | **13만 원 정도 들었어.** | **← 오늘 #3 (cost = cost 과거)** |

---

## ✨ 상황 4 — 많은 것이 필요할 때 (It takes a lot of ~)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | It takes a lot of patience. | 인내심이 많이 필요해요. | 도입 |
| 2 | It takes a lot of experience. | 경험이 많이 필요해요. | 도입 |
| 3 | It takes a lot of courage. | 용기가 많이 필요해요. | 도입 |
| 4 | It takes a lot of preparation. | 준비가 많이 필요해요. | 도입 |
| 5 | It takes a lot of practice. | 연습이 많이 필요해요. | 도입 |

---

## 🔑 오늘의 변환/응용 패턴 정리

```
[시간 표현]
It takes long.              오래 걸려요 (현재)
It takes a long time.       시간이 오래 걸려요 (현재)
It takes a lot of time.     시간이 많이 걸려요 (현재)
It took about two and a half hours.  두 시간 반 정도 걸렸어요 (과거)

❌ It takes a lot. → 시간 의미로 쓸 수 없음!
   It takes a lot of patience/courage/practice 등 → 가능 (time 외 명사)

[비용 표현]
It cost about 130,000원.    13만 원 정도 들었어 (과거 / cost는 과거형도 cost)
It costs about ~.            ~ 정도 들어요 (현재)

[take ↔ cost 구조 비교]
It took   + 얼마의 시간   → 시간 소요
It cost   + 얼마의 돈     → 비용 소요
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **takes / took** | v. | (시간이) 걸리다 | take → took (불규칙 과거) |
| **costs / cost** | v. | (돈이) 들다 | cost → cost (과거형 동일) |
| **a long time** | phrase | 오랜 시간 | `It takes a long time.` |
| **a lot of time** | phrase | 많은 시간 | `It takes a lot of time.` |
| **two and a half hours** | phrase | 두 시간 반 | `2.5 hours`를 영어로 |
| **patience** | n. | 인내심 | `It takes a lot of patience.` |
| **courage** | n. | 용기 | `It takes a lot of courage.` |
| **preparation** | n. | 준비 | `It takes a lot of preparation.` |

---

## 📝 파생 연습 (5문제)

1. **영작**: "여기서 거기까지 한 시간 정도 걸려요." (현재)
2. **변환**: 위 문장을 과거형으로.
3. **영작**: "그거 얼마 들었어?" (`How much did it cost?`)
4. **오류 수정**: `It takes a lot.` (시간이 오래 걸린다는 뜻으로 쓴 경우) → 올바르게 고치세요.
5. **영작**: "그 일은 많은 연습이 필요해요."

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요. (제가 잘못 이해한 부분이 없는지 확인)
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요. 그 후엔 제 질문에 답하면서 추가 예문/유사 표현을 계속 만들어 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 시간·비용 표현 — it takes / it took / it cost
- 오늘의 핵심:
  1. It takes long / a long time / a lot of time → 오래 걸리다
  2. It took about two and a half hours. → 두 시간 반 정도 걸렸어요 (과거)
  3. It cost about 130,000원. → 13만 원 정도 들었어 (과거, cost는 변화 없음)
  4. It takes a lot of + 명사 → 많은 ~이 필요하다 (patience, courage, practice 등)
  5. It takes a lot. 은 시간에 대해 쓸 수 없음 → 반드시 of time 필요
- 핵심 예문:
  · It takes long. (오래 걸려요)
  · It takes a long time. (시간이 오래 걸려요)
  · It takes a lot of time. (시간이 많이 걸려요)
  · It took about two and a half hours. (두 시간 반 정도 걸렸어요)
  · It cost about 130,000원. (13만 원 정도 들었어)
  · It takes a lot of patience / courage / practice / preparation. (인내심/용기/연습/준비가 많이 필요해요)
- 핵심 어휘: takes/took, costs/cost, a long time, a lot of time, two and a half hours, patience, courage, preparation, practice
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ 부분정답 / ❌). 잘못된 부분은 어떻게 고쳐야 하는지 짧게 설명.
3) 같은 패턴으로 난이도/응용을 조금씩 바꿔가며 총 10문제.
4) 마지막에 제가 자주 틀린 패턴 1~2가지를 짧은 복습 노트로 정리.

처음 5문제는 레슨에 나온 표현 그대로, 나머지 5문제는 같은 문법 포인트를 다른 단어/상황으로 응용해 주세요.

[레슨 핵심 패턴]
- 현재 시간: It takes long. / It takes a long time. / It takes a lot of time.
- 과거 시간: It took about + [시간]. (예: It took about two and a half hours.)
- 과거 비용: It cost about + [금액]. (예: It cost about 130,000원.)
- 많이 필요: It takes a lot of + [명사]. (patience / courage / practice / preparation / experience)
- 주의: It takes a lot. → 시간에는 쓸 수 없음

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 카페에서 만나서 영어로 잡담하는 중입니다.

규칙:
- 첫 인사는 당신이 먼저. ("How's it going?" 같은 자연스러운 질문)
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 제가 답하면 어색한 부분은 자연스럽게 다시 말해 주는 형식으로 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 화제를 유도해 주세요:
  · It takes / It took + 시간
  · It costs / It cost + 금액
  · It takes a lot of + 명사 (patience, practice 등)
  · How long does it take? / How much does it cost?

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 오늘 한 일 중 시간이 오래 걸린 것
2) 얼마나 걸렸는지 (구체적 시간)
3) 최근에 돈을 쓴 것 중 기억에 남는 것
4) 얼마나 들었는지
5) 무엇을 잘 하려면 많이 필요하다고 생각하는 것 (patience, practice, experience 등)

제 답변 5개를 받은 다음, 그것을 재료로 다음을 만들어 주세요:
- It took about [시간].
- It cost about [금액].
- It takes a lot of [명사].
- (필요하면) It takes a long time.

각 영어 문장은 한국어 의역을 같이 붙여 주세요. 마지막에 제가 따라 말할 수 있도록 5문장을 표로 정리해 주세요.
```
