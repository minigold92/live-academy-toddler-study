# 학습 자료 생성 스펙 (Lesson_NNN.md)

## 입력
- `transcripts/whisper/NN_videoID.txt` — 한국어 자막 (영어 예문 포함)

## 출력
- `study/Lesson_NNN.md` — 학습자용 마크다운 (NN은 입력 파일의 두자리 번호)

## 포맷 — 반드시 `study/Lesson_005.md`와 동일한 구조 사용

순서 (필수 섹션):
1. **제목** (`# Lesson NNN — 한 줄 상황/주제 요약`)
2. **🎯 오늘의 핵심 포인트** (blockquote, 1~5개 bullet)
3. **💡 강사의 한마디** (강사가 강조한 학습법/태도 발언이 있을 때만)
4. **✨ 상황 N — 한 줄 상황 설명** (테이블; "구분" 컬럼에 복습/오늘 #N/도입 표시, 🆕 마크는 오늘 새로 배운 항목)
5. **🧩 문맥으로 이어 말하기** (강사가 마지막에 합친 완성 문장이 있을 때)
6. **🔑 오늘의 변환/응용 패턴** (코드블록으로 명시적 패턴 정리)
7. **📖 핵심 어휘** (오늘 처음 나온 것만; 단어/품사/뜻/메모 컬럼)
8. **🗣️ 발음 팁** (강사가 직접 언급한 발음 코칭만)
9. **📝 파생 연습** (3~5문제, `<details>` 블록으로 정답 숨김)
10. **🤖 AI와 함께 공부하기** (4가지 모드 — 아래 템플릿)
11. **🎬 영상** (YouTube URL: `https://www.youtube.com/watch?v=VIDEOID`)

## AI 인터랙션 섹션 — 4모드 고정 (Lesson_005.md 그대로, 레슨 자료 부분만 갈아끼움)

각 코드블록 안의 `[레슨 자료]` / `[레슨 핵심 패턴]` 부분을 해당 레슨 내용으로 갈아끼우는 방식.

- 모드 1: 튜터 모드
- 모드 2: 퀴즈/드릴 모드
- 모드 3: 회화 롤플레이
- 모드 4: 응용 확장

## 작성 규칙
- **트랜스크립트에 실제로 등장한 영어 문장과 한국어 의역만 쓰기**. 임의 창작 금지.
- "복습"으로 표시되는 부분은 트랜스크립트에서 강사가 "지난 레슨에서 했죠?", "복습이에요" 등으로 명시한 부분만.
- "오늘 포인트"는 강사가 "오늘은 ~", "이번 레슨에서는 ~", "지금 ~을 배워볼게요" 등으로 새로 도입한 부분.
- 발음 팁은 강사가 명시적으로 발음을 지적한 경우만 (연음, 무음 처리, 빠르게/천천히 등).
- 어휘는 강사가 한국어로 뜻이나 어원/구조를 설명한 단어만 (모든 단어 다 적지 말 것).
- 강사 한마디는 학습 태도/철학/방법론 관련 멘트만 (있을 때만).
- 파생 연습은 레슨에서 다룬 패턴을 응용한 짧은 문제로.
- 코드블록의 한국어/영어 문장 줄맞춤은 표 형식이 깨지지 않게 신경.
- 영문 표기/대소문자는 강사가 말한 그대로 (예: `It's` `gonna` `be going to`).
