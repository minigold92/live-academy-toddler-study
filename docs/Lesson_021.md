# Lesson 021 — **annoying vs. annoyed / frustrating vs. frustrated**: 짜증 표현 구분하기 (Part 2)

> 📺 **원본 영상**: [[ LESSON #19 - PART 2 ] "짜증나" - annoyed / annoying / frustrated / frustrating](https://www.youtube.com/watch?v=Ks0qJXeTCcA)
> ⏱ 12분 34초 · 👁 377K회

> 🎯 **오늘의 핵심 포인트**
> 1. **`annoying / annoyed`** → 싫은 일이 반복적으로 일어날 때 느끼는 짜증 (신경에 거슬림)
> 2. **`frustrating / frustrated`** → 내 마음대로 안 돼서 느끼는 답답한 짜증
> 3. **상황 묘사** → `-ing형`: `It's so annoying` / `It was so frustrating`
> 4. **내 감정** → `-ed형`: `I was so annoyed` / `I was so frustrated`
> 5. **`keep -ing`** → 자꾸 ~한다 (반복되는 행동 묘사)

---

## ✨ 상황 1 — annoying / annoyed (반복되는 신경 거슬리는 짜증)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **He keeps texting me late at night when I'm trying to go to bed.** | **걔가 내가 자려고 할 때 밤 늦게 자꾸 문자를 보내.** | **← 오늘 #5 (keep -ing = 자꾸 ~하다)** |
| 2 | 🆕 **It's so annoying.** | **너무 짜증나.** | **← 오늘 #3 (상황 묘사 -ing형)** |
| 3 | 🆕 **Last night I was so annoyed.** | **어젯밤에는 너무 짜증이 났어.** | **← 오늘 #4 (내 감정 -ed형, 과거)** |
| 4 | 🆕 **I just turned off my phone.** | **그냥 폰을 꺼버렸어.** | **← 오늘 결과 문장** |

### 🧩 문맥으로 이어 말하기
> **He keeps texting me late at night when I'm trying to go to bed. It's so annoying. Last night I was so annoyed, I just turned off my phone.**
> (내가 자려고 할 때 걔가 자꾸 밤 늦게 문자를 보내. 너무 짜증나. 어젯밤에는 너무 짜증이 나서 그냥 폰을 꺼버렸어.)

---

## ✨ 상황 2 — frustrating / frustrated (내 마음대로 안 돼서 답답한 짜증)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I couldn't find my car key anywhere.** | **어디에서도 내 차 키를 찾을 수가 없었어.** | **← 오늘 상황 설명** |
| 2 | 🆕 **It was so frustrating.** | **너무 짜증났어. (답답했어.)** | **← 오늘 #3 (상황 묘사 -ing형)** |
| 3 | 🆕 **After about 30 minutes, I was so frustrated.** | **30분 정도 지나서 너무 답답하고 짜증이 났어.** | **← 오늘 #4 (내 감정 -ed형)** |
| 4 | 🆕 **I gave up and I just took the bus.** | **포기하고 그냥 버스 탔어.** | **← 오늘 결과 문장** |

### 🧩 문맥으로 이어 말하기
> **I couldn't find my car key anywhere. It was so frustrating. After about 30 minutes, I was so frustrated. I gave up and I just took the bus.**
> (어디에서도 차 키를 찾을 수가 없었어. 너무 짜증났어. 30분 정도 지나서 너무 답답해서 포기하고 그냥 버스 탔어.)

---

## 🔑 오늘의 핵심 대조

```
[annoy vs. frustrate 구분]
annoy     → 싫은 일이 자꾸 반복돼서 신경에 거슬리는 짜증
frustrate → 내 마음대로 안 돼서, 뭔가가 안 풀려서 느끼는 답답한 짜증

[상황 묘사 -ing vs. 내 감정 -ed]
상황 묘사  : It's so annoying.          (그 상황이 짜증나)
내 감정    : I was so annoyed.          (내가 짜증났어)

상황 묘사  : It was so frustrating.     (그 상황이 너무 답답했어)
내 감정    : I was so frustrated.       (내가 너무 답답했어)

[keep -ing = 자꾸 ~하다]
He keeps texting me.                    걔가 자꾸 문자를 보내.
She keeps calling me.                   걔가 자꾸 전화해.
It keeps happening.                     계속 그 일이 생겨.

[turn off = 끄다]
I turned off my phone.                  폰을 껐어.
(전원을 끄는 모든 상황에 turn off 사용 가능)

[give up = 포기하다]
I gave up.                              포기했어. (과거형)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **annoying** | adj. | 짜증나게 만드는 | 반복·지속되는 자극 → 신경 거슬림 |
| **annoyed** | adj. | 짜증난 (내 감정) | `I was so annoyed` |
| **frustrating** | adj. | 답답하게 만드는 | 내 마음대로 안 돼서 |
| **frustrated** | adj. | 답답한, 좌절스러운 (내 감정) | `I was so frustrated` |
| **keep -ing** | v. | 자꾸 ~하다 | `He keeps texting me` |
| **text** | v. | 문자를 보내다 | text를 동사로 쓰는 용법 |
| **late at night** | adv. | 밤 늦게 | 시간 표현 |
| **turn off** | v. | 끄다 | `turned off my phone` (과거형) |
| **give up** | v. | 포기하다 | `I gave up` (과거형) |
| **after about 30 minutes** | adv. | 30분 정도 지나서 | `about = 대략` |
| **anywhere** | adv. | 어디에도, 어디에서도 | 부정문에서 강조 |

## 📝 파생 연습 (5문제)

1. **골라 쓰기**: `annoying / frustrated` 중 하나 — "내 컴퓨터가 자꾸 꺼져서 너무 ______."
2. **영작**: "걔가 자꾸 늦어." (`keep -ing` 사용)
3. **빈칸 채우기**: "어디에서도 지갑을 찾을 수 없었어." → `I couldn't find my wallet _____.`
4. **영작**: "그 상황이 너무 답답했어." (`frustrating` 사용)
5. **영작**: "나 너무 짜증났어서 그냥 나왔어." (`annoyed` + `I just left` 사용)

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 주변에 자꾸 반복적으로 하는 짜증나는 행동이 있는 사람이 있나요? (annoy 상황)
2) 그게 얼마나 짜증났어요? (내 감정)
3) 최근에 뭔가 내 마음대로 안 돼서 답답했던 상황이 있었나요? (frustrate 상황)
4) 그때 어떤 기분이었나요?
5) 결국 어떻게 했나요? (포기했나요, 해결했나요 등)

제 답변을 받은 다음, 오늘 배운 annoying/annoyed, frustrating/frustrated 패턴으로 각각 영어 문장을 만들어 주세요.
한국어 의역과 함께 마지막에 5문장을 표로 정리해 주세요.
```
