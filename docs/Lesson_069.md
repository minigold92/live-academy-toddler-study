# Lesson 069 — 모임 약속 대화: **get together / by then / run late / make it**

> 📺 **원본 영상**: [[ 기초영어회화 ] 전치사 활용 연습](https://www.youtube.com/watch?v=B4e0xZZN2QE)
> ⏱ 14분 14초 · 👁 131K회

> 🎯 **오늘의 핵심 포인트**
> 1. **We get together** = 우리는 모인다 (`get together` = 모이다)
> 2. **based on everyone's schedule** = 모두의 스케줄을 토대로/바탕으로
> 3. **I can be there by then** = 그때까지 거기 갈 수 있을 것 같아 (`by` = 기한 개념, 그 이전에도 가능)
> 4. **by** vs **until** — 둘 다 "까지"이지만 의미가 다름
> 5. **if you're running late** = 늦어지고 있으면 (`run late` = 늦어지다)
> 6. **if you can't make it** = 못 오면/못 가면 (`make it` = 참석하다, 오다/가다)
> 7. **do let me know** = 꼭 좀 알려줘 (`do` = 강조 — 주저 말고 연락해 달라는 뉘앙스)

---

## ✨ 상황 1 — 모임 약속 대화 전체

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **We get together once a week.** | **우리는 일주일에 한 번 모여요.** | **← 오늘 #1** |
| 2 | 🆕 **We decide the exact date and time based on everyone's schedule, so it changes often.** | **모두의 스케줄을 바탕으로 정확한 날짜와 시간을 정하기 때문에 자주 바뀌어요.** | **← 오늘 #2** |
| 3 | 🆕 **This week we're meeting on Thursday at 8 o'clock in the evening.** | **이번 주는 목요일 저녁 8시에 만나요.** | **← 오늘 #3 (on + 요일, at + 시각, in the evening)** |
| 4 | 🆕 **Alright, I think I can be there by then.** | **알겠어요, 그때까지는 거기 갈 수 있을 것 같아요.** | **← 오늘 #4 (by then)** |
| 5 | 🆕 **We usually wait until everyone arrives.** | **우리는 보통 모두가 도착할 때까지 기다려요.** | **← 오늘 #5 (until)** |
| 6 | 🆕 **But do let me know if you're running late or if you can't make it.** | **그런데 늦어지거나 못 오면 꼭 좀 알려줘요.** | **← 오늘 #6** |
| 7 | Of course. | 물론이죠. | 도입 |

### 🧩 문맥으로 이어 말하기 (전체 대화)
> **A: We get together once a week. We decide the exact date and time based on everyone's schedule, so it changes often. This week we're meeting on Thursday at 8 o'clock in the evening.**
> **B: Alright, I think I can be there by then.**
> **A: Great! We usually wait until everyone arrives, but do let me know if you're running late or if you can't make it.**
> **B: Of course.**

## 🔑 오늘의 핵심 표현 정리

```
[by vs until — 둘 다 "까지"이지만 다름]
by then   : I can be there by then.      ← 그 시각까지 (더 일찍 도착할 수도 있음, 기한)
until     : We wait until everyone arrives. ← ~하는 순간까지 계속 기다림 (지속)

[run late vs be late]
be late   : She is late.         (늦은 상태)
run late  : She is running late. (늦어지고 있는 중)

[make it]
I can make it.    → 나 갈 수 있어. / 참석할 수 있어.
I can't make it.  → 나 못 가. / 참석 못 해.

[do + 동사원형 — 강조]
Let me know.      → 알려줘.
Do let me know.   → 꼭 좀 알려줘. (주저 말고, 미안해 하지 말고)

[전치사 정리]
on Thursday        (특정 요일 → on)
at 8 o'clock       (특정 시각 → at)
in the evening     (아침/오후/저녁 → in)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **get together** | 구동사 | 모이다 | `We get together once a week.` |
| **based on** | 전치사구 | ~을 토대로, 바탕으로 | `based on everyone's schedule` |
| **by then** | 부사구 | 그때까지 (기한) | `I can be there by then.` |
| **until** | 전치사/접속사 | ~까지 (지속) | `wait until everyone arrives` |
| **run late** | 표현 | 늦어지다 (진행 중) | `if you're running late` |
| **make it** | 표현 | 참석하다, 오다/가다 | `if you can't make it` |
| **do let me know** | 표현 | 꼭 좀 알려줘 | `do`가 동사 강조 |

---

## 🗣️ 발음 팁

1. **get together** — 빠르게 말하면 "게러게더"처럼 연음.
2. **based on everyone's** — `d`와 `o` 연음: "베이스돈 에브리원즈".
3. **by then** — "바이 덴". `th` 발음 주의.
4. **if you're running late** — "이퓨어 러닝 레잇"처럼 `if you're`가 연음.

---

## 📝 파생 연습 (5문제)

1. **영작**: "우리는 격주로 모여요." (every two weeks)
2. **영작**: "그때까지 거기 갈 수 있을 것 같아요." (by then)
3. **구별**: 모두가 올 때까지 기다린다 → `by` / `until` 중 어느 것?
4. **영작**: "늦어지면 꼭 좀 알려줘." (do let me know / if you're running late)
5. **영작**: "미안한데 내가 못 가게 됐어." (I can't make it)

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 패턴을 제 실제 모임 상황으로 연습하고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:

2) 날짜/시간을 어떻게 정하는지

- We get together + 빈도 문장
```
