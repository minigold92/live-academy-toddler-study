# Lesson 018 — **Does/Did/Would it take long?**: 시간이 걸리다 의문형 완성 (Part 2)

> 📺 **원본 영상**: [[ LESSON#17 - PART 2 ] - "오래 걸려요?" / "얼마나 (오래) 걸려요?"](https://www.youtube.com/watch?v=_49xUFx5n4M)
> ⏱ 8분 18초 · 👁 439K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **현재 의문형**: `Does it take long?` / `Does it take a lot of time?` — 오래 걸리나요?
> 2. **과거 의문형**: `Did it take long?` — 오래 걸렸나요?
> 3. **미래 의문형**: `Would it take long?` — 오래 걸릴까요? (`is it gonna` 보다 자연스러운 표현)
> 4. **정보 의문문**: `How long does/did/would it take?` — 얼마나 걸려요/걸렸어요/걸릴까요?
> 5. **`would`** 사용 이유: 앞으로 어떻게 될지는 그때 가봐야 알기 때문에 `would`가 더 자연스럽다

---

## ✨ 상황 1 — Yes/No 의문문 (현재·과거·미래)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **Does it take long?** | **오래 걸리나요?** | **← 오늘 #1 (현재)** |
| 2 | 🆕 **Does it take a lot of time?** | **시간이 많이 걸리나요?** | **← 오늘 #1 (현재)** |
| 3 | 🆕 **Did it take long?** | **오래 걸렸나요?** | **← 오늘 #2 (과거)** |
| 4 | 🆕 **Did it take a lot of time?** | **시간이 많이 걸렸나요?** | **← 오늘 #2 (과거)** |
| 5 | 🆕 **Would it take long?** | **오래 걸릴까요?** | **← 오늘 #3 (미래, would 권장)** |

---

## ✨ 상황 2 — How long 의문문 (얼마나 걸려요?)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **How long does it take?** | **얼마나 오래 걸려요?** | **← 오늘 #4 (현재)** |
| 2 | 🆕 **How long did it take?** | **얼마나 걸렸어요?** | **← 오늘 #4 (과거)** |
| 3 | 🆕 **How long would it take?** | **얼마나 걸릴까요?** | **← 오늘 #4 (미래)** |

---

## 🔑 오늘의 패턴 정리

```
[Yes/No 의문문]
현재  : Does it take long?       오래 걸리나요?
과거  : Did it take long?        오래 걸렸나요?
미래  : Would it take long?      오래 걸릴까요?   ← would 권장

[How long 의문문]
현재  : How long does it take?   얼마나 오래 걸려요?
과거  : How long did it take?    얼마나 걸렸어요?
미래  : How long would it take?  얼마나 걸릴까요?

[만드는 방법]
Does it take long?  → long을 앞으로 + How long = How long does it take?
Did it take long?   → long을 앞으로 + How long = How long did it take?
Would it take long? → long을 앞으로 + How long = How long would it take?

[왜 would?]
앞으로 있을 일의 시간은 그때 가봐야 알기 때문에
"걸릴 건가요?"(is it gonna) 보다 "걸릴까요?"(would it) 가 더 자연스럽다.
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **would** | 조동사 | ~할까요? / ~할 것이다 | 미래 의문문에서 가장 자연스러운 형태 |
| **how long** | 의문사 | 얼마나 오래 | `how long does/did/would it take?` |

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: Does/Did/Would it take long? — 시간이 걸리다 의문형 완성 (Lesson 17 Part 2)
- 오늘의 핵심:
  1. 현재 의문: Does it take long? / Does it take a lot of time?
  2. 과거 의문: Did it take long?
  3. 미래 의문: Would it take long? (is it gonna보다 자연스러움)
  4. How long 의문: How long does/did/would it take?
  5. would를 쓰는 이유: 앞으로 어떻게 될지는 그때 가봐야 알기 때문
- 핵심 예문:
  · Does it take long? (오래 걸리나요?)
  · Did it take long? (오래 걸렸나요?)
  · Would it take long? (오래 걸릴까요?)
  · How long does it take? (얼마나 오래 걸려요?)
  · How long did it take? (얼마나 걸렸어요?)
  · How long would it take? (얼마나 걸릴까요?)
- 핵심 어휘: would, how long
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ / ❌).
3) 총 10문제.
4) 마지막에 자주 틀린 패턴 복습 노트 정리.

[레슨 핵심 패턴]
- Does it take long? / Does it take a lot of time? (현재 의문)
- Did it take long? (과거 의문)
- Would it take long? (미래 의문, would 권장)
- How long does it take? (현재 정보 의문)
- How long did it take? (과거 정보 의문)
- How long would it take? (미래 정보 의문)

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 나는 새로운 취미를 배우려고 하고 있고, 당신에게 조언을 구하고 있습니다.

규칙:
- 첫 인사는 당신이 먼저.
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 어색한 부분은 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 화제를 유도해 주세요:
  · Does it take long to learn?
  · How long did it take you?
  · Would it take a lot of time?
  · How long would it take to get good at it?

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 의문문 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 누군가에게 물어보고 싶은 것 — 뭔가를 하는 데 얼마나 걸리는지 (현재)
2) 최근에 했던 일 — 얼마나 걸렸는지 궁금했던 것 (과거)
3) 앞으로 할 일 중에 얼마나 걸릴지 궁금한 것 (미래, would)
4) 오래 걸릴 것 같나요, 별로 안 걸릴 것 같나요? (would로 대답)
5) 그게 돈도 많이 들까요? (it costs 연결)

제 답변을 받은 다음, 오늘 배운 의문문 패턴으로 각각 영어 문장을 만들어 주세요.
한국어 의역과 함께 마지막에 5문장을 표로 정리해 주세요.
```
