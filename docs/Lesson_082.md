# Lesson 082 — 주말 계획 + 변수: **I'm planning to / but if it rains / we might**

> 📺 **원본 영상**: [영어 말하기 운동 ➠ ~ 할 계획인데 (만약) 비가 오면 안 할 수도 있어](https://www.youtube.com/watch?v=oxcmKf9-GIk)
> ⏱ 6분 25초 · 👁 49K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **계획 표현** → `I'm planning to + 동사` (= 할 계획이다)
> 2. **변수 조건** → `But if it rains` (= 근데 비가 오면) — 통째로 입에 붙이기
> 3. **불확실한 결과** → `we might` / `we might not` (= 할 수도 있어 / 안 할 수도 있어)
> 4. **축약 발음** → `We are planning to` → **`We're planning to`** (were 과거형과 발음 동일)

> 💡 강사의 한마디:
> "소리와 의미를 외우세요. 이해는 한 번이면 충분해요."

---

## ✨ 상황 1 — 이번 주말에 강아지를 공원에 데려갈 계획, 근데 비가 올 수도

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I'm planning to take my dogs to the park this weekend.** | **이번 주말에 강아지들을 공원에 데려갈 계획이에요.** | **← 오늘 #1 (계획)** |
| 2 | 🆕 **But if it rains,** | **근데 비가 오면,** | **← 오늘 #2 (조건절)** |
| 3 | 🆕 **we might just stay home.** | **그냥 집에 있을 수도 있어요.** | **← 오늘 #3 (불확실한 결과)** |
| 4 | We might not. | 안 갈 수도 있어. | 파생 (might not) |

### 🧩 문맥으로 이어 말하기
> **I'm planning to take my dogs to the park this weekend. But if it rains, we might just stay home.**
> (이번 주말에 강아지들을 공원에 데려갈 계획인데, 비가 오면 그냥 집에 있을 수도 있어요.)

---

## ✨ 상황 2 — 세차장이 닫으면 다음 주로 미룰 수도

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | If the car wash is closed, | 세차장이 닫으면, | 도입 (조건절 응용) |
| 2 | I might do it next week. | 다음 주에 할 수도 있어. | 도입 |

---

## ✨ 상황 3 — 아이들을 에버랜드에 데려갈 계획 (We're planning to)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **We're planning to take our kids to Everland this weekend.** | **이번 주말에 아이들을 에버랜드에 데려갈 계획이에요.** | **← 오늘 #1 응용 (We're)** |
| 2 | But if it rains, we might not. | 근데 비가 오면 안 갈 수도 있어. | 조건 + might not |

---

## 🔑 오늘의 변환/응용 패턴 정리

```
[계획 + 조건 + 결과 패턴]
I'm planning to  + [동사구]         → 계획
We're planning to + [동사구]        → 우리 계획 (We're = were 발음)
But if it rains,                    → 근데 비가 오면 (조건절, 통째로 외우기)
we might + [동사]                   → 할 수도 있어
we might not                        → 안 할 수도 있어
I might do it next week.            → 다음 주에 할 수도 있어
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **planning to** | v.phrase | ~할 계획이다 | `I'm / We're planning to + 동사원형` |
| **might** | modal v. | ~할 수도 있다 | 불확실한 가능성; `might not` = 안 할 수도 |
| **stay home** | v.phrase | 집에 있다 | `stay home` / `stay at home` |

---

## 🗣️ 발음 팁

1. **We're planning to** — `We are`를 `were`(과거형)와 똑같이 발음. "월"이 아니라 이를 약간 당기면서 "위" 소리. → "위얼 플래닝 투"처럼 한 호흡으로.
2. **But if it rains** — 통째로 한 호흡으로. "벗 이핏 레인즈" 정도로 후루룩 이어서.

---

## 📝 파생 연습 (4문제)

1. **영작**: "이번 주말에 부모님 댁에 갈 계획이에요." (take → visit 응용)
2. **변환**: 위 문장에 "근데 차가 막히면" 조건절을 붙여 보세요.
3. **완성**: 조건 결과로 "다음 주에 갈 수도 있어요." 한 문장으로.
4. **변환**: `I'm planning to` → `We're planning to`로 바꾸고 자연스럽게 발음해 보세요.

<details>
<summary>정답 보기</summary>

1. `I'm planning to visit my parents this weekend.`
2. `But if there's traffic,` (또는 `But if the roads are bad,`)
3. `I might go next week.`
4. `We're planning to visit my parents this weekend.` — `We're` 발음은 `were`와 동일.
</details>

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 4가지를 1줄씩으로 다시 설명해 주세요. (제가 잘못 이해한 부분이 없는지 확인)
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요. 그 후엔 제 질문에 답하면서 추가 예문/유사 표현을 계속 만들어 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 주말 계획 표현 + 조건절 + 불확실한 결과 (I'm planning to / but if it rains / we might)
- 오늘의 핵심:
  1. 계획: I'm / We're planning to + 동사
  2. 조건: But if it rains (통째로 외우기)
  3. 불확실한 결과: we might / we might not
  4. 발음: We're = were 발음과 동일
- 핵심 예문:
  · I'm planning to take my dogs to the park this weekend. (이번 주말에 강아지들을 공원에 데려갈 계획이에요)
  · But if it rains, we might just stay home. (비가 오면 집에 있을 수도 있어요)
  · We're planning to take our kids to Everland this weekend. (아이들을 에버랜드에 데려갈 계획이에요)
  · We might not. (안 갈 수도 있어)
  · I might do it next week. (다음 주에 할 수도 있어)
- 핵심 어휘: planning to, might, might not, stay home
- 발음 팁: We're 발음, But if it rains 연음
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ 부분정답 / ❌). 잘못된 부분은 어떻게 고쳐야 하는지 짧게 설명.
3) 같은 패턴으로 난이도/응용을 조금씩 바꿔가며 총 10문제.
4) 마지막에 제가 자주 틀린 패턴 1~2가지를 짧은 복습 노트로 정리.

처음 5문제는 레슨에 나온 표현 그대로, 나머지 5문제는 같은 문법 포인트를 다른 단어/상황으로 응용해 주세요.

[레슨 핵심 패턴]
- 계획: I'm planning to + 동사원형 (예: I'm planning to take my dogs to the park.)
- 조건: But if it rains, (예: But if it rains, we might just stay home.)
- 불확실: we might / we might not (예: We might not.)
- 다음으로 미룸: I might do it next week.
- 활용 어휘: plan, stay home, might, take (someone) to (somewhere)

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 카페에서 만나서 영어로 잡담하는 중입니다.

규칙:
- 첫 인사는 당신이 먼저. ("How was your weekend?" 같은 자연스러운 질문)
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 제가 답하면 어색한 부분은 자연스럽게 다시 말해 주는 형식으로 부드럽게 교정 ("Oh, you mean ~ ?" 같은 식).
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 화제를 유도해 주세요:
  · I'm planning to
  · But if it rains
  · we might just stay home
  · We might not
  · I might do it next week

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 3가지 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 이번 주말에 하려는 계획 (동사로 답하게 유도)
2) 그 계획을 방해할 수 있는 변수 (날씨, 일, 건강 등)
3) 변수가 생기면 어떻게 할 것 같은지
4) 다음 주에 하려는 다른 계획
5) 그 계획에도 혹시 변수가 있는지

제 답변 5개를 받은 다음, 그것을 재료로 다음을 만들어 주세요:
- I'm / We're planning to + [계획]
- But if [변수], [결과] + might / might not
- I might do it next week.

각 영어 문장은 한국어 의역, 발음 팁(필요하면)을 같이 붙여 주세요. 마지막에 제가 따라 말할 수 있도록 5문장을 표로 정리해 주세요.
```
