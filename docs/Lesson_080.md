# Lesson 080 — 감각동사 감각 기르기 (look / sound + 형용사 / like 명사 / like 문장)

> 📺 **원본 영상**: [실시간 영어 회화 강의 🔴 감각동사 연습 (5/30)](https://www.youtube.com/watch?v=YPqYxDg0pRo)
> ⏱ 156분 45초 · 👁 42K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. 감각동사 + 형용사 → `It sounds weird` / `You look great` / `That banana looks ripe`
> 2. 감각동사 + like + 명사 → `It looks like real leather` / `It sounds like a cat`
> 3. 감각동사 + like + 완성형 문장 → `It sounds like two cats are fighting`
> 4. `like` 뒤로 명사가 올 수도 있고, 완성형 문장(주어+동사)이 올 수도 있음
> 5. `That sounds like fun` — fun은 명사로도 쓰여 like 뒤에 올 수 있음 (회화체 관용 표현)

> 💡 강사의 한마디:
> "고양이 같은데? 라고 할 때 감각동사가 없잖아요 우리말에서는. 근데 영어로 했을 때는 감각동사가 의외로 쓰이는 경우가 많겠구나 — 이런 걸 좀 우리가 느껴야 돼요."

---

## ✨ 상황 1 — 감각동사 + 형용사 (보이다/들리다/느껴지다 + 상태)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It sounds weird.** | **이상하게 들려요. / 이상한데.** | **← 오늘 #1** |
| 2 | 🆕 **You look great.** | **좋아 보여요. / 멋있어 보여요.** | **← 오늘 #1** |
| 3 | 🆕 **You look very tired.** | **되게 피곤해 보여요.** | **← 오늘 #1** |
| 4 | 🆕 **It smells good.** | **좋은 냄새 나요.** | **← 오늘 #1** |
| 5 | 🆕 **That looks good.** | **좋아 보이네요. / 맛있겠다.** | **← 오늘 #1** |
| 6 | 🆕 **That banana looks ripe.** | **그 바나나 익은 것 같아요.** | **← 오늘 #1** |

---

## ✨ 상황 2 — 감각동사 + like + 명사 (∼처럼 보이다/들리다)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It looks like an apple.** | **사과처럼 생겼어요.** | **← 오늘 #2** |
| 2 | 🆕 **It looks like real leather.** | **진짜 가죽 같아요.** | **← 오늘 #2** |
| 3 | 🆕 **It sounds like a cat.** | **고양이 같은데요. (고양이 소리가 나요.)** | **← 오늘 #2** |
| 4 | 🆕 **That sounds like fun.** | **재밌겠다. / 재밌었겠다.** | **← 오늘 #2 (fun = 명사, 관용 표현)** |
| 5 | 🆕 **That looks like fun.** | **재밌어 보여요.** | **← 오늘 #2** |

### 🧩 문맥으로 이어 말하기

> **Wow, that sounds like fun!**
> (와, 재밌겠다! — 상대방이 재밌는 일을 했다거나 할 거라고 했을 때)

> **That sounds like a fun thing to do.**
> → (줄여서) **That sounds like fun.**
> (재밌는 것 같은데.)

---

## ✨ 상황 3 — 감각동사 + like + 완성형 문장 (∼인 것 같다, 추정)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It sounds like two cats are fighting.** | **고양이 두 마리가 싸우는 것 같아요.** | **← 오늘 #3** |
| 2 | 🆕 **It sounds like there is a problem outside.** | **밖에 무슨 문제가 있는 것 같아요.** | **← 오늘 #3** |
| 3 | 🆕 **It looks like he fell asleep on the beach.** | **해변에서 잠들었나봐요. (엄청 탄 사람을 보고)** | **← 오늘 #3** |

---

## 🔑 오늘의 패턴 정리

```
[유형 1] 감각동사 + 형용사
감각동사 (look / sound / smell / feel / taste) + 형용사
→ It sounds weird.        (소리가 이상하다)
→ You look great.         (좋아 보인다)
→ That looks good.        (좋아 보인다 / 맛있겠다)
→ That banana looks ripe. (그 바나나 익은 것 같다)

[유형 2] 감각동사 + like + 명사
감각동사 + like + 명사
→ It looks like real leather. (진짜 가죽 같다)
→ It sounds like a cat.       (고양이 소리 같다)
→ That sounds like fun.       (재밌겠다 — fun 명사 관용)

[유형 3] 감각동사 + like + 완성형 문장
감각동사 + like + 주어 + 동사 ...
→ It sounds like two cats are fighting.
   (고양이 두 마리가 싸우는 것 같다)
→ It looks like he fell asleep on the beach.
   (해변에서 잠들었나봐)
→ It sounds like there is a problem outside.
   (밖에 문제가 있는 것 같다)

[핵심 포인트]
- like 뒤: 명사 또는 완성형 문장 모두 가능
- 형용사 뒤에는 like 없이 바로 붙임 (It sounds weird ✅ / It sounds like weird ❌)
- that sounds like fun: fun이 명사라서 like 사용 — 예외적 관용 표현
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **look** | 감각동사 | ∼해 보이다 | look + 형용사 / like + 명사·문장 |
| **sound** | 감각동사 | ∼하게 들리다 | sound + 형용사 / like + 명사·문장 |
| **smell** | 감각동사 | ∼한 냄새가 나다 | smell good / smell like ... |
| **ripe** | adj. | (과일이) 익은 | That banana looks ripe. |
| **weird** | adj. | 이상한 | It sounds weird. |
| **real leather** | n. | 진짜 가죽 | It looks like real leather. |
| **That sounds like fun** | 표현 | 재밌겠다 | fun이 명사로 쓰인 관용 표현 |
| **fell asleep** | 동사구 | 잠들었다 | fall asleep의 과거형 |

---

## 🗣️ 발음 팁

1. **It sounds weird** — `sounds`의 s와 `weird`의 w를 이어서. "잇 사운즈 위얼드"처럼.
2. **You look great** — `look great`을 한 덩어리로. "유 룩 그뤠잇"처럼.
3. **That sounds like fun** — `sounds like`를 빠르게. "댓 사운즈 라익 펀"처럼.
4. **It sounds like two cats are fighting** — `like` 이후 전체를 자연스럽게 이어서.
5. **That banana looks ripe** — `looks ripe`를 "룩스 라입"처럼 연음.

---

## 📝 파생 연습 (5문제)

1. **영작**: "그거 진짜 맛있겠다." (감각동사 + 형용사, 먹을 거 보면서)
2. **영작**: "저 소리 기차 같은데요." (It sounds like + 명사)
3. **영작**: "그거 재밌겠다!" (That sounds like fun 사용)
4. **영작**: "밖에서 뭔가 떨어지는 것 같아요." (It sounds like + 완성형 문장)
5. **구분**: `It sounds strange`와 `It sounds like a cat`의 문법 구조 차이를 설명하면?

<details>
<summary>정답 보기</summary>

1. `That looks delicious.` 또는 `That looks so good.`
2. `It sounds like a train.`
3. `That sounds like fun!`
4. `It sounds like something is falling outside.`
5. `It sounds strange` = 감각동사 + 형용사 (like 없음) / `It sounds like a cat` = 감각동사 + like + 명사 (like 필요)
</details>

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 카페에서 만나서 이런저런 이야기를 하고 있습니다.

규칙:
- 첫 인사는 당신이 먼저.
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 제가 like를 빠트리거나 잘못 쓰면 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 유도해 주세요:
  · 감각동사 + 형용사 (looks good / sounds weird 등)
  · 감각동사 + like + 명사 (looks like / sounds like + 명사)
  · 감각동사 + like + 완성형 문장

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 감각동사 세 가지 유형을 제 일상 상황으로 연습하고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 오늘 먹었거나 봤던 음식 중에 인상적인 것이 있나요?
2) 최근에 들었던 이상한 소리나 재밌는 소리가 있었나요?
3) 주변에서 뭔가 ∼처럼 보인다 / ∼인 것 같다 싶었던 경험이 있나요?

제 답변을 받은 다음, 아래를 만들어 주세요:
- 감각동사 + 형용사 문장 2개
- 감각동사 + like + 명사 문장 1개
- 감각동사 + like + 완성형 문장 1개

각 문장에 한국어 의역을 붙여 주세요. 마지막에 5문장 표로 정리해 주세요.
```
