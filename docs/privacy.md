# 개인정보처리방침

본 사이트(`live-eng.minigold92.co.kr`)는 영어 회화 학습을 돕기 위한 개인 비상업 정적 사이트입니다. 본 사이트가 직접 수집·저장하는 개인정보는 없습니다. 다만 다음 제3자 서비스가 일반 웹 서버 수준의 접속 정보를 처리할 수 있습니다.

## 1. 제3자 서비스

### 1-1. GitHub Pages (호스팅)
- 처리자: GitHub, Inc.
- 정보: IP, User-Agent, 요청 URL 등 표준 접속 로그
- 정책: <https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement>

### 1-2. Cloudflare (도메인/DNS 및 트래픽 보호)
- 처리자: Cloudflare, Inc.
- 정보: IP, User-Agent, 요청 URL 등 기술 로그
- 정책: <https://www.cloudflare.com/privacypolicy/>

### 1-3. YouTube (영상 링크)
일부 페이지에 YouTube 영상 링크가 포함되어 있습니다. 영상을 시청할 경우 YouTube의 정책이 적용됩니다.
- 처리자: Google LLC
- 정책: <https://policies.google.com/privacy?hl=ko>

## 2. 쿠키
본 사이트가 직접 발행하는 쿠키는 없습니다. 위 제3자 서비스가 보안·세션 유지 목적으로 쿠키를 발행할 수 있습니다.

## 3. 콘텐츠 출처
본 사이트의 학습 자료는 YouTube [라이브 아카데미 토들러](https://youtube.com/playlist?list=PLEzsBdrpZXC8tdzAqQHEQ66TocGI-Kagh) 강의 영상의 자막을 기반으로 학습 목적에 맞게 재구성한 보조 자료입니다. 강의 콘텐츠의 권리는 원작자에게 있습니다. 학습 후 **원본 영상 시청을 강력히 권장**합니다.

원작자께서 자료 삭제·수정을 요청하실 경우 즉시 따르겠습니다.
