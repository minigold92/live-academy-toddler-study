# Lesson 017 — **It takes / It costs**: 시간이 걸리다, 돈이 들다 (현재·과거·미래 + 부정)

> 📺 **원본 영상**: [[ LESSON #17 - PART 1 ] "시간이 걸리다", "돈이 들다" - [ 영어회화 / 라이브아카데미 토들러 ]](https://www.youtube.com/watch?v=rFGmMnQFQaw)
> ⏱ 12분 21초 · 👁 491K회

> 🎯 **오늘의 핵심 포인트**
> 1. **시간이 걸리다** → `it takes` (현재) / `it took` (과거) / `it's gonna take` (미래)
> 2. **돈이 들다** → `it costs` (현재) / `it cost` (과거: 동사 변화 없음!) / `it's gonna cost` (미래)
> 3. **`cost`의 과거형은 `cost`** → `costs`(현재 3인칭)와 `cost`(과거)를 정확히 구분해야 함
> 4. **시간 부정**: `it doesn't take very long` (시간이 별로 안 걸려요)
> 5. **돈 부정**: `it doesn't cost very much` (돈이 별로 안 들어요)

---

## ✨ 상황 1 — 시간이 걸리다 (현재·과거·미래)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It takes 5 minutes.** | **5분이 걸려요.** | **← 오늘 #1 (현재)** |
| 2 | 🆕 **It takes a lot of time.** | **시간이 많이 걸려요.** | **← 오늘 #1 (현재)** |
| 3 | 🆕 **It takes really long.** | **시간이 진짜 오래 걸려요.** | **← 오늘 #1 (현재)** |
| 4 | 🆕 **It took a lot of time.** | **시간이 많이 걸렸어요.** | **← 오늘 #1 (과거: took)** |
| 5 | 🆕 **It took really long.** | **시간이 진짜 오래 걸렸어요.** | **← 오늘 #1 (과거)** |
| 6 | 🆕 **It's gonna take a lot of time.** | **시간이 많이 걸릴 거예요.** | **← 오늘 #1 (미래)** |
| 7 | 🆕 **It's gonna take really long.** | **시간이 진짜 오래 걸릴 거예요.** | **← 오늘 #1 (미래)** |

---

## ✨ 상황 2 — 돈이 들다 (현재·과거·미래)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It costs 5,000원.** | **5,000원이 들어요.** | **← 오늘 #2 (현재: costs — s 필수!)** |
| 2 | 🆕 **It costs a lot of money.** | **돈이 많이 들어요.** | **← 오늘 #2 (현재)** |
| 3 | 🆕 **It cost a lot of money.** | **돈이 많이 들었어요.** | **← 오늘 #3 (과거: cost, s 없음!)** |
| 4 | 🆕 **It's gonna cost a lot of money.** | **돈이 많이 들 거예요.** | **← 오늘 #2 (미래)** |

---

## ✨ 상황 3 — 부정문 (별로 안 걸려요 / 별로 안 들어요)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It doesn't take very long.** | **시간이 별로 안 걸려요.** | **← 오늘 #4 (현재 부정)** |
| 2 | 🆕 **It didn't take very long.** | **시간이 별로 안 걸렸어요.** | **← 오늘 #4 (과거 부정)** |
| 3 | 🆕 **It's not gonna take very long.** | **시간이 별로 안 걸릴 거예요.** | **← 오늘 #4 (미래 부정)** |
| 4 | 🆕 **It doesn't cost very much.** | **돈이 별로 안 들어요.** | **← 오늘 #5 (현재 부정)** |
| 5 | 🆕 **It didn't cost very much.** | **돈이 별로 안 들었어요.** | **← 오늘 #5 (과거 부정)** |

---

## 🔑 오늘의 변환 패턴 정리

```
[시간 — take]
현재  : It takes 5 minutes.            5분이 걸려요.
과거  : It took a lot of time.         시간이 많이 걸렸어요.
미래  : It's gonna take a lot of time. 시간이 많이 걸릴 거예요.

현재 부정 : It doesn't take very long.    시간이 별로 안 걸려요.
과거 부정 : It didn't take very long.     시간이 별로 안 걸렸어요.
미래 부정 : It's not gonna take very long.시간이 별로 안 걸릴 거예요.

[돈 — cost]
현재  : It costs a lot of money.        돈이 많이 들어요.   (s 필수)
과거  : It cost a lot of money.         돈이 많이 들었어요. (s 없음! cost = 과거형)
미래  : It's gonna cost a lot of money. 돈이 많이 들 거예요.

현재 부정 : It doesn't cost very much.   돈이 별로 안 들어요.
과거 부정 : It didn't cost very much.    돈이 별로 안 들었어요.

[long vs. much]
시간 → very long   (길고 짧음의 개념)
돈   → very much   (많고 적음의 개념)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **take** | v. | (시간이) 걸리다 | 과거형: `took` |
| **cost** | v. | (돈이) 들다, 비용이 들다 | 과거형: `cost` (변화 없음!) |
| **costs** | v. | (현재 3인칭) 든다 | `it costs` — s 빠뜨리면 과거로 오해됨 |

---

## 🗣️ 발음 팁

1. **`costs`** — `sts`로 끝나는 발음은 까다롭습니다. "코스츠" (ch 발음 금지). 천천히 연습.
2. **`it costs`와 `it cost`의 구분** — `costs`(현재) 뒤의 `s`를 정확히 발음하지 않으면 `cost`(과거)로 들립니다. 반드시 구분!

---

## 📝 파생 연습 (5문제)

1. **영작**: "거기 가는 데 2시간이 걸려요." (현재)
2. **변환 (과거)**: `It takes a lot of time.` → 과거형으로
3. **변환 (미래)**: `It costs a lot of money.` → 미래형으로
4. **영작**: "수리비가 별로 안 들었어요." (과거 부정)
5. **영작**: "수술은 시간이 오래 걸릴 거예요." (미래)

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: It takes / It costs — 시간이 걸리다, 돈이 들다 (현재·과거·미래·부정)
- 오늘의 핵심:
  1. 시간: it takes(현재) / it took(과거) / it's gonna take(미래)
  2. 돈: it costs(현재, s 필수) / it cost(과거, s 없음!) / it's gonna cost(미래)
  3. cost의 과거형은 cost — costs(현재)와 cost(과거)를 반드시 구분
  4. 시간 부정: it doesn't take very long / it didn't / it's not gonna take very long
  5. 돈 부정: it doesn't cost very much / it didn't cost very much
- 핵심 예문:
  · It takes 5 minutes. (5분이 걸려요)
  · It took a lot of time. (시간이 많이 걸렸어요)
  · It's gonna take a lot of time. (시간이 많이 걸릴 거예요)
  · It costs a lot of money. (돈이 많이 들어요)
  · It cost a lot of money. (돈이 많이 들었어요)
  · It doesn't take very long. (시간이 별로 안 걸려요)
  · It doesn't cost very much. (돈이 별로 안 들어요)
- 발음 팁: costs (sts 발음), costs vs cost 구분
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ / ❌).
3) 총 10문제.
4) 마지막에 자주 틀린 패턴 복습 노트 정리.

[레슨 핵심 패턴]
- It takes + 시간 (현재) / It took + 시간 (과거) / It's gonna take + 시간 (미래)
- It costs + 금액/a lot of money (현재, s 필수) / It cost (과거) / It's gonna cost (미래)
- It doesn't take very long (시간 부정, 현재)
- It didn't take very long (시간 부정, 과거)
- It doesn't cost very much (돈 부정, 현재)
- It didn't cost very much (돈 부정, 과거)

자, 첫 문제 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 집에서 직장/학교까지 얼마나 걸려요?
2) 최근에 시간이 오래 걸렸던 일이 있었나요?
3) 최근에 돈이 많이 들었던 것이 있었나요?
4) 앞으로 할 계획 중에 시간이 많이 걸릴 것 같은 것
5) 그게 돈이 많이 들 것 같나요, 별로 안 들 것 같나요?

제 답변을 받은 다음, 오늘 배운 it takes / it costs 패턴으로 각각 영어 문장을 만들어 주세요.
한국어 의역과 함께 마지막에 5문장을 표로 정리해 주세요.
```
