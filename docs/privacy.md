# 개인정보처리방침

본 사이트(`live-eng.minigold92.co.kr`)는 영어회화 학습을 돕기 위해 운영되는 비영리/학습용 정적 사이트입니다. 본 사이트가 직접 수집하는 개인정보는 없습니다. 다만 다음 제3자 서비스가 방문 정보를 처리할 수 있습니다.

## 1. 수집·이용되는 정보

### 1-1. GitHub Pages (호스팅 제공)
사이트 호스팅을 위해 GitHub Pages를 이용하며, GitHub는 접속 시 IP 주소, User-Agent, 페이지 URL 등 표준 웹 서버 로그를 일정 기간 보관할 수 있습니다.
- 처리자: GitHub, Inc.
- 정책: <https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement>

### 1-2. Cloudflare (도메인/DNS 및 트래픽 보호)
도메인 라우팅 및 보안을 위해 Cloudflare를 경유합니다. Cloudflare는 IP, User-Agent, 요청 URL 등 기술 로그를 처리할 수 있습니다.
- 처리자: Cloudflare, Inc.
- 정책: <https://www.cloudflare.com/privacypolicy/>

### 1-3. Google AdSense (광고)
사이트 운영 비용 충당을 위해 Google AdSense 광고를 게재할 수 있습니다. AdSense는 사용자 관심 기반 광고 표시를 위해 다음을 수집할 수 있습니다:
- 쿠키·광고 식별자 (게재한 광고 클릭/표시 측정)
- IP 주소, User-Agent, 페이지 URL
- 페이지 콘텐츠 키워드 기반 광고 매칭 정보

사용자는 [Google 광고 설정](https://adssettings.google.com/)에서 맞춤 광고를 끄거나 조정할 수 있습니다.
- 처리자: Google LLC
- 정책: <https://policies.google.com/technologies/ads?hl=ko>

### 1-4. YouTube (영상 임베드)
일부 페이지에 YouTube 영상 링크가 포함되어 있습니다. 영상을 시청할 경우 YouTube의 정책이 적용됩니다.
- 처리자: Google LLC
- 정책: <https://policies.google.com/privacy?hl=ko>

## 2. 쿠키 사용

본 사이트가 직접 발행하는 쿠키는 없습니다. AdSense·Google·Cloudflare 등 제3자 서비스가 광고·보안·세션 유지 목적으로 쿠키를 발행할 수 있으며, 사용자는 브라우저 설정에서 쿠키를 차단·삭제할 수 있습니다.

## 3. 본인의 권리

- 위 제3자 서비스가 보관 중인 정보 조회·삭제는 각 서비스 정책에 따릅니다.
- 본 사이트는 별도로 회원 가입·이메일 수집을 하지 않습니다.

## 4. 콘텐츠 출처

본 사이트의 학습 자료는 YouTube [라이브 아카데미 토들러](https://youtube.com/playlist?list=PLEzsBdrpZXC8tdzAqQHEQ66TocGI-Kagh) 강의의 영상 자막을 기반으로 재구성된 학습 보조 자료입니다. 강의 콘텐츠의 권리는 원작자에게 있습니다.
