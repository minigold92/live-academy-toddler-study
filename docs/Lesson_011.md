# Lesson 011 — 문장 연결 실전: **because / and / so** 응용 (라면 먹은 날 이야기)

> 📺 **원본 영상**: [[LESSON #11] 문맥을 만들어서 자연스러운 말의 흐름 익히기 - and / so / because - PART 2 [기초 영어회화]](https://www.youtube.com/watch?v=wjXyGcnx2G8)
> ⏱ 8분 20초 · 👁 474K회

> 🎯 **오늘의 핵심 포인트**
> 1. **so** = 당연한 흐름의 연결 (배고파서 → 뭔가를 먹어야 했다)
> 2. **and** = 두 상황의 단순 나열 (논리적 필연 관계가 약할 때)
> 3. **because** = 듣는 사람이 예상하는 대안을 하지 않은 이유를 설명할 때
> 4. 같은 상황도 문맥의 흐름에 따라 연결어 선택이 달라짐
> 5. 실전 긴 문장: 배고픔 → 피곤함 → 비 → 결국 라면

> 💡 강사의 한마디:
> "직접 문장을 만들어 보고, 틀려 보고, 다시 고쳐 보는 반복이 없으면 절대로 편하게 쓸 수 없어요."

---

## ✨ 상황 — 배고팠는데 결국 라면 먹은 날 (전체 스토리)

### 장면 1: 배고팠고 뭔가 먹어야 했어 → so (당연한 흐름)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I was hungry** | **배가 고팠어요.** | **← 상황** |
| 2 | 🆕 **so I had to eat something.** | **그래서 뭔가를 먹어야 했어요.** | **← 오늘 #1 (so: 당연한 결과)** |
| 3 | 🆕 **I was hungry so I had to eat something.** | **배가 고파서 뭔가를 먹어야 했어요.** | **← 전체 문장** |

### 장면 2: 피곤해서 뭘 만들고 싶지 않았어 → and (단순 나열)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I was so tired after work** | **일 끝나고 너무 피곤했어요.** | **← 상황 묘사** |
| 2 | 🆕 **and I didn't want to cook anything.** | **그리고 아무것도 만들어 먹고 싶지 않았어요.** | **← 오늘 #2 (and: 단순 나열)** |
| 3 | 🆕 **I was so tired after work and I didn't want to cook anything.** | **일 끝나고 너무 피곤해서 아무것도 해먹고 싶지 않았어요.** | **← 전체 문장** |

### 장면 3: 비 와서 나가기 싫었어 → because (대안 포기의 이유 설명)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I didn't want to go out** | **나가고 싶지 않았어요.** | **← 결과 먼저** |
| 2 | 🆕 **because it was raining outside.** | **밖에 비가 오고 있었기 때문이에요.** | **← 오늘 #3 (because: 이유)** |
| 3 | 🆕 **and I didn't want to go out because it was raining outside.** | **밖에 비가 와서 나가고 싶지 않았어요.** | **← 전체 문장** |

### 장면 4: 그래서 라면 먹었어 → so (최종 반응)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **so I just ate ramen.** | **그래서 그냥 라면을 먹었어요.** | **← 오늘 #4 (so: 최종 대처)** |

### 🧩 문맥으로 이어 말하기 (전체 스토리)

> **I was hungry so I had to eat something. But I was so tired after work and I didn't want to cook anything. And I didn't want to go out because it was raining outside. So I just ate ramen.**
>
> (배가 고파서 뭔가를 먹어야 했는데, 일 끝나고 너무 피곤해서 아무것도 해먹고 싶지 않았어요. 그리고 밖에 비가 와서 나가기도 싫었어요. 그래서 그냥 라면 먹었어요.)

---

## 🔑 오늘의 연결어 선택 기준 정리

```
[연결어 선택 기준]

so       :  당연한 흐름 (배고프면 → 먹어야 함)
            I was hungry so I had to eat something.

and      :  두 상황 단순 나열 (논리적 필연 관계 약함)
            I was so tired after work and I didn't want to cook anything.

because  :  듣는 사람이 예상하는 대안을 안 한 이유를 설명
            I didn't want to go out because it was raining outside.

[전체 구조]
배고픔(상황) → so → 뭔가 먹어야 함(결과)
피곤함(상황) → and → 해먹기 싫음(나열)
비(이유) → because ← 나가기 싫음(결과)
→ so → 라면(최종 대처)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **hungry** | adj. | 배가 고픈 | `I was hungry` |
| **had to** | expr. | ~해야 했다 | `have to`의 과거형 |
| **eat something** | expr. | 뭔가를 먹다 | 부정문: `cook anything` (not something) |
| **so tired** | expr. | 너무 피곤한 | `so` = 정말, 매우 (강조) |
| **after work** | expr. | 일 끝나고, 퇴근 후 | |
| **cook anything** | expr. | 무엇이든 요리하다 | 부정문: `I didn't want to cook anything` |
| **go out** | v. | 나가다, 외출하다 | `I didn't want to go out` |
| **it was raining** | expr. | 비가 오고 있었다 | 과거 진행 표현 |
| **just** | adv. | 그냥, 그저 | `I just ate ramen` — 별 수 없이 그냥 |
| **ramen** | n. | 라면 | |
| **didn't want to** → **didn't wanna** | expr. | ~하고 싶지 않았다 | 구어 연음: `didn't wanna` |

---

## 🗣️ 발음 팁

1. **didn't want to → didn't wanna** — 구어에서 `want to`를 `wanna`로 연음: "디든워나".
2. **so tired** — `so`를 살짝 길게 강조: "쏘우 타이얼드".

---

## 📝 파생 연습 (4문제)

1. **영작**: "배가 고파서 뭔가를 먹어야 했어요." (so 사용)
2. **영작**: "일 끝나고 너무 피곤해서 아무것도 해먹고 싶지 않았어요." (and 사용)
3. **영작**: "밖에 비가 와서 나가고 싶지 않았어요." (because 사용, 결과 먼저)
4. **조합**: 위 세 문장을 but / and로 이어서 전체 스토리 완성.

```

규칙:
- 첫 인사는 당신이 먼저. ("What did you have for dinner last night?" 같은 자연스러운 질문)

- 제가 오늘의 스토리를 영어로 말해볼 수 있도록 자연스럽게 이야기를 끌어내 주세요:

  · I was so tired after work and I didn't want to cook anything.
  · I didn't want to go out because it was raining outside.
  · So I just ate ramen.

10턴 대화 후 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 정리해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 because / and / so 실전 패턴을 제 일상으로 옮겨 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 최근에 계획과 다르게 흘러간 하루 (또는 저녁)
2) 처음에 어떤 상황이었는지 (배고픔 / 피곤함 / 날씨 등)
3) 원래 하려고 했던 것
4) 왜 그걸 못 했는지 (이유)
5) 결국 어떻게 했는지 (결과·대처)

제 답변을 받아 다음을 만들어 주세요:
- so 사용 문장 1개 (당연한 흐름)
- and 사용 문장 1개 (단순 나열)
- because 사용 문장 1개 (이유, 결과 먼저)
- so 사용 결말 문장 1개 (최종 대처)
- 전체를 이어서 하나의 스토리로 구성, 한국어 의역 포함
```
