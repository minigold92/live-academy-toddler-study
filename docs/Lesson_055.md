# Lesson 055 — 동사 happen — 대화 속 "무슨 일이 있었어?" 패턴

> 📺 **원본 영상**: [[ 기초 영어 단어 ] - happen](https://www.youtube.com/watch?v=awlESfrBeh4)
> ⏱ 8분 29초 · 👁 166K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **happen** = 어떤 일이 일어나다, 발생하다
> 2. **What happened?** = 무슨 일이 있었어? / **When did that happen?** = 언제 그랬어?
> 3. 의문사만 바꾸면 응용 가능: `When` → `How` → `Why`
> 4. **I'm sorry** (대화 중) = 사과가 아니라 "뭐라고요?" 확인 표현
> 5. **It's been all over the news.** = 뉴스에서 며칠째 계속 얘기하고 있어. (현재 완료 포함)

## ✨ 상황 2 — 의문사 교체 응용

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **When did that happen?** | **언제 그랬어?** | **← 오늘 #2** |
| 2 | 🆕 **When did this happen?** | **이거 언제 이랬어?** (눈앞에 있을 때) | **← 오늘 #2** |
| 3 | 🆕 **How did that happen?** | **어쩌다가 그랬어?** | **← 오늘 #3 (When → How)** |
| 4 | 🆕 **Why did that happen?** | **왜 그런 일이 일어났어?** | **← 오늘 #3 (When → Why)** |

---

## 🔑 오늘의 패턴 정리

```
[happen 기본 패턴]
What happened?              →  무슨 일이 있었어?
It happened just a few days ago. →  막 며칠 전에 일어난 일이야.

[의문사 교체]
When did that happen?       →  언제 그랬어?
When did this happen?       →  이거 언제 이랬어? (눈앞)
How did that happen?        →  어쩌다가 그랬어?
Why did that happen?        →  왜 그런 일이 일어났어?

[보너스 — 현재 완료]
It's been all over the news. →  며칠째 뉴스에서 계속 얘기하고 있어.
(It's all over the news.    →  지금 뉴스에서 얘기하고 있어. 현재만)
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **happen / happened** | v. | 일어나다, 발생하다 | 주어로 `it` / `that` / `this` 자주 쓰임 |
| **commit suicide** | 표현 | 자살하다 | `commit suicide` = 가장 공식적(formal)인 표현 |
| **all over the news** | 표현 | 뉴스에 가득한, 뉴스에서 계속 얘기하는 | `It's all over the news.` 통째로 암기 |

---

## 🗣️ 발음 팁

1. **I'm sorry, what happened?** — 여기서 `I'm sorry`는 사과가 아님. 억양을 올려서 "뭐라고요?" 뉘앙스로.
2. **It happened** — `it`의 `t`와 `happened`의 `h`가 부드럽게 연결. "잇 해픈드".
3. **You didn't hear?** — 평서문 어순이지만 억양을 올려 의문문처럼. 이미 정보를 들은 상태에서 재확인할 때 이 형태.

---

## 📝 파생 연습 (5문제)

1. **영작**: "무슨 일이 있었어?" (기본형)
2. **영작**: "이거 어쩌다가 이렇게 됐어?" (깨진 물건을 보며)
3. **변환**: `When did that happen?` → `Why`로 바꿔서
4. **영작**: "그 사고는 어제 일어났어."
5. **조합**: "뭐라고? 언제 그랬어? 뉴스에서 며칠째 얘기하고 있던데." 세 문장으로.

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 happen 패턴을 제 일상으로 옮겨 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 최근에 놀랐던 일이 있나요? (뉴스, 주변 사건 등)

3) 어쩌다가 그렇게 됐나요?

5) 그 소식을 어디서 처음 들었나요?

제 답변을 받아 happen 패턴으로 영어 문장을 만들어 주세요.
각 문장에 한국어 의역을 붙이고 마지막에 5문장 표로 정리해 주세요.
