# Lesson 075 — 할 일이 있다/많다/밀렸다 (I have something to do → I have a lot of work to catch up on)

> 📺 **원본 영상**: [단순한 영어 문장 → 섬세한 영어 문장 💡  할 일이 있다 👉🏻 몇 가지만 해결하면 된다](https://www.youtube.com/watch?v=m9DcvA8rm0w)
> ⏱ 7분 40초 · 👁 72K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. 기본 구조 → `I have something to do` (할 게 있다)
> 2. something 자리에 다양한 표현 넣기 → `some things`, `a lot`, `a lot of work`
> 3. "밀린 일이 많다" → `I have a lot of work to catch up on`
> 4. `catch up on` vs `catch up with` — on은 '내용/업무', with는 '사람'
> 5. "몇 가지만 더 끝내면 돼" → `I just have a few more things to take care of`

---

## ✨ 상황 1 — 할 게 있다: something → a lot 변화

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I have something to do.** | **저 할 게 있어요.** | **← 오늘 #1 (단수, 한 가지)** |
| 2 | 🆕 **I have some things to do.** | **할 게 좀 있어요.** | **← 오늘 #2 (복수, 여러 가지)** |
| 3 | 🆕 **I have a lot to do.** | **할 게 많아요.** | **← 오늘 #3** |
| 4 | 🆕 **I have a lot of things to do.** | **할 것들이 많아요.** | **← 오늘 #3** |
| 5 | 🆕 **I have a lot of work to do.** | **할 일이 많아요. (업무)** | **← 오늘 #3 (work = 업무 관련)** |

---

## ✨ 상황 2 — 밀린 일, 끝내야 할 일

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I have a lot of work to catch up on.** | **밀린 일이 많아요.** | **← 오늘 #4 (catch up on)** |
| 2 | 🆕 **I have so much work to catch up on.** | **밀린 일이 정말 너무 많아요.** | **← 오늘 #4 (so much = 강조)** |
| 3 | 🆕 **I just have a few more things to take care of.** | **몇 가지만 더 끝내면 돼요.** | **← 오늘 #5** |

### 🧩 문맥으로 이어 말하기
> **How's the preparation going? — I just have a few more things to take care of.**
> (준비는 어떻게 돼가? — 몇 가지만 더 마무리하면 돼.)

> **She's improved a lot, but she still has a lot of work to do.**
> (그녀는 많이 발전했지만, 아직 더 노력해야 할 부분이 많아.)

---

## 🔑 오늘의 변환 패턴 정리

```
[기본 구조: I have [명사] to [동사]]

something   + do          → I have something to do.       (할 게 있다)
some things + do          → I have some things to do.     (할 게 좀 있다)
a lot       + do          → I have a lot to do.           (할 게 많다)
a lot of things + do      → I have a lot of things to do. (할 것들이 많다)
a lot of work + do        → I have a lot of work to do.   (할 일이 많다, 업무)

[응용: do 자리 바꾸기]
a lot of work + catch up on  → I have a lot of work to catch up on.
                               (밀린 일이 많다)
a few more things + take care of → I just have a few more things to take care of.
                               (몇 가지만 더 끝내면 된다)

[catch up 구분]
catch up on + 내용/업무   → I have a lot of work to catch up on.
catch up with + 사람      → I need to catch up with my friend.
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **something** | 대명사 | 어떤 것 (단수) | 한 가지를 생각하면서 |
| **some things** | 대명사 | 몇 가지 것들 (복수) | 여러 가지를 생각하면서 |
| **catch up on** | 동사구 | (밀린 것을) 따라잡다 | 내용/업무에 사용 |
| **catch up with** | 동사구 | (사람을) 따라잡다/만나다 | 사람에 사용 |
| **take care of** | 동사구 | 처리하다, 해결하다, 마무리하다 | 일 처리; 아이/동물이면 "돌보다" |
| **a few more** | 한정사 | 몇 가지 더 | `just a few more` = 몇 가지만 더 |

## 📝 파생 연습 (5문제)

1. **영작**: "저 할 게 좀 있어요." (복수)
2. **영작**: "밀린 일이 정말 너무 많아요."
3. **영작**: "몇 가지만 더 처리하면 돼요."
4. **구분**: `catch up on`과 `catch up with` 각각 예문 하나씩 만들어 보기
5. **응용**: "그는 많이 발전했지만 아직 노력해야 할 부분이 많아." (He, improve, a lot of work to do 사용)

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: I have [명사] to [동사] 구조 — 할 일 표현하기
- 오늘의 핵심:
  1. 기본: I have something to do. (할 게 있다)
  2. 복수: I have some things to do. (할 게 좀 있다)
  3. 많다: I have a lot to do. / I have a lot of work to do. (할 게 많다/업무가 많다)
  4. 밀린 일: I have a lot of work to catch up on. (catch up on = 내용/업무)
  5. 몇 가지만 더: I just have a few more things to take care of.
- 핵심 예문:
  · I have something to do. (할 게 있어요)
  · I have a lot of work to do. (할 일이 많아요)
  · I have so much work to catch up on. (밀린 일이 정말 너무 많아요)
  · I just have a few more things to take care of. (몇 가지만 더 끝내면 돼요)
  · She's improved a lot, but she still has a lot of work to do. (아직 노력할 부분이 많아)
- 핵심 어휘: something/some things, catch up on(내용), catch up with(사람), take care of, a few more
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ / ❌).
3) 총 10문제. 처음 5개는 레슨 표현 그대로, 나머지 5개는 응용.
4) 마지막에 자주 틀린 패턴 복습 노트.

[레슨 핵심 패턴]
- I have something to do. / I have some things to do.
- I have a lot to do. / I have a lot of work to do.
- I have a lot of work to catch up on. (catch up on = 내용/업무)
- I just have a few more things to take care of.
- catch up on [내용] / catch up with [사람] 구분

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 동료입니다. 우리는 직장에서 점심을 같이 먹으면서 대화 중입니다.

규칙:
- 첫 인사는 당신이 먼저. ("Hey, how's everything going?" 같은 질문)
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 어색한 부분은 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 유도해 주세요:
  · I have a lot of work to do
  · I have a lot of work to catch up on
  · I just have a few more things to take care of

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 "I have [명사] to [동사]" 패턴을 제 실제 상황으로 만들고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 오늘 또는 이번 주에 해야 할 일이 있나요?
2) 요즘 밀린 일이 있나요? (업무, 공부, 집안일 등)
3) 거의 다 끝났는데 조금만 더 남은 일이 있나요?

제 답변을 받은 다음, 아래를 만들어 주세요:
- I have something/some things/a lot to do 문장 2개
- I have a lot of work to catch up on 패턴 문장 1개
- I just have a few more things to take care of 패턴 문장 1개
- 짧은 대화 예시 1개

각 문장에 한국어 의역을 붙여 주세요. 마지막에 5문장 표로 정리해 주세요.
```
