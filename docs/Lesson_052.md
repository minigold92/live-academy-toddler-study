# Lesson 052 — 현재 완료형으로 "이미/아직" 말하기 (already / yet)

> 📺 **원본 영상**: [[ 기초영어회화 ] 완료시제 연습 - "아직/벌써" + 완료시제](https://www.youtube.com/watch?v=8zdLUDXuMHY)
> ⏱ 8분 51초 · 👁 114K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **현재 완료형** = `have/has + 과거분사` — 과거부터 현재까지 시간에 걸친 일을 표현
> 2. **이미·벌써** → `have + already + 과거분사`
> 3. **아직 ~안 했다** → `haven't/hasn't + 과거분사 + yet`
> 4. `already`(이미)와 `yet`(아직)은 같은 맥락의 반대 방향 표현

> 💡 강사의 한마디:
> "어휘든 표현이든 단기간에 완벽히 이해해서 갑자기 잘 쓸 수 있는 게 아니에요. 어떤 상황에서 무슨 얘기를 할 때 쓰는지 감을 익히는 게 중요하고, 그 과정은 오랜 시간에 걸쳐서 점차 잡아가는 거예요."

---

## ✨ 상황 1 — 영화를 벌써 두 번 봤어 (이미 일어난 일)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I've already seen the movie twice.** | **난 그 영화 벌써 두 번 봤어.** | **← 오늘 #1 (already + 현재 완료)** |
| 2 | 🆕 **I haven't seen the movie yet.** | **난 그 영화 아직 안 봤어.** | **← 오늘 #2 (부정 완료 + yet)** |

---

## ✨ 상황 2 — 제품 발표와 정보 누출 (already / yet 조합)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **They haven't officially announced the product yet.** | **그 제품을 아직 공식적으로 발표하지 않았어.** | **← 오늘 #2 (부정 완료 + yet)** |
| 2 | 🆕 **But most of the information has already been leaked.** | **그런데 대부분의 정보가 이미 다 누출됐어.** | **← 오늘 #1 (already + 완료 수동태)** |

### 🧩 문맥으로 이어 말하기
> **They haven't officially announced the product yet, but most of the information has already been leaked.**
> (그 제품 아직 공식 발표도 안 됐는데, 대부분의 정보가 벌써 다 누출됐어.)

---

## 🔑 오늘의 패턴 정리

```
[이미·벌써 — already + 현재 완료]
I've already seen the movie twice.
→ 과거부터 지금까지의 시간 안에 "벌써" 일어난 일

[아직 안 했다 — haven't + yet]
I haven't seen the movie yet.
→ 과거부터 지금까지 아직 일어나지 않은 일

[수동태 + 완료형]
has + been + 과거분사
예: has already been leaked   (이미 누출됐다)
    hasn't been announced yet (아직 발표 안 됐다)
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **already** | adv. | 이미, 벌써 | 완료형 긍정문에서 `have`와 과거분사 사이에 위치 |
| **yet** | adv. | 아직 | 완료형 부정문 끝에 위치 |
| **officially** | adv. | 공식적으로 | `official` + `-ly` |
| **announce** | v. | 발표하다 | `announced` (규칙 과거/과거분사) |
| **leak / leaked** | v. | 누출되다 / 누출하다 | `be leaked` = 누출이 되다 (수동태) |

---

## 🗣️ 발음 팁

1. **I've already** — `I've`는 빠르게 "아이브"로. `already`의 첫 음절 `al-` 강세.
2. **haven't seen** — `haven't`의 `t` 약화. 빠르게 "해븐 씬".
3. **has already been leaked** — `has`가 약화되어 "흐즈". 전체를 한 흐름으로 연음.

---

## 📝 파생 연습 (5문제)

1. **영작**: "나 그 책 벌써 두 번 읽었어." (movie → book 응용)
2. **변환 (부정 + yet)**: "나 아직 저녁 안 먹었어." 한 문장으로.
3. **영작**: "그 게임 아직 공식 출시 안 됐어."
4. **변환 (already)**: `I finished my homework.` → 완료형 + already
5. **조합**: 위 3, 4번을 but으로 연결해 한 문장으로.

<details>
<summary>정답 보기</summary>

1. `I've already read the book twice.`
2. `I haven't eaten dinner yet.`
3. `They haven't officially released the game yet.`
4. `I've already finished my homework.`
5. `They haven't officially released the game yet, but I've already finished my homework.` (예시 — 내용 자유)
</details>

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 4가지를 1줄씩으로 다시 설명해 주세요. (제가 잘못 이해한 부분이 없는지 확인)
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요. 그 후엔 제 질문에 답하면서 추가 예문/유사 표현을 계속 만들어 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 현재 완료형 + already/yet — 이미 일어난 일과 아직 안 일어난 일 표현하기
- 오늘의 핵심:
  1. 현재 완료형: have/has + 과거분사 (과거~현재 시간에 걸친 일)
  2. 이미·벌써: I've already + 과거분사
  3. 아직 안 했다: haven't + 과거분사 + yet
  4. already(이미)와 yet(아직)은 같은 맥락의 반대 방향 표현
- 핵심 예문:
  · I've already seen the movie twice. (벌써 두 번 봤어)
  · I haven't seen the movie yet. (아직 안 봤어)
  · They haven't officially announced the product yet, but most of the information has already been leaked.
- 핵심 어휘: already, yet, officially, announce, leak/leaked
- 발음 팁: I've already 연음, haven't seen의 t 약화, has already been leaked 한 흐름으로
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ 부분정답 / ❌). 잘못된 부분은 어떻게 고쳐야 하는지 짧게 설명.
3) 같은 패턴으로 난이도/응용을 조금씩 바꿔가며 총 10문제.
4) 마지막에 제가 자주 틀린 패턴 1~2가지를 짧은 복습 노트로 정리.

처음 5문제는 레슨에 나온 표현 그대로, 나머지 5문제는 같은 문법 포인트를 다른 단어/상황으로 응용해 주세요.

[레슨 핵심 패턴]
- 이미·벌써: I've already + 과거분사 + 목적어 (예: I've already seen the movie twice.)
- 아직 안 했다: I haven't + 과거분사 + yet (예: I haven't seen the movie yet.)
- 완료 수동태: has/have already been + 과거분사 (예: has already been leaked)
- 활용 어휘: already, yet, officially, announce, leak

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 카페에서 만나서 영어로 잡담하는 중입니다.

규칙:
- 첫 인사는 당신이 먼저. ("Have you seen that new movie yet?" 같은 자연스러운 질문)
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 제가 답하면 어색한 부분은 자연스럽게 다시 말해 주는 형식으로 부드럽게 교정 ("Oh, you mean ~ ?" 같은 식).
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 화제를 유도해 주세요:
  · I've already seen it twice.
  · I haven't tried it yet.
  · They haven't officially announced it yet.
  · Most of the information has already been leaked.

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 already/yet 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 최근에 이미 해본 일 (여행, 음식, 영화 등)
2) 아직 못 해본 일
3) 뉴스나 주변에서 아직 공식 발표가 안 된 것
4) 이미 다 알려진 것 (정보 누출 상황)
5) 오래전부터 지금까지 계속 하고 있는 일

제 답변 5개를 받은 다음, 그것을 재료로 다음을 만들어 주세요:
- I've already + 과거분사 문장
- I haven't + 과거분사 + yet 문장
- has/have already been + 과거분사 문장 (수동태)

각 영어 문장은 한국어 의역을 같이 붙여 주세요. 마지막에 제가 따라 말할 수 있도록 5문장을 표로 정리해 주세요.
```
