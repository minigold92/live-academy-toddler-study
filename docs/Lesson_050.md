# Lesson 050 — "해야 될 거야 / 해야 될 수도 있어" — I'm gonna have to / I might have to

> 📺 **원본 영상**: [[ 기초영어회화 ] "오늘 이걸 못끝내면 주말에 출근해야 될 수도 있어요 😭."](https://www.youtube.com/watch?v=ZWm1kpXs-nY)
> ⏱ 9분 58초 · 👁 131K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. 조건절 앞에서 `have to` 단독 사용보다 **미래/가능성**을 섞는 게 자연스럽다
> 2. **확실한 의무 (조건 만족 시)**: **`I'm gonna have to do something`**
> 3. **불확실한 의무 (그럴 수도)**: **`I might have to do something`**
> 4. 부정 (안 해도 된다): **`I won't have to`** (→ `I'm not gonna have to`보다 음절이 짧아 더 자연스러움)
> 5. **부정 + 불확실**: **`I might not have to do something`**

---

## ✨ 상황 1 — 기초: "이번 주 토요일에 출근해야 돼요"

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | I have to go to work this Saturday. | 저 이번 주 토요일에 출근해야 돼요. | 복습 (have to 현재형) |
| 2 | (On/At this Saturday ❌) this Saturday | 이번 주 토요일에 (전치사 없이) | 도입 (this/last/next + 요일 전치사 불필요) |

---

## ✨ 상황 2 — 조건 + 의무: "못 끝내면 출근해야 될 거야"

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | If I don't finish this today, | 오늘 이걸 못 끝내면 | 도입 (조건절, `if I don't` 권장) |
| 2 | 🆕 **I'm gonna have to go to work this Saturday.** | **이번 주 토요일에 출근해야 될 거야.** | **← 오늘 #1 (미래 + 의무)** |
| 3 | 🆕 **I might have to go to work this Saturday.** | **이번 주 토요일에 출근해야 될 수도 있어.** | **← 오늘 #2 (가능성 + 의무)** |

---

## ✨ 상황 3 — 조건 반전: "끝내면 출근 안 해도 돼"

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | If I finish this today, | 오늘 이걸 끝내면 | 도입 |
| 2 | 🆕 **I won't have to go to work this Saturday.** | **이번 주 토요일에 출근 안 해도 될 거야.** | **← 오늘 #3 (won't have to)** |
| 3 | I'm not gonna have to go to work this Saturday. | (같은 의미, 음절이 길어서 덜 선호) | 도입 (참고) |
| 4 | 🆕 **I might not have to go to work this Saturday.** | **이번 주 토요일에 출근 안 해도 될 수도 있어.** | **← 오늘 #4 (might not have to)** |

---

## 🔑 오늘의 변환 패턴 정리

```
[have to 앞에 미래/가능성 붙이기]

조건 만족 → 반드시 해야 됨    :  I'm gonna have to + 동사
조건 만족 → 그럴 수도 있음    :  I might have to + 동사
조건 만족 → 안 해도 됨 (확실) :  I won't have to + 동사
조건 만족 → 안 해도 될 수도   :  I might not have to + 동사

[전치사 없이 쓰는 시간 표현]
this Saturday  (이번 주 토요일에)
last Saturday  (지난 주 토요일에)
next Saturday  (다음 주 토요일에)
→ on/at 붙이지 않음

[if I don't vs if I can't]
if I don't finish  →  원인·결과 조건 (더 자연스러움)
if I can't finish  →  능력 부재 강조 (이 문맥에서는 덜 어울림)

[won't vs I'm not gonna]
I won't have to     →  더 짧고 자연스럽게 많이 쓰임
I'm not gonna have to  →  음절이 많아 덜 선호
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **I'm gonna have to** | v.구 | ~해야 될 거야 | `going to` + `have to` 결합 |
| **I might have to** | v.구 | ~해야 될 수도 있어 | `might` + `have to` |
| **I won't have to** | v.구 | ~안 해도 될 거야 | `will not` + `have to` 축약 |
| **I might not have to** | v.구 | ~안 해도 될 수도 있어 | `might not` + `have to` |
| **go to work** | v.구 | 출근하다 | `go to work this Saturday` |
| **this/last/next + 요일** | 표현 | 이번/지난/다음 주 요일 | 전치사(on/at) 불필요 |

---

## 🗣️ 발음 팁

1. **I'm gonna have to** — 빠르게 말할 때 "아임 거나 해브 투" → "아임 거나 해브 투" (각 단어 약화됨).
2. **I won't have to** — `won't`의 `t`를 묵음 처리: "아이 원 해브 투". 강사도 항상 묵음 처리한다고 언급.

---

## 📝 파생 연습 (5문제)

1. **영작**: "내가 일찍 떠나지 않으면, 버스를 놓칠 수도 있어." (`I might have to` 는 아니지만 `might` 응용)
2. **영작**: "오늘 야근해야 될 거야." (`I'm gonna have to`)
3. **영작**: "내일 일찍 일어나지 않아도 될 수도 있어." (`I might not have to`)
4. **영작**: "이 보고서를 오늘 안에 끝내면 내일 쉬어도 될 거야." (`I won't have to`)
5. **선택**: `if I can't finish` vs `if I don't finish` — 이 레슨의 문맥에서 어느 쪽이 더 자연스럽나요? 이유는?

<details>
<summary>정답 보기</summary>

1. `If I don't leave early, I might miss the bus.`
2. `I'm gonna have to work late today.`
3. `I might not have to wake up early tomorrow.`
4. `If I finish this report today, I won't have to come in tomorrow.`
5. `if I don't finish` — 능력의 문제가 아니라 완료 여부(원인·결과)에 대한 조건이므로 `don't`가 더 자연스러움
</details>

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 조건절 + 의무 표현 — I'm gonna have to / I might have to / I won't have to
- 오늘의 핵심:
  1. I'm gonna have to + 동사 (조건 만족 시 반드시 해야 됨)
  2. I might have to + 동사 (그럴 수도 있음)
  3. I won't have to + 동사 (안 해도 될 거야 — won't가 더 자연스러움)
  4. I might not have to + 동사 (안 해도 될 수도 있어)
  5. this/last/next + 요일 = 전치사 불필요
- 핵심 예문:
  · If I don't finish this today, I'm gonna have to go to work this Saturday. (못 끝내면 토요일에 출근해야 될 거야)
  · If I don't finish this today, I might have to go to work this Saturday. (그럴 수도 있어)
  · If I finish this today, I won't have to go to work this Saturday. (끝내면 출근 안 해도 될 거야)
  · If I finish this today, I might not have to go to work this Saturday. (안 해도 될 수도 있어)
- 발음 팁: I won't have to — won't의 t 묵음 처리
- 핵심 어휘: I'm gonna have to, I might have to, I won't have to, I might not have to, go to work, this/last/next Saturday
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 채점 (✅ / ⚠️ / ❌). 잘못된 부분은 짧게 설명.
3) 총 10문제. 처음 5문제는 레슨 표현, 나머지 5문제는 응용.
4) 마지막에 자주 틀린 패턴 복습 노트 정리.

[레슨 핵심 패턴]
- If I don't/finish ~, I'm gonna have to + 동사 (반드시 해야 됨)
- If I don't ~, I might have to + 동사 (해야 될 수도 있음)
- If I do/finish ~, I won't have to + 동사 (안 해도 됨)
- If I do/finish ~, I might not have to + 동사 (안 해도 될 수도)
- this/last/next Saturday/Monday/... (전치사 없이)

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 동료입니다. 우리는 직장에서 프로젝트 마감에 대해 이야기하고 있습니다.

규칙:
- 첫 인사는 당신이 먼저. (오늘 마감이나 주말 계획에 대해 자연스럽게 시작)
- 매 턴마다 영어 1~2문장 + (괄호 안에 한국어 의역).
- 어색한 부분은 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 유도:
  · If I don't finish this today, I'm gonna have to ~
  · I might have to ~
  · I won't have to ~ (if I finish)
  · I might not have to ~

10턴 후 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 조건 + 의무 표현을 제 일상 상황으로 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 지금 해야 할 일 (끝내면 좋고, 못 끝내면 어떤 결과가 생기는 것)
2) 그 결과가 생긴다면 어떤 의무/행동이 따르는지
3) 반대로 잘 끝낸다면 하지 않아도 되는 것
4) 이번 주말에 할 수도 있고 안 할 수도 있는 일
5) 다음 주에 해야 할 수도 있는 일

제 답변 5개를 받은 다음:
- If I don't ~, I'm gonna have to ~ (1문장)
- If I don't ~, I might have to ~ (1문장)
- If I ~, I won't have to ~ (1문장)
- If I ~, I might not have to ~ (1문장)
- this/next + 요일 활용 자유 문장 (1문장)

을 만들어 주세요. 각 문장은 한국어 의역과 함께 표로 정리해 주세요.
```
