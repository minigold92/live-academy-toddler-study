# Lesson 038 — **I slept wrong** — "잘못 ~했다" 표현 + Lesson 37 복습 연결

> 📺 **원본 영상**: [[ 기초영어회화 ] "잠을 잘못 잔 것 같아. 목이 너무 아파."](https://www.youtube.com/watch?v=v2pQyesI_qY)
> ⏱ 3분 42초 · 👁 181K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **`I slept wrong`** — 잠을 잘못 잤다 (wrong을 부사처럼 사용)
> 2. **`I think I slept wrong`** — 잠을 잘못 잔 것 같아 (I think + 완성 문장)
> 3. **`I wrote my address wrong`** — 주소를 잘못 썼어요 (동사 + 목적어 + wrong)
> 4. 목적어 대신 대명사: **`I wrote it wrong`** (it 필수, `I wrote wrong` 불가)
> 5. Lesson 37 hurt 표현과 연결: 잘못 잔 것 같아 → 목을 다쳤어 → 정말 아파

## 🔑 오늘의 변환 패턴 정리

```
[잘못 ~하다 — 동사 + (목적어) + wrong]
I slept wrong.               잠을 잘못 잤어요.
I wrote my address wrong.    주소를 잘못 썼어요.
I wrote it wrong.            그걸 잘못 썼어요. (목적어 → 대명사 it)
※ I wrote wrong. (X)  — 목적어나 대명사 반드시 필요

[I think + 완성 문장 — ~인 것 같아요]
I think I slept wrong.       잠을 잘못 잔 것 같아요.
I think I ate too much.      너무 많이 먹은 것 같아요.

[Lesson 37 + 오늘 연결 패턴]
I think I slept wrong. → I hurt my neck. → It really hurts.
(잘못 잔 것 같아)       (목을 다쳤어)     (정말 아파)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **slept** | v. | sleep의 과거형 | `I slept wrong` |
| **wrong** | adv. | 잘못, 틀리게 | 동사 뒤에 붙여 "잘못 ~하다" |
| **address** | n. | 주소 | `my address` |

---

## 🗣️ 발음 팁

1. **I think I slept wrong** — `slept`의 `t`를 끊고 `wrong`으로 이어. "아이 띵크 아이 슬렙ˈ롱".
2. **I wrote it wrong** — `wrote it`에서 `t`와 `it`의 `i` 연음. "아이 로웃ˈ잇 롱".

---

## 📝 파생 연습 (4문제)

1. **영작**: "어젯밤에 잠을 잘못 잔 것 같아요. 어깨가 아파요."
2. **변환**: "이름을 잘못 쓴 것 같아요." (`I think I wrote…`)
3. **응용**: "번호를 잘못 입력한 것 같아요." (`number`, `entered`)
4. **연결**: Lesson 37 + 오늘 패턴으로 4문장 이어서 말해보기.
