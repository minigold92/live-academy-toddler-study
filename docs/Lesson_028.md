# Lesson 028 — 감각동사 Part 2 — `감각동사 + like + 명사` / `감각동사 + like + 완성 문장`

> 📺 **원본 영상**: [감각동사 - PART 2 - look like + 명사 / sound like + 완성형 문장](https://www.youtube.com/watch?v=lo4YD4FASOM)
> ⏱ 19분 30초 · 👁 349K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **감각동사 + like + 명사** → `This looks like real leather` (진짜 가죽 같다)
> 2. **감각동사 + like + 완성 문장** → `It sounds like it was a really serious problem` (심각한 문제 같더라)
> 3. **형용사 vs 명사 구분** → 형용사면 like 없이(`This looks real`), 명사면 like 필요(`This looks like real leather`)
> 4. **kind of** → `It kind of looks like ~` (약간 ~같은데)
> 5. 복습: `Everything looks so good` / `That sounds good` (제안 동의 표현)

---

## ✨ 상황 1 — 감각동사 + like + 명사 (명사 닮음 표현)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 복습 (L.27) | `Everything looks so good.` | 다 너무 맛있어 보인다. |
| 2 | 복습 (L.27) | `That sounds good. / That sounds great.` | 좋아, 그렇게 하자. (제안 동의) |
| 3 | 🆕 **This looks like real leather.** | **이거 진짜 가죽 같다.** | **← 오늘 #1 (like + 명사)** |
| 4 | 🆕 **This looks real.** | **이거 진짜 같다.** | **← 형용사 → like 없이** |
| 5 | 🆕 **This looks weird. It kind of looks like a cigar.** | **이거 희한하게 생겼다. 약간 시가처럼 생겼는데.** | **← 오늘 #1 + kind of** |
| 6 | 🆕 **That looks really convenient.** | **그거 되게 편해 보인다.** | **← 오늘 #1 응용** |

---

## ✨ 상황 2 — 감각동사 + like + 완성 문장

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It sounded like it was a really serious problem.** | **정말 심각한 문제인 것 같았어. (들은 얘기 기준)** | **← 오늘 #2 (과거형)** |
| 2 | 🆕 **It smelled like there was a dead animal in the house.** | **집 안에 죽은 동물이라도 있는 것 같은 냄새가 났어.** | **← 오늘 #2** |
| 3 | 🆕 **This tastes like a professional made it.** | **전문가가 만든 것 같은 맛이 나.** | **← 오늘 #2** |
| 4 | 🆕 **It feels like I walked all day.** | **하루 종일 걸은 기분이야.** | **← 오늘 #2** |
| 5 | 🆕 **It feels like I've been walking all day.** | **하루 종일 걷고 있는 것 같아. (지금도 진행 중)** | **← 현재완료진행 (지금도 계속)** |
| 6 | 🆕 **It looks like it's gonna rain.** | **비 올 것 같아.** | **← 오늘 #2 응용** |

### 🧩 문맥으로 이어 말하기
> **This looks weird. It kind of looks like a cigar.**
> (이거 희한하게 생겼다. 약간 시가처럼 생겼는데.)

> **It sounded like it was a really serious problem.**
> (정말 심각한 문제인 것 같았어.)

---

## 🔑 오늘의 변환/응용 패턴 정리

```
[감각동사 + 형용사 vs 감각동사 + like + 명사]
This looks real.                  →  이거 진짜 같다. (real = 형용사)
This looks like real leather.     →  이거 진짜 가죽 같다. (real leather = 명사)
★ 형용사 → like 없이 / 명사 → like 필요

[kind of — "약간, 좀"]
It kind of looks like a cigar.   →  약간 시가처럼 생겼는데.

[감각동사 + like + 완성 문장]
It sounded like + 문장  →  ~인 것 같았어 (들은 것 기준)
It smelled like + 문장  →  ~인 것 같은 냄새가 났어
It tastes like + 문장   →  ~인 것 같은 맛이 나
It feels like + 문장    →  ~인 것 같은 기분이야
It looks like + 문장    →  ~일 것 같아 (시각적 예측)

[It feels like I walked vs I've been walking]
It feels like I walked all day.         →  하루 종일 걸은 기분 (한참 후 회상)
It feels like I've been walking all day.→  하루 종일 걷는 중인 기분 (지금도 계속)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **leather** | n. | 가죽 | `real leather` = 진짜 가죽 |
| **cigar** | n. | 시가 (담배 종류) | `It kind of looks like a cigar.` |
| **kind of** | adv. | 약간, 좀 | `kind of looks like ~` = 약간 ~같은데 |
| **convenient** | adj. | 편리한 | `That looks really convenient.` |
| **dead** | adj. | 죽은 | `a dead animal` = 죽은 동물 |
| **professional** | n./adj. | 전문가 / 전문적인 | `a professional made it` = 전문가가 만든 |
| **fishy** | adj. | 비린, 수상한 | `Something smells a little fishy.` = 뭔가 수상한데 |

---

## 🗣️ 발음 팁

1. **It kind of looks like a cigar** — `kind of`를 "카인더브" 처럼 빠르게 연음. 전체를 한 호흡으로.
2. **tastes like** — `tastes`의 두 번째 `s` 발음이 까다로움. 억지로 정확히 발음하려 하기보다 살짝 뭉개도 됨.
3. **It looks like it's gonna rain** — `like it's gonna`를 "라이킷스 가나" 처럼 빠르게 연음.

---

## 📝 파생 연습 (5문제)

1. **변환**: "이거 진짜 비싼 가죽 같다." (`looks like` + 명사 표현으로)
2. **영작**: "비 올 것 같아." (`It looks like` + 완성 문장)
3. **영작**: "전문가가 만든 것 같은 냄새가 나." (`smells like` + 완성 문장)
4. **영작**: "하루 종일 일한 기분이야." (`It feels like` 응용)
5. **구분**: "이거 진짜 같다."와 "이거 진짜 가죽 같다."를 각각 영어로.

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 감각동사 + like 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 최근에 뭔가 다른 것처럼 보인 것이 있었나요?
2) 들은 얘기 중 심각하거나 이상하게 느꼈던 것이 있었나요?

5) 오늘 날씨가 어때 보이나요?

- tastes like + 완성 문장 1개
- feels like + 완성 문장 2개

각 영어 문장은 한국어 의역을 같이 붙여 주세요. 마지막에 5문장을 표로 정리해 주세요.
```
