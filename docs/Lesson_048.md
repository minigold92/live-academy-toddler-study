# Lesson 048 — "원래 이렇게 ~해?" 한 문장으로 압축 — Is it always this crowded? / Is it supposed to get this hot?

> 📺 **원본 영상**: [[ 기초영어회화 ] 이거 엄청 어렵다 + 원래 이래? = 이거 원래 이렇게 어려워?](https://www.youtube.com/watch?v=ew_T9ts-ikI)
> ⏱ 15분 05초 · 👁 356K회

> 🎯 **오늘의 핵심 포인트**
> 1. "원래 이래?" + "이렇게 ~해?"를 한 문장으로: **`Is it always this crowded?`**
> 2. 상태 변화에는: **`Is it supposed to get this hot?`** (`get` = 상태 변화)
> 3. **`this + 형용사`** = "이렇게 ~한" (crowded, hot, hard, loud 등)
> 4. 사람 성향 압축: **`Is he always that loud?`**
> 5. 아니요 답변 시: **`No, it's not supposed to do that`** (`do` = action 강조)

> 💡 강사의 한마디:
> "틀렸다고 기분 나빠할 필요 없어요. 중요한 건 얼마나 더 배우는지예요."

## ✨ 상황 1 — 숙제 검사: "원래 이래?" 세 가지 대화

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | It's so crowded here. | 여기 사람 진짜 많다. | 복습 |
| 2 | Is it always like this? | 원래 이래? | 복습 (지난 레슨) |
| 3 | It's always like this in the morning. | 아침에는 원래 이래. | 복습 |
| 4 | Is this normal? | 이게 정상이야? | 도입 (normal 활용) |
| 5 | It gets really hot when I use it for a couple of hours. | 두 시간 정도 사용하면 엄청 뜨거워져요. | 복습 |
| 6 | Is it supposed to do that? | 원래 그래요? (그렇게 반응하는 게 정상이에요?) | 복습 |
| 7 | No, it's not supposed to do that, but it can if you use it in direct sunlight. | 아니요, 원래는 안 그러는데, 직사광선에서 사용하면 그럴 수 있어요. | 복습 |
| 8 | I'm normally not like this. I must be nervous. | 나 원래 안 이러는데. 긴장했나봐. | 복습 |

---

## ✨ 상황 2 — 두 문장을 한 문장으로 압축: this + 형용사

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | It's so crowded here. Is it always like this? | 여기 사람 진짜 많다. 원래 이래? (두 문장) | 복습 |
| 2 | 🆕 **Is it always this crowded here?** | **여기 원래 이렇게 사람이 많아? (압축)** | **← 오늘 #1** |
| 3 | 🆕 **Yeah, it's always this crowded here.** | **어, 여기 원래 이렇게 사람이 많아.** | **← 오늘 #1 답변** |
| 4 | It gets really hot. Is it supposed to do that? | 엄청 뜨거워져요. 원래 이러는 거예요? (두 문장) | 복습 |
| 5 | 🆕 **Is it supposed to get this hot?** | **원래 이렇게 뜨거워지는 거야? (압축)** | **← 오늘 #2** |
| 6 | He's really loud. Is he always like that? | 제 진짜 시끄럽다. 쟤 항상 저래? (두 문장) | 복습 |
| 7 | 🆕 **Is he always that loud?** | **쟤 원래 저렇게 시끄러워? (압축)** | **← 오늘 #3** |

---

## ✨ 상황 3 — "이거 원래 이렇게 어려운 거야?" (be + 형용사 압축)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | This is really hard. Is it supposed to be like this? | 이거 되게 어렵다. 원래 이래? (두 문장) | 도입 |
| 2 | 🆕 **Is this supposed to be this hard?** | **이게 원래 이렇게 어려운 거야? (압축)** | **← 오늘 #4** |

---

## 🔑 오늘의 변환 패턴 정리

```
[두 문장 → 한 문장 압축]
"~하다" + "원래 이래?" → "원래 이렇게 ~해?"

이렇게 혼잡하다 + 원래 이래?
→ Is it always this crowded?

이렇게 뜨거워진다 + 원래 이래?
→ Is it supposed to get this hot?

이렇게 어렵다 + 원래 이래?
→ Is this supposed to be this hard?

저렇게 시끄럽다 + 항상 저래?
→ Is he always that loud?

[핵심 구조]
Is it always   + this + 형용사?   (항상 이렇게 ~해?)
Is it supposed to be + this + 형용사?  (이렇게 ~한 게 정상이야?)
Is it supposed to get + this + 형용사?  (이렇게 ~해지는 게 정상이야?)
Is he always   + that + 형용사?   (항상 저렇게 ~해?)

[do vs be 구분]
is it supposed to be like this?   → 상태가 이런 것이 정상이냐
is it supposed to do this?        → 이런 반응/행동을 하는 것이 정상이냐

[normally = 보통은, 보통 상태에서]
I'm normally not like this.       나 원래 안 이러는데
I must be nervous.                긴장했나봐 (must = ~인가봐, ~임에 틀림없어)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **crowded** | adj. | 붐비는, 사람이 많은 | `It's so crowded here` |
| **this + 형용사** | 구문 | 이렇게 ~한 | `this crowded`, `this hot`, `this hard` |
| **that + 형용사** | 구문 | 저렇게 ~한 | `that loud` — 저 사람/저것에 대해 |
| **get + 형용사** | v.구 | ~해지다 (상태 변화) | `get hot` = 뜨거워지다 (`be hot` = 뜨거운 상태) |
| **normally** | adv. | 보통은, 보통 상태에서는 | `I'm normally not like this` |
| **I must be** | v.구 | ~인가봐, ~임에 틀림없어 | `I must be nervous` ≠ 긴장을 해야만 해 |
| **direct sunlight** | n.구 | 직사광선 | `in direct sunlight` |
| **in direct sunlight** | prep.구 | 직사광선에서, 직사광선 아래에서 | |

---

## 🗣️ 발음 팁

1. **Is it always this crowded?** — `this crowded`에서 `this`의 `s`와 `crowded`의 `cr` 연음에 주의.
2. **I must be nervous** — `must`의 `t`는 약화되어 "머스 비 너버스"처럼 발음.

---

## 📝 파생 연습 (5문제)

1. **압축**: "이 커피 진짜 쓰다. 원래 이렇게 쓴 거야?" → 한 문장으로.
2. **영작**: "여기 항상 이렇게 시끄러워?" (`noisy` 사용)
3. **압축**: "저 사람 진짜 빠르다. 항상 저렇게 빠른 거야?" → 한 문장으로.
4. **영작**: "이 시험 원래 이렇게 어려운 거야?" (`be supposed to be` 사용)
5. **영작**: "나 원래 이렇게 조용한 편이 아닌데, 오늘 좀 피곤한가봐." (`normally`, `must be` 사용)

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: "원래 이렇게 ~해?" 한 문장 압축 — this/that + 형용사
- 오늘의 핵심:
  1. Is it always this crowded? (이렇게 붐비는 게 항상인지)
  2. Is it supposed to get this hot? (이렇게 뜨거워지는 게 정상인지)
  3. Is this supposed to be this hard? (이렇게 어려운 게 정상인지)
  4. Is he always that loud? (사람 성향 — 저렇게 시끄러운 게 항상인지)
  5. I'm normally not like this. I must be nervous. (normally + must be)
- 핵심 예문:
  · Is it always this crowded here? (여기 원래 이렇게 사람 많아?)
  · Yeah, it's always this crowded here. (어, 여기 원래 이렇게 많아)
  · Is it supposed to get this hot? (원래 이렇게 뜨거워지는 거야?)
  · Is this supposed to be this hard? (이게 원래 이렇게 어려운 거야?)
  · Is he always that loud? (쟤 원래 저렇게 시끄러워?)
  · I'm normally not like this. I must be nervous. (나 원래 안 이러는데, 긴장했나봐)
- 핵심 어휘: crowded, this+형용사, that+형용사, get+형용사(상태 변화), normally, I must be, direct sunlight
- 발음 팁: I must be nervous — must의 t 약화
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 함께 어딘가를 방문하거나 새로운 상황을 경험하고 있습니다.

규칙:
- 첫 인사는 당신이 먼저. (주변 상황을 관찰하며 자연스럽게 시작)

- 대화 속에서 아래 표현이 자연스럽게 등장하도록 유도:
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 "원래 이렇게 ~해?" 압축 패턴을 제 일상으로 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 어딘가 방문했는데 이렇게 붐비거나 조용하거나 시끄러웠던 곳
2) 어떤 물건이나 음식이 예상보다 뜨겁거나 차갑거나 매웠던 상황
3) 어떤 것이 예상보다 어렵거나 쉬웠던 경험
4) 주변에 특정 성향(시끄럽다, 조용하다, 빠르다)이 뚜렷한 사람
5) "나 원래 이렇지 않은데"라고 말하고 싶었던 상황

제 답변 5개를 받은 다음:
- Is it always this + 형용사? (1문장)
- Is it supposed to be/get + this + 형용사? (1문장)
- Is he/she always that + 형용사? (1문장)
- I'm normally not like this. I must be ~. (1문장)
- 위 패턴 중 하나를 응용한 자유 문장 (1문장)

을 만들어 주세요. 각 문장은 한국어 의역과 함께 표로 정리해 주세요.
```
