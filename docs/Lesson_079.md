# Lesson 079 — 하고 싶었지만 못했다 (I wanted to..., but..., so I couldn't)

> 📺 **원본 영상**: [영어 실시간 말하기 연습](https://www.youtube.com/watch?v=HWO6USt1n3g)
> ⏱ 117분 53초 · 👁 49K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. 기본 구조 → `I wanted to [동사구] this week, but [변수], so I couldn't.`
> 2. `I wanted to` — "하고 싶었다" (과거 소망)
> 3. `but [변수]` — 뜻대로 되지 않은 이유(장애물)
> 4. `so I couldn't` — "그래서 못했다" (결과)
> 5. 시제 일치 주의 → `I have no time` (❌) → `I had no time` (✅) — 과거 일에는 과거 시제

> 💡 강사의 한마디:
> "I wanted to 다음에 여러분이 하고 싶었던 얘기를 채워 넣으시는 거예요. 근데 변수가 있었죠. 변수에 대해서 but. 하고 싶었는데 but, 문제. 그래서 so I couldn't. 이 문맥 흐름이 정말 중요한 거예요."

---

## ✨ 상황 1 — 하고 싶었지만 변수 때문에 못했다

시청자들이 실제로 이번 주에 경험한 일을 영어로 말하는 라이브 연습 세션. 같은 구조로 다양한 상황을 표현합니다.

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I wanted to make cherry blossom bread this week, but the oven broke down, so I couldn't.** | **이번 주에 벚꽃 빵을 만들고 싶었는데, 오븐이 망가져서 못 했어요.** | **← 오늘 #1** |
| 2 | 🆕 **I wanted to make kimchi this week, but I hurt my back, so I couldn't.** | **이번 주에 김치를 만들고 싶었는데, 허리를 다쳐서 못 했어요.** | **← 오늘 #1** |
| 3 | 🆕 **I wanted to make kimchi this week, but I had no time, so I couldn't.** | **이번 주에 김치를 만들고 싶었는데, 시간이 없어서 못 했어요.** | **← 오늘 #1 (시제 교정)** |

### 🧩 문맥으로 이어 말하기

> **I wanted to make cherry blossom bread this week.**
> **But the oven broke down, so I couldn't.**
> (이번 주에 벚꽃 빵을 만들려고 했는데, 오븐이 망가져서 못 했어요.)

> **I wanted to make kimchi this week.**
> **But I hurt my back, so I couldn't.**
> (이번 주에 김치를 만들려고 했는데, 허리를 다쳐서 못 했어요.)

---

## ✨ 상황 2 — 원하지 않지만 해야 할 수도 있다 (I really don't want to... but I might have to)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I really don't want to go hiking this weekend.** | **이번 주말에 진짜 등산하러 가고 싶지 않아요.** | **← 오늘 #2** |
| 2 | 🆕 **I really don't want to go hiking this weekend, but if the weather is good, I might have to.** | **이번 주말에 진짜 등산 가기 싫은데, 날씨가 좋으면 가야 될 수도 있어요.** | **← 오늘 #2** |

### 🧩 대화 속 생략 — 질문에 답할 때 반복 부분 생략

> **A: Do you have to go hiking this weekend?**
> (이번 주말에 등산하러 가야 돼?)
>
> **B: I really don't want to.**
> (진짜 가기 싫어요. — "go hiking this weekend" 생략)

## 🔑 오늘의 패턴 정리

```
[기본 구조]
I wanted to + [동사구] + this week,
  but + [변수/장애물],
  so I couldn't.

예)
I wanted to make cherry blossom bread this week,
  but the oven broke down,
  so I couldn't.

I wanted to make kimchi this week,
  but I hurt my back,
  so I couldn't.

[시제 일치 — 중요!]
I wanted to make kimchi, but I have no time. (❌)
I wanted to make kimchi, but I had no time. (✅)
→ I wanted to (과거)에 맞춰 변수도 과거 시제로

[반대 상황 — 하고 싶지 않지만 해야 할 수도]
I really don't want to + [동사구],
  but if [조건],
  I might have to.

예)
I really don't want to go hiking this weekend,
  but if the weather is good,
  I might have to.

[대화 속 생략]
Q: Do you have to go hiking this weekend?
A: I really don't want to. (← go hiking this weekend 생략 가능)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **I wanted to** | 동사구 | 하고 싶었다 | 과거 소망 — wanted의 발음: "완티드 투" → "워너" |
| **break down** | 동사구 | 고장나다, 망가지다 | the oven broke down = 오븐이 고장났다 |
| **hurt one's back** | 동사구 | 허리/등을 다치다 | I hurt my back (과거형도 hurt) |
| **I had no time** | 표현 | 시간이 없었다 | have no time (현재) → had no time (과거) |
| **I might have to** | 표현 | 해야 될 수도 있다 | 어쩔 수 없이 할 가능성 |
| **so I couldn't** | 표현 | 그래서 못 했다 | 결과를 나타내는 so + couldn't |

---

## 🗣️ 발음 팁

1. **I wanted to** — 빠르게 말할 때 "아이 워너"처럼. `wanted to` → `wanna`로 줄어들기도 함.
2. **the oven broke down** — `broke down`을 한 덩어리로. "브로욱 다운"처럼 연음.
3. **so I couldn't** — `couldn't`의 t를 거의 발음 안 함. "쿠든"처럼.
4. **I might have to** — `might have to`를 빠르게. "마잇 해브 투" → "마이 해브 투"처럼.
5. **Do you have to?** — 빠르게 말하면 `Do you`가 "듀"처럼 들릴 수 있음.

## 📝 파생 연습 (5문제)

괄호 안의 힌트를 사용해서 문장을 완성하세요.

1. **영작**: "이번 주에 운동하러 가고 싶었는데, 비가 와서 못 갔어요." (work out, it rained)
2. **시제 교정**: `I wanted to call her, but I have no phone.` → 올바른 문장으로
3. **영작**: "이번 주말에 진짜 청소하기 싫은데, 손님이 오면 해야 될 수도 있어요." (clean, guests come)
4. **생략 연습**: `Do you have to work this weekend?` → "I really don't want to."로 자연스럽게 답하기
5. **영작**: "이번 주에 친구를 만나고 싶었는데, 일이 너무 많아서 못 만났어요." (meet, too much work)

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 친한 친구입니다. 우리는 주말에 카페에서 만나서 이번 한 주를 어떻게 보냈는지 이야기하고 있습니다.

규칙:
- 첫 인사는 당신이 먼저. ("How was your week?" 같은 질문)

- 대화 속에서 아래 표현이 자연스럽게 등장하도록 유도해 주세요:

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 "I wanted to..., but..., so I couldn't" 패턴을 제 실제 경험으로 만들고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 이번 주에 하려고 했는데 못한 일이 있나요? (운동, 요리, 약속, 공부 등)
2) 그걸 못하게 된 이유(변수)는 뭐였나요?
3) 반대로 하고 싶지 않은데 어쩔 수 없이 해야 할 것 같은 일이 있나요?

제 답변을 받은 다음, 아래를 만들어 주세요:
- I wanted to..., but..., so I couldn't. 문장 2개
- I really don't want to..., but if..., I might have to. 문장 1개
- 짧은 대화 예시 1개 (Q: Do you have to...? → A: I really don't want to.)

각 문장에 한국어 의역을 붙여 주세요. 마지막에 4문장 표로 정리해 주세요.
```
