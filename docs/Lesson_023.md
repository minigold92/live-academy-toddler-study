# Lesson 023 — "예전에는 ~했었는데, 지금은 ~해" — `used to` + `hardly ~ anymore` 패턴

> 📺 **원본 영상**: [[ LESSON #21 ] - 예전에는 ~~ 었는데 지금은 ~~ ](https://www.youtube.com/watch?v=F8vUB75aD34)
> ⏱ — · 👁 746회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **예전에는 ~했었어** → `I used to + 동사` (과거 습관/상태 표현)
> 2. **지금은 그럴 시간이 없어** → `but now I just don't have the time`
> 3. **이제 거의 ~하지 않아** → `I hardly + 동사 + anymore`
> 4. **be동사 문장의 used to** → `used to be` (was/were → be 원형)
> 5. **더 이상 ~하지 않아** → `it's not so + 형용사 + anymore`

---

## ✨ 상황 1 — 예전엔 규칙적으로 운동했는데, 지금은 못 해

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I used to exercise quite regularly.** | **예전에는 꽤 규칙적으로 운동했었어요.** | **← 오늘 #1 (과거 습관)** |
| 2 | 🆕 **But now I just don't have the time.** | **근데 지금은 그럴 시간이 없어요.** | **← 오늘 #2** |
| 3 | 🆕 **I hardly exercise anymore.** | **이제 거의 운동을 안 해요.** | **← 오늘 #3** |

### 🧩 문맥으로 이어 말하기
> **I used to exercise quite regularly. But now I just don't have the time. I hardly exercise anymore.**
> (예전에는 꽤 규칙적으로 운동했었는데, 지금은 그럴 시간이 없어서 거의 운동을 안 해요.)

---

## ✨ 상황 2 — 예전엔 극장에 자주 갔었는데, 지금은 TV로 봐

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I used to go to the movies quite often.** | **예전에는 영화를 보러 극장에 꽤 자주 갔었어요.** | **← 오늘 #1 응용** |
| 2 | 🆕 **But now I just watch movies on my television or on my computer.** | **근데 지금은 그냥 TV나 컴퓨터로 영화를 봐요.** | **← 오늘 #2 응용** |
| 3 | 🆕 **It's just more convenient.** | **그냥 그게 더 편해요.** | **← 인상 표현** |

### 🧩 문맥으로 이어 말하기
> **I used to go to the movies quite often. But now I just watch movies on my television or on my computer. It's just more convenient.**
> (예전에는 극장에 꽤 자주 갔었는데, 지금은 그냥 TV나 컴퓨터로 봐요. 그냥 그게 더 편하거든요.)

---

## ✨ 상황 3 — 이 동네, 예전엔 인기 많았는데 지금은 아니야

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **This area used to be really popular.** | **이 동네는 예전에 정말 인기가 많았어요.** | **← 오늘 #4 (used to be)** |
| 2 | 🆕 **But I don't know why.** | **근데 왜 그런지 모르겠어요.** | **← 도입** |
| 3 | 🆕 **Now it's not so popular anymore.** | **지금은 더 이상 그렇게 인기가 많지 않아요.** | **← 오늘 #5** |

### 🧩 문맥으로 이어 말하기
> **This area used to be really popular. But I don't know why, now it's not so popular anymore.**
> (이 동네 예전에는 정말 인기가 많았는데, 왜 그런지 모르겠지만 지금은 더 이상 그렇게 인기가 없어요.)

---

## 🔑 오늘의 변환/응용 패턴 정리

```
[used to — 과거 습관/상태]
일반동사:  I used to exercise.              →  예전에는 운동했었어.
be동사:    This area used to be popular.    →  예전에 이 동네는 인기 많았어.
                        ↑ was → be (원형으로!)

[지금과의 대조]
예전:  I used to go to the movies.
지금:  But now I just watch movies on my television.

[이제 거의/더 이상 안 해]
I hardly exercise anymore.          →  이제 거의 운동 안 해.
It's not so popular anymore.        →  더 이상 그렇게 인기 없어.
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **used to** | 조동사구 | 예전에는 ~했었다 | 현재는 그렇지 않다는 뉘앙스 포함 |
| **quite** | adv. | 꽤, 상당히 | `quite regularly` = 꽤 규칙적으로 |
| **regularly** | adv. | 규칙적으로 | 발음 어려움 — 반복 연습 필요 |
| **hardly** | adv. | 거의 ~하지 않다 | 부정 부사 (≠ hard) |
| **anymore** | adv. | 더 이상 (~하지 않다) | 부정문/감소 문맥에서 사용 |
| **convenient** | adj. | 편리한 | `more convenient` = 더 편한 |
| **popular** | adj. | 인기 있는 | `used to be popular` = 예전엔 인기 많았어 |

---

## 🗣️ 발음 팁

1. **regularly** — 발음이 까다로운 단어. "레귤러리" 처럼 들리며, `r` 소리를 살리면서 천천히 반복 연습 필요.
2. **used to** — `used`의 `d`가 거의 탈락. "유즈투" 처럼 연음.
3. **I just don't have the time** — `just`가 "그저/단지"의 느낌을 살려줌. 빠르게 하면 "아이 저스돈헤브더타임" 처럼 연음.

---

## 📝 파생 연습 (5문제)

1. **변환**: `I go to bed late.` → "예전에는 ~했었어" 형태로.
2. **영작**: "예전에는 이 동네에 카페가 많았어요." (`used to be` 응용)
3. **조합**: "지금은 거의 극장을 안 가요." (`hardly ~ anymore` 응용)
4. **영작**: "이 앱 예전엔 엄청 인기 많았는데." (`used to be` 응용)
5. **변환**: "근데 왜 그런지 모르겠어요." 한 문장으로.

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요. (제가 잘못 이해한 부분이 없는지 확인)
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요. 그 후엔 제 질문에 답하면서 추가 예문/유사 표현을 계속 만들어 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: used to (과거 습관/상태) + hardly ~ anymore + not so ~ anymore
- 오늘의 핵심:
  1. I used to + 동사 (예전에는 ~했었어)
  2. But now I just don't have the time (지금은 그럴 시간이 없어)
  3. I hardly + 동사 + anymore (이제 거의 안 해)
  4. used to be (be동사일 때: was → be 원형)
  5. It's not so + 형용사 + anymore (더 이상 그렇게 ~하지 않아)
- 핵심 예문:
  · I used to exercise quite regularly. (예전엔 꽤 규칙적으로 운동했었어요.)
  · But now I just don't have the time. (지금은 그럴 시간이 없어요.)
  · I hardly exercise anymore. (이제 거의 운동 안 해요.)
  · I used to go to the movies quite often. (예전엔 극장에 자주 갔었어요.)
  · But now I just watch movies on my television or on my computer. (지금은 TV나 컴퓨터로 봐요.)
  · It's just more convenient. (그냥 더 편해요.)
  · This area used to be really popular. (이 동네 예전엔 인기 많았어요.)
  · Now it's not so popular anymore. (지금은 더 이상 그렇게 인기 없어요.)
- 핵심 어휘: used to, quite, regularly, hardly, anymore, convenient, popular
- 발음 팁: regularly 발음 연습, used to 연음
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ 부분정답 / ❌). 잘못된 부분은 어떻게 고쳐야 하는지 짧게 설명.
3) 같은 패턴으로 난이도/응용을 조금씩 바꿔가며 총 10문제.
4) 마지막에 제가 자주 틀린 패턴 1~2가지를 짧은 복습 노트로 정리.

처음 5문제는 레슨에 나온 표현 그대로, 나머지 5문제는 같은 문법 포인트를 다른 단어/상황으로 응용해 주세요.

[레슨 핵심 패턴]
- 과거 습관: I used to + 동사 (예: I used to exercise quite regularly.)
- be동사 과거 상태: 주어 + used to be + 형용사/명사 (예: This area used to be popular.)
- 지금은 안 해: I hardly + 동사 + anymore (예: I hardly exercise anymore.)
- 더 이상 아니야: It's not so + 형용사 + anymore
- 활용 어휘: exercise, go to the movies, popular, convenient, regularly, hardly, anymore

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 카페에서 만나서 영어로 잡담하는 중입니다.

규칙:
- 첫 인사는 당신이 먼저. ("Hey, do you still work out?" 같은 자연스러운 질문)
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 제가 답하면 어색한 부분은 자연스럽게 다시 말해 주는 형식으로 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 화제를 유도해 주세요:
  · I used to + 동사 (예전엔 ~했었어)
  · But now I just don't have the time
  · I hardly ~ anymore
  · used to be + 형용사
  · It's not so ~ anymore

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 예전에 자주 하던 취미나 활동이 있나요?
2) 지금은 왜 그걸 안 하나요? (시간, 돈, 관심 등)
3) 지금은 그걸 얼마나 자주 하나요?
4) 예전에 좋아하거나 자주 가던 장소가 있나요?
5) 그 장소가 지금은 어떤가요?

제 답변 5개를 받은 다음, 그것을 재료로 다음을 만들어 주세요:
- I used to ~ 문장 2개
- But now I just ~ 문장 1개
- I hardly ~ anymore 문장 1개
- 장소/상태 used to be ~ 문장 1개

각 영어 문장은 한국어 의역, 발음 팁(필요하면)을 같이 붙여 주세요. 마지막에 제가 따라 말할 수 있도록 5문장을 표로 정리해 주세요.
```
