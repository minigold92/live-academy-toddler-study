# Lesson 016 — **something / someone / somewhere** + 형용사로 막연하게 말하기

> 📺 **원본 영상**: [[ LESSON #16 ] something / somewhere / someone - 영어회화 기초 [라이브아카데미 토들러]](https://www.youtube.com/watch?v=hyHjOMAPmMs)
> ⏱ 4분 59초 · 👁 347K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **뭔가** → `something`, **누군가** → `someone`, **어딘가** → `somewhere`
> 2. 구체적으로 말하지 않아도 자연스럽게 대화할 수 있는 핵심 단어들
> 3. **something / someone / somewhere 뒤에 형용사**를 붙이면 묘사 추가 가능
> 4. **비교급도 뒤에 가능**: `something lighter` (더 가벼운 것)
> 5. 나중에 관계절(`something that ~`)로 더 구체적으로 확장 가능 (예고)

---

## ✨ 상황 1 — 막연하게 필요 / 원하는 것

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I need something.** | **나 뭔가 좀 필요해.** | **← 오늘 #1** |
| 2 | 🆕 **I wanna eat something.** | **나 뭔가 좀 먹고 싶어.** | **← 오늘 #1** |
| 3 | 🆕 **I have to go somewhere.** | **나 어디 좀 가야 돼.** | **← 오늘 #1** |
| 4 | 🆕 **I have to meet someone.** | **나 누구 좀 만나야 돼.** | **← 오늘 #1** |

---

## ✨ 상황 2 — 형용사를 붙여서 묘사 추가

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I need something light.** | **나 뭔가 가벼운 게 필요해.** | **← 오늘 #3 (something + 형용사)** |
| 2 | 🆕 **I need something lighter.** | **나 뭔가 더 가벼운 게 필요해.** | **← 오늘 #4 (비교급)** |
| 3 | 🆕 **I want to eat something sweet.** | **나 뭔가 좀 단 게 먹고 싶어.** | **← 오늘 #3** |
| 4 | 🆕 **I want to eat something really spicy.** | **나 진짜 매운 게 좀 먹고 싶어.** | **← 오늘 #3** |
| 5 | 🆕 **I want to go somewhere quiet.** | **나 어딘가 조용한 데 가고 싶어.** | **← 오늘 #3 (somewhere + 형용사)** |

## 🔑 오늘의 패턴 정리

```
[기본]
I need something.         나 뭔가 필요해.
I wanna eat something.    나 뭔가 먹고 싶어.
I have to go somewhere.   나 어디 가야 돼.
I have to meet someone.   나 누구 만나야 돼.

[something / somewhere + 형용사]
I need something light.           나 뭔가 가벼운 게 필요해.
I need something lighter.         나 뭔가 더 가벼운 게 필요해.  (비교급)
I want to eat something sweet.    나 뭔가 단 게 먹고 싶어.
I want to eat something spicy.    나 뭔가 매운 게 먹고 싶어.
I want to go somewhere quiet.     나 조용한 데 가고 싶어.

[형용사 위치 주의]
영어:  something + 형용사  (형용사가 뒤에 옴)
한국어: 형용사 + 것/데 (형용사가 앞에 옴)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **something** | pron. | 뭔가, 어떤 것 | 뒤에 형용사 붙이기 가능 |
| **someone** | pron. | 누군가 | = somebody |
| **somewhere** | adv. | 어딘가에 | 뒤에 형용사 붙이기 가능 |
| **light** | adj. | 가벼운 | `something light` |
| **sweet** | adj. | 단 | `something sweet` |
| **spicy** | adj. | 매운 | `something spicy` |
| **quiet** | adj. | 조용한 | `somewhere quiet` |

## 📝 파생 연습 (4문제)

1. **영작**: "나 뭔가 따뜻한 거 마시고 싶어."
2. **영작**: "나 어딘가 따뜻한 데 가고 싶어."
3. **영작**: "나 뭔가 더 저렴한 거 필요해." (cheap → cheaper 비교급 사용)
4. **영작**: "나 아무나 만나야 돼." (아무나 = someone)

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: something / someone / somewhere + 형용사로 막연하게 말하기
- 오늘의 핵심:
  1. something(뭔가) / someone(누군가) / somewhere(어딘가)로 구체적 정보 없이 자연스럽게 말하기
  2. 뒤에 형용사를 붙여서 묘사 추가 (something light, somewhere quiet)
  3. 비교급도 가능 (something lighter)
  4. 영어에서는 형용사가 something 뒤에 옴 (한국어와 어순 반대)
- 핵심 예문:
  · I need something. (나 뭔가 필요해)
  · I wanna eat something. (나 뭔가 먹고 싶어)
  · I have to go somewhere. (나 어디 가야 돼)
  · I have to meet someone. (나 누구 만나야 돼)
  · I need something light. (나 뭔가 가벼운 게 필요해)
  · I need something lighter. (나 뭔가 더 가벼운 게 필요해)
  · I want to eat something sweet. (나 뭔가 단 게 먹고 싶어)
  · I want to go somewhere quiet. (나 조용한 데 가고 싶어)
- 핵심 어휘: something, someone, somewhere, light/lighter, sweet, spicy, quiet
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ / ❌).
3) 총 10문제.
4) 마지막에 자주 틀린 패턴 복습 노트 정리.

[레슨 핵심 패턴]
- I need/want something. / I have to go somewhere. / I have to meet someone.
- something / somewhere + 형용사 (형용사가 뒤에 옴)
- 비교급 활용: something + 형용사 비교급
- 활용 형용사: light/lighter, sweet, spicy, quiet, warm, cold, cheap/cheaper, good/better

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 점심을 먹으러 어디 갈지 고민하고 있습니다.

규칙:
- 첫 인사는 당신이 먼저.
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 어색한 부분은 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 화제를 유도해 주세요:
  · I want to eat something warm / spicy / light
  · I want to go somewhere quiet
  · I need something lighter
  · I have to meet someone later
  · I want to eat something really spicy

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 지금 뭔가 먹고 싶은 것이 있나요? (어떤 느낌의 음식인지)
2) 지금 어디 가고 싶은 곳이 있나요? (어떤 곳인지)
3) 지금 만나야 할 사람이 있나요?
4) 지금 필요한 것이 있나요? (어떤 특성이 있는지)
5) 더 ~~한 것이 필요한 상황이 있나요? (비교급 상황)

제 답변을 받은 다음, 오늘 배운 something/someone/somewhere 패턴으로 각각 영어 문장을 만들어 주세요.
한국어 의역과 함께 마지막에 5문장을 표로 정리해 주세요.
```
