# Lesson 076 — A뿐만 아니라 B도 (Not only A but also B)

> 📺 **원본 영상**: [학생 때 배워서 다 아는 영어? 🤔 NOT ONLY A BUT ALSO B](https://www.youtube.com/watch?v=RtTe8SDH84Y)
> ⏱ 16분 51초 · 👁 120K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. 기본 구조 → `Not only A but also B` (A뿐만 아니라 B도)
> 2. 세 가지 사용 맥락: ① 어떤 것/장소/사람이 A이면서 B이기도 함 ② 같은 행동을 두 대상에 함 ③ 다른 두 가지 행동을 함
> 3. `also` 대신 완성형 문장 사용 가능 → `This is not only my bedroom but it's also my office`
> 4. 강조 도치형 → `Not only is this my bedroom, but it's also my office` (not only를 문장 앞으로)
> 5. 도치형에서 뒷부분은 반드시 완성형 문장으로

---

## ✨ 상황 1 — 어떤 것/사람이 A이면서 B이기도 함

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **This is not only my bedroom but also my office.** | **여기는 제 침실일 뿐만 아니라 제 사무실이기도 해요.** | **← 오늘 #1** |
| 2 | 🆕 **This is not only my bedroom but it's also my office.** | **여기는 제 침실이기도 하지만, 제 사무실이기도 해요.** | **← 오늘 #1 (it's 추가형)** |
| 3 | 🆕 **Russell Crowe is not only a famous Hollywood actor but also a very talented musician and singer.** | **러셀 크로우는 유명한 할리우드 배우일 뿐만 아니라 재능 있는 뮤지션이자 가수이기도 해요.** | **← 오늘 #1** |
| 4 | 🆕 **September 21st is not only my birthday but also my wedding anniversary.** | **9월 21일은 제 생일일 뿐만 아니라 제 결혼기념일이기도 해요.** | **← 오늘 #1** |

---

## ✨ 상황 2 — 같은 행동을 두 대상에 함

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I invited not only my friends but also a few co-workers.** | **저는 친구들뿐만 아니라 직장 동료 몇 명도 초대했어요.** | **← 오늘 #2** |
| 2 | 🆕 **I deleted not only the copied file but also the original.** | **저는 복사본만 삭제한 게 아니라 원본까지 지워버렸어요.** | **← 오늘 #2** |

---

## ✨ 상황 3 — 다른 두 가지 행동을 함

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **He not only starred in that show but also directed a few episodes.** | **그는 그 TV쇼에 출연만 한 게 아니라 몇 편을 감독하기도 했어요.** | **← 오늘 #3** |
| 2 | 🆕 **In 2010, she not only won the gold medal but also set a new world record.** | **2010년에 그녀는 금메달을 따기만 한 게 아니라 세계 신기록도 세웠어요.** | **← 오늘 #3** |

---

## ✨ 상황 4 — 강조 도치형 (not only를 문장 앞으로)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **Not only is this my bedroom, but it's also my office.** | **여기가 제 침실이기만 한 게 아니라, 제 사무실이기도 합니다.** | **← 오늘 #4 (강조 도치)** |
| 2 | 🆕 **In 2010, not only did she win the gold medal, but she also set a new world record.** | **2010년에, 금메달을 따기만 한 게 아니라, 세계 신기록도 세웠어요.** | **← 오늘 #4 (강조 도치)** |

## 🔑 오늘의 패턴 정리

```
[기본형]
주어 + be동사 + not only A but also B
→ This is not only my bedroom but also my office.

[동사 포함형]
주어 + 동사 + not only A but also B
→ I invited not only my friends but also a few co-workers.
→ He not only starred in the show but also directed a few episodes.

[it's 완성형]
This is not only my bedroom but it's also my office.
September 21st is not only my birthday but it's also my wedding anniversary.

[강조 도치형 — not only를 문장 앞으로]
Not only + 조동사 + 주어 + 동사 ..., but 주어 + also + 동사 ...
Not only is this my bedroom, but it's also my office.
Not only did she win the gold medal, but she also set a new world record.
                   ↑ 주의: 도치 후 뒷절은 반드시 완성형 문장

[핵심 어휘]
star (v.)        → 출연하다 (영화/미드에 주인공으로 나오다)
direct (v.)      → 감독하다
set a record     → 기록을 세우다
win the gold medal → 금메달을 따다
the copied file  → 복사된 파일 (복사본)
the original     → 원본
co-worker        → 직장 동료
talented         → 재능 있는
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **talented** | adj. | 재능 있는 | talented musician = 재능 있는 뮤지션 |
| **star (in)** | v. | (작품에) 출연하다 | He starred in the show. |
| **direct** | v. | 감독하다 | directed a few episodes |
| **set a new world record** | 동사구 | 세계 신기록을 세우다 | 통째로 외우기 |
| **the original** | n. | 원본 | ↔ the copied file (복사본) |
| **co-worker** | n. | 직장 동료 | a few co-workers |
| **wedding anniversary** | n. | 결혼기념일 | anniversary = 기념일 |

## 📝 파생 연습 (5문제)

1. **영작**: "이 카페는 커피숍일 뿐만 아니라 서점이기도 해요." (bookstore)
2. **영작**: "저는 친구들뿐만 아니라 가족도 초대했어요."
3. **변환**: `She won the gold medal and set a world record.` → `not only A but also B` 패턴으로
4. **강조 도치**: `This is not only my bedroom but also my office.` → 강조 도치형으로
5. **구분**: 강조 도치형에서 뒷절은 어떤 형태여야 하는가?

<details>
<summary>정답 보기</summary>

1. `This cafe is not only a coffee shop but also a bookstore.`
2. `I invited not only my friends but also my family.`
3. `She not only won the gold medal but also set a new world record.`
4. `Not only is this my bedroom, but it's also my office.`
5. 완성형 문장 (주어 + 동사가 갖춰진 절) — `but also my office`처럼 명사만 쓰면 안 됨.

## 🤖 AI와 함께 공부하기

ChatGPT / Gemini / Claude 등 어떤 AI에든 **아래 블록 전체를 그대로 복사해서 붙여넣으면** 그 모드로 학습 세션이 시작됩니다.

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: Not only A but also B — A뿐만 아니라 B도
- 오늘의 핵심:
  1. 기본형: This is not only A but also B. / He not only did A but also did B.
  2. 세 가지 맥락: ① 어떤 것이 A이면서 B ② 같은 행동을 두 대상에 ③ 다른 두 행동을 함
  3. it's 완성형: but it's also my office (also 앞에 it's 추가 가능)
  4. 강조 도치형: Not only is this my bedroom, but it's also my office.
  5. 도치형 뒷절은 반드시 완성형 문장 (not only is this my bedroom, but also my office ❌)
- 핵심 예문:
  · This is not only my bedroom but also my office. (침실이자 사무실)
  · I invited not only my friends but also a few co-workers. (같은 행동, 두 대상)
  · He not only starred in the show but also directed a few episodes. (다른 두 행동)
  · In 2010, she not only won the gold medal but also set a new world record.
  · Not only is this my bedroom, but it's also my office. (강조 도치)
- 핵심 어휘: talented, star(v.), direct(v.), set a new world record, the original, co-worker
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ / ❌).
3) 총 10문제. 처음 5개는 레슨 표현 그대로, 나머지 5개는 응용.
4) 마지막에 자주 틀린 패턴 복습 노트.

[레슨 핵심 패턴]
- Not only A but also B (기본형, 3가지 맥락)
- but it's also + 문장 (완성형 추가)
- 강조 도치: Not only + 조동사 + 주어 + 동사 ..., but 주어 + also + 동사 ...
- 활용 어휘: talented, star in, direct, set a record, co-worker, the original

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 카페에서 만나서 여러 사람들 / 장소 / 사건에 대해 얘기하고 있습니다.

규칙:
- 첫 인사는 당신이 먼저.
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 어색한 부분은 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 유도해 주세요:
  · not only A but also B (기본형)
  · not only A but it's also B (완성형)
  · Not only + 도치형 (강조)

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 "Not only A but also B" 패턴을 제 일상 상황으로 만들고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 주변에 두 가지 역할을 하는 장소나 물건이 있나요? (예: 거실이 공부방이기도 함)
2) 최근에 두 가지 일을 동시에 한 경험이 있나요?
3) 존경하는 사람이 두 가지 이상의 재능이나 업적을 가진 경우가 있나요?

제 답변을 받은 다음, 아래를 만들어 주세요:
- not only A but also B (기본형) 문장 2개
- not only A but it's also B (완성형) 문장 1개
- 강조 도치형 문장 1개

각 문장에 한국어 의역을 붙여 주세요. 마지막에 5문장 표로 정리해 주세요.
```
