# Lesson 036 — **a lot of** vs **a lot** — "뭐뭐가 많다" vs "뭐뭐를 많이 한다"

> 📺 **원본 영상**: [(무엇)이 많다 / (무엇)을 많이 한다 - "a lot of"](https://www.youtube.com/watch?v=HsmLR7rZtTg)
> ⏱ 9분 31초 · 👁 155K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **`There is/are a lot of + 명사`** — ~이 많다 (셀 수 없으면 is, 셀 수 있으면 are)
> 2. **`I/We have a lot of + 명사`** — 나/우리에게 ~이 많다
> 3. **`동사 + a lot`** — ~을 많이 한다 (부사 용법, of 없음)
> 4. 목적어가 있을 때는 **`동사 + 목적어 + a lot`** 또는 **`동사 + a lot of + 목적어`** 둘 다 가능
> 5. `many/much` 대신 `a lot of`를 쓰면 셀 수 있는지 없는지 따질 필요 없어서 편하다

> 💡 강사의 한마디:
> "many, much 쓰다가 실수하는 것보다, a lot of 하나 잘 쓰는 게 훨씬 낫아요."

---

## ✨ 상황 1 — 뭐뭐가 많다 (there is/are, have)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | There are a lot of people. | 사람이 많아요. | 복습 (셀 수 있는 복수 → are) |
| 2 | There is a lot of traffic. | 교통량이 많아요. | 복습 (셀 수 없음 → is) |
| 3 | There is a lot of time. | 시간이 많아요. | 복습 (time 셀 수 없음) |
| 4 | We have a lot of time. | 우리 시간 많아요. | 복습 (I/We have a lot of) |
| 5 | I don't have a lot of friends. | 나는 친구가 많지 않아요. / 별로 없어요. | 복습 (부정) |

---

## ✨ 상황 2 — 뭐뭐를 많이 한다 (동사 + a lot)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I exercise a lot.** | **나는 운동을 많이 해요.** | **← 오늘 #1 (동사 + a lot)** |
| 2 | 🆕 **I work out a lot.** | **나는 운동을 많이 해요.** | **← 오늘 #1 동의어** |
| 3 | 🆕 **I drink a lot.** | **나는 술을 많이 마셔요.** | **← 오늘 #2** |
| 4 | 🆕 **I don't drink a lot.** | **나는 술을 많이 안 마셔요. / 별로 안 마셔요.** | **← 오늘 #2 부정** |
| 5 | 🆕 **I sleep a lot.** | **나는 잠을 많이 자요.** | **← 오늘 #3** |
| 6 | 🆕 **I don't sleep a lot.** | **나는 잠을 별로 안 자요.** | **← 오늘 #3 부정** |
| 7 | 🆕 **He talks a lot.** | **걔는 말을 많이 해요. / 말이 많아요.** | **← 오늘 #4 (3인칭 단수 talks)** |

---

## ✨ 상황 3 — 목적어가 있을 때: a lot / a lot of 모두 가능

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I watch TV a lot.** | **나는 TV를 많이 봐요.** | **← 오늘 #5 (목적어 + a lot)** |
| 2 | 🆕 **I watch a lot of TV.** | **나는 TV를 많이 봐요.** | **← 오늘 #5 (a lot of + 목적어)** |
| 3 | 🆕 **I hang out with my friends a lot.** | **나는 친구들을 많이 만나요.** | **← 오늘 #6** |
| 4 | 🆕 **I eat hamburgers a lot.** | **나는 햄버거를 많이 먹어요.** | **← 오늘 #7** |
| 5 | 🆕 **I eat a lot of hamburgers.** | **나는 햄버거를 많이 먹어요.** | **← 오늘 #7 (a lot of 버전)** |

---

## 🔑 오늘의 변환 패턴 정리

```
[뭐뭐가 많다]
There are a lot of people.        사람이 많아요. (셀 수 있는 복수)
There is a lot of traffic.        교통량이 많아요. (셀 수 없음)
I have a lot of time.             나 시간 많아요.
I don't have a lot of friends.    나 친구 별로 없어요.

[뭐뭐를 많이 한다 — 목적어 없는 동사]
I exercise / sleep / drink + a lot   나는 운동/잠/술을 많이 해요/자요/마셔요.
(부정) I don't + 동사 + a lot        나는 ~ 별로 안 해요/안 자요.

[뭐뭐를 많이 한다 — 목적어 있는 동사]
I watch TV a lot.           = I watch a lot of TV.
I eat hamburgers a lot.     = I eat a lot of hamburgers.
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **work out** | v.구 | 운동하다 | = exercise |
| **hang out with** | v.구 | ~와 만나다, 어울리다 | `hang out with friends` |
| **traffic** | n. | 교통량 | 셀 수 없는 명사 → there is a lot of traffic |

---

## 📝 파생 연습 (5문제)

1. **영작**: "나는 커피를 많이 마셔요." (두 가지 방법으로)
2. **변환 (부정)**: `I travel a lot.` → "나는 여행을 별로 안 해요."
3. **영작**: "거기 차가 많아요." (traffic 사용)
4. **영작**: "나는 요즘 음악을 많이 들어요." (listen to + a lot)
5. **조합**: "나는 일을 많이 하고 잠을 별로 안 자요." (두 문장 이어서)

<details>
<summary>정답 보기</summary>

1. `I drink coffee a lot.` / `I drink a lot of coffee.`
2. `I don't travel a lot.`
3. `There is a lot of traffic there.`
4. `I listen to music a lot.` (또는 `I listen to a lot of music.`)
5. `I work a lot and I don't sleep a lot.`
</details>

- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.

[레슨 자료]

  1. There is/are a lot of + 명사 — ~이 많다 (셀 수 없으면 is, 복수이면 are)

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 a lot / a lot of 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 내가 많이 하는 활동 (목적어 없이)
2) 내가 별로 안 하는 활동
3) 내가 많이 먹거나 마시는 것
4) 내 생활에서 많은 것 (사람, 시간, 일 등)
5) 내가 많이 보거나 듣는 것

제 답변 5개를 받은 다음, 그것을 재료로 다음을 만들어 주세요:
- 동사 + a lot / 동사 + 목적어 + a lot / 동사 + a lot of + 목적어
- There is/are a lot of / I have a lot of
- 부정형도 함께

각 영어 문장은 한국어 의역을 같이 붙여 주세요. 마지막에 제가 따라 말할 수 있도록 5문장을 표로 정리해 주세요.
```
