# Lesson 044 — "젊었을 때"부터 "나이가 들수록"까지 — 나이·시절 표현 모음

> 📺 **원본 영상**: [[ 기초영어회화 ] - 젊었을 때 / 어렸을 때 / 학생이었을 때 / 나이가 들면서/들수록](https://www.youtube.com/watch?v=YN7l3dg-oog)
> ⏱ 6분 39초 · 👁 165K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **`When I was young`** vs **`When I was younger`** — 아직 젊은 축이라면 비교급 사용
> 2. **어렸을 때**: `When I was little` (강사 선호) / `When I was a kid` / `When I was a child`
> 3. **성장 시절**: `When I was growing up` (영어에서 자주 쓰는 표현)
> 4. **학창 시절**: `When I was in middle/high school` / `in university` / `in school`(→ 대학 느낌)
> 5. **나이가 들면서**: `As I get older` / **나이가 들수록**: `The older I get`

---

## ✨ 상황 1 — "젊었을 때" vs "지금보다 젊었을 때"

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | When I was young | 내가 젊었을 때 (지금은 젊지 않다는 전제) | 도입 |
| 2 | 🆕 **When I was younger** | **지금보다 젊었을 때** | **← 오늘 #1 (비교급 권장)** |

---

## ✨ 상황 2 — 어렸을 때 / 성장기 표현

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **When I was little** | **어렸을 때** | **← 오늘 #2 (강사 선호 표현)** |
| 2 | When I was a kid | 어렸을 때 | 도입 (같은 의미, 선택 가능) |
| 3 | When I was a child | 어렸을 때 | 도입 (같은 의미) |
| 4 | 🆕 **When I was growing up** | **내가 성장하고 있었을 때, 자랄 때** | **← 오늘 #3 (영어에서 자주 씀)** |

---

## ✨ 상황 3 — 학창 시절

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **When I was in middle school** | **중학교 다닐 때** | **← 오늘 #4** |
| 2 | 🆕 **When I was in high school** | **고등학교 다닐 때** | **← 오늘 #4** |
| 3 | 🆕 **When I was in university** | **대학교 다닐 때** | **← 오늘 #4** |
| 4 | When I was in school | 학교 다닐 때 (대학교 느낌이 강함) | 도입 |
| 5 | In middle school / In high school | 중학교 때 / 고등학교 때 (앞부분 생략형) | 도입 |

---

## ✨ 상황 4 — 나이가 들면서 / 들수록

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **As I get older** | **나이가 들면서** | **← 오늘 #5** |
| 2 | As she gets older / As we get older | 그녀가 / 우리가 나이가 들면서 | 도입 (주어 변형) |
| 3 | 🆕 **The older I get** | **나이가 들수록** | **← 오늘 #5** |
| 4 | Time seems to go faster as we get older. | 나이가 들면서 시간이 더 빨리 가는 것 같다. | 오늘 예문 |
| 5 | The older we get, the faster time seems to go. | 나이가 들수록 시간이 더 빨리 가는 것 같다. | 오늘 예문 (심화) |

## 🔑 오늘의 표현 패턴 정리

```
[젊었을 때 / 어렸을 때]
When I was young       →  (지금은 늙었다는 전제, 주의해서 사용)
When I was younger     →  지금보다 젊었을 때 (비교급 권장)
When I was little      →  어렸을 때 (강사 선호)
When I was growing up  →  자라고 있었을 때, 어릴 때

[학창 시절]
When I was in middle school   →  중학교 다닐 때
When I was in high school     →  고등학교 다닐 때
When I was in university      →  대학교 다닐 때
In high school / In university →  (앞부분 생략형)

[나이 관련]
As I get older              →  나이가 들면서
The older I get             →  나이가 들수록
Time seems to go faster as we get older.
The older we get, the faster time seems to go.
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **younger** | adj. | (지금보다) 더 젊은 | `young`의 비교급 |
| **growing up** | v.구 | 성장하고 있는, 자라는 | `When I was growing up` 관용 표현 |
| **seems to** | v.구 | ~인 것 같다 | `Time seems to go faster` — 실제 사실이 아니라 느낌 |
| **as** | conj. | ~하면서, ~할수록 | `As I get older` |
| **The + 비교급, the + 비교급** | 구문 | ~할수록 더 ~하다 | `The older we get, the faster time seems to go` |

---

## 🗣️ 발음 팁

1. **When I was younger** — `younger`의 첫 음절 강세: "영-거". 끝 `r`은 미국식으로 굴림.
2. **When I was growing up** — `growing up`을 한 호흡으로: "그로윙업".
3. **in university** — `in`과 `university` 사이 전치사 주의. 앞에 `at` 쓰지 않음.

---

## 📝 파생 연습 (5문제)

1. **선택**: 30대 초반인 사람이 "젊었을 때 운동을 많이 했어요."라고 할 때 — `young` vs `younger` 중 어느 쪽이 더 자연스럽나요?
2. **영작**: "저는 어렸을 때 피아노를 쳤어요." (`When I was little` 사용)
3. **영작**: "고등학교 다닐 때 영어를 많이 공부했어요."
4. **영작**: "나이가 들면서 새로운 걸 배우는 게 더 어려워지는 것 같아요." (`seems to` 사용)
5. **변환**: "나이가 들면서" → "나이가 들수록"으로 바꿔서 문장 만들기

<details>
<summary>정답 보기</summary>

1. `younger` — "When I was younger, I exercised a lot." (`young`은 지금은 늙었다는 전제라서 30대에게는 어색)
2. `When I was little, I played the piano.`
3. `When I was in high school, I studied English a lot.` (또는 `In high school, I studied English a lot.`)
4. `It seems to get harder to learn new things as I get older.`
5. `The older I get, the harder it seems to get to learn new things.` (또는 자유롭게 응용)

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 나이·시절 표현을 제 실제 경험으로 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 어렸을 때 좋아했던 것 (음식, 놀이, TV 프로그램 등)
2) 자라면서 기억에 남는 일
3) 학창 시절(중·고·대학) 하던 것
4) 나이가 들면서 달라진 것 (취향, 습관 등)
5) 나이가 들수록 더 그렇게 되는 것

제 답변 5개를 받은 다음, 오늘 배운 패턴으로 문장을 만들어 주세요:
- When I was little / younger / growing up + 과거 문장
- When I was in + 학교 + 과거 문장
- As I get older + 현재 문장
- The older I get, the + 비교급 문장

각 문장은 한국어 의역과 함께 표로 정리해 주세요.
```
