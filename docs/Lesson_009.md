# Lesson 009 — **의문문 Part 3**: be 동사 시제 변환 + **will it?** + **How was it?**

> 📺 **원본 영상**: [영어로 질문하기 [Part 3] [ LESSON #009 ] -- 영어회화 ](https://www.youtube.com/watch?v=pOggApqOz3Y)
> ⏱ — · 👁 —

> 🎯 **오늘의 핵심 포인트**
>
> 1. **be 동사 과거 의문문**: `Is it fun?` → **`Was it fun?`** (is → was)
> 2. **be 동사 미래 의문문 (인상)**: `be going to` 대신 **`will`** 권장 → **`Will it be fun?`**
> 3. 인상에 대한 미래 의문문은 `is it gonna be?`보다 `will it be?`가 더 자연스러움
> 4. **How + be 동사 의문문** = 어때요? / 어땠어요? → **`How is it?`** / **`How was it?`**
> 5. `good`은 상황에 따라 좋다/괜찮다/맛있다 등 다양하게 쓰임

---

## ✨ 상황 1 — be 동사 의문문 시제 변환 (현재 → 과거 → 미래)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | It's fun. / It is fun. | 재밌어요. | 복습 |
| 2 | Is it fun? | 재밌어요? | 복습 (현재 의문문) |
| 3 | 🆕 **Was it fun?** | **재밌었어요?** | **← 오늘 #1 (과거 의문문: was)** |
| 4 | It's good. | 좋아요. / 맛있어요. | 도입 |
| 5 | Is it good? | 좋아요? / 맛있어요? | 현재 의문문 |
| 6 | 🆕 **Was it good?** | **좋았어요? / 맛있었어요?** | **← 오늘 #2 (과거 의문문 응용)** |
| 7 | It's okay. | 괜찮아요. | 도입 |
| 8 | Is it okay? | 괜찮아요? | 현재 의문문 |
| 9 | 🆕 **Was it okay?** | **괜찮았어요?** | **← 오늘 #3 (과거 의문문 응용)** |

---

## ✨ 상황 2 — be 동사 미래 의문문: Will it be?

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | It's gonna be good. / It will be good. | 좋을 거예요. | 미래 평상문 |
| 2 | 🆕 **Will it be fun?** | **재밌을까요?** | **← 오늘 #4 (미래 인상 의문문: will it be)** |
| 3 | 🆕 **Will it be good?** | **좋을까요? / 맛있을까요?** | **← 오늘 #5 (will it be 응용)** |
| 4 | It will be okay. | 괜찮을 거예요. | 미래 평상문 |
| 5 | 🆕 **Will it be okay?** | **괜찮을까요?** | **← 오늘 #6 (will it be 응용)** |
| 6 | Is it going to be cold there? | 거기 추울까요? (예측 가능한 상황) | 참고 (be going to도 가능한 경우) |

---

## ✨ 상황 3 — How + be 동사 의문문: How is it? / How was it?

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | Is it okay? | 괜찮아요? | 도입 |
| 2 | 🆕 **How is it?** | **어때요?** | **← 오늘 #7 (how + is it = 어때요?)** |
| 3 | Was it okay? | 괜찮았어요? | 도입 |
| 4 | 🆕 **How was it?** | **어땠어요?** | **← 오늘 #8 (how + was it = 어땠어요?)** |
| 5 | — | 상대방 답변 예시: | — |
| 6 | It was fun. | 재밌었어요. | 답변 패턴 |
| 7 | It was good. | 좋았어요. / 맛있었어요. | 답변 패턴 |
| 8 | It was okay. | 괜찮았어요. | 답변 패턴 |

### 🧩 문맥으로 이어 말하기
> (영화를 보고 온 친구에게)
> **How was it?** → *It was fun. / It was good.*
> (어땠어? → 재밌었어. / 좋았어.)

---

## 🔑 오늘의 패턴 정리

```
[be 동사 의문문 시제 3종 세트]
현재  :  Is it fun?       재밌어요?
과거  :  Was it fun?      재밌었어요?       ← is → was
미래  :  Will it be fun?  재밌을까요?       ← will + be 원형

[인상 미래: be going to보다 will 권장]
어색함 :  Is it going to be fun?  (정해진 계획처럼 들려 어색)
자연스러움:  Will it be fun?        (주관적 예상·기대에 어울림)

[How + be 동사]
How is it?   어때요?      (현재)
How was it?  어땠어요?    (과거)
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **will** | aux. | ~할 것이다 (미래) | 인상 예측에 `be going to`보다 자연스러움 |
| **will it be** | expr. | ~할까요? | `Will it be fun? / good? / okay?` |
| **good** | adj. | 좋은 / 맛있는 / 괜찮은 | 문맥에 따라 다양하게 해석 |
| **okay** | adj. | 괜찮은 | `It's okay` = 그 상황이 괜찮다 (사람 아님) |
| **how** | wh. | 어떻게, 어때 | `How is it?` `How was it?` |

---

## 🗣️ 발음 팁

1. **Was it** — `t`를 살짝 끊어서: "워짓?" 처럼 연음.
2. **Will it be** — 한 호흡으로: "윌잇비?" 처럼.
3. **How was it** — 빠르게: "하우워짓?" 처럼.

---

## 📝 파생 연습 (5문제)

1. **변환 (과거)**: `Is it good?` → 과거 의문문으로.
2. **변환 (미래)**: `It's fun.` → `will`을 써서 미래 의문문으로.
3. **영작**: "어땠어요?" (영화를 보고 온 친구에게)
4. **영작**: "맛있었어요?" (상대방이 만든 음식을 먹고 나서)
5. **조합**: 친구가 새 카페를 갔다 왔다. → "어땠어?" 묻고 → 친구가 "좋았어."로 답하는 대화 두 줄 영작.

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트를 1줄씩 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: be 동사 의문문 시제 변환 (현재/과거/미래) + how was it?
- 오늘의 핵심:
  1. be 동사 과거 의문문: Was it + 형용사? (is → was)
  2. be 동사 미래 의문문: Will it be + 형용사? (인상 예측에 will 권장)
  3. How is it? = 어때요? / How was it? = 어땠어요?
  4. good = 좋다/맛있다/괜찮다 (문맥에 따라 다양)
- 핵심 예문:
  · Was it fun? (재밌었어요?)
  · Was it good? (좋았어요? / 맛있었어요?)
  · Was it okay? (괜찮았어요?)
  · Will it be fun? (재밌을까요?)
  · Will it be good? / Will it be okay?
  · How is it? (어때요?)
  · How was it? (어땠어요?)
  · It was fun. / It was good. / It was okay. (답변 패턴)
- 핵심 어휘: will, will it be, good(다의어), okay, how
- 발음 팁: Was it 연음, Will it be 연음, How was it 연음
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 채점 (✅ / ⚠️ / ❌). 틀린 부분 짧게 설명.
3) 총 10문제. 마지막에 복습 노트.

[레슨 핵심 패턴]
- be 동사 현재 의문문: Is it + 형용사?
- be 동사 과거 의문문: Was it + 형용사?
- be 동사 미래 의문문: Will it be + 형용사?
- How 의문문: How is it? / How was it?
- 답변 패턴: It was fun. / It was good. / It was okay.

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 주말에 만나 각자 경험한 것에 대해 영어로 잡담하는 중입니다.

규칙:
- 첫 인사는 당신이 먼저.
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 어색한 부분은 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 화제를 유도해 주세요:
  · Was it fun? / Was it good? / Was it okay?
  · How was it?
  · Will it be fun? / Will it be good?
  · It was fun. / It was good. / It was okay. (답변)

10턴 대화 후 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 be 동사 시제 의문문과 how 의문문 패턴을 제 일상으로 옮겨 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 최근에 경험한 것 (영화, 음식, 여행, 수업 등)
2) 곧 있을 경험이나 이벤트 (예상되는 인상)
3) 상대방에게 경험에 대해 묻고 싶은 것

제 답변을 받아 다음을 만들어 주세요:
- Was it + 형용사? 과거 의문문 2개
- Will it be + 형용사? 미래 의문문 1개
- How was it? 사용 짧은 대화 1쌍 (질문 + 답변)
- 각 문장은 한국어 의역 포함, 마지막에 표로 정리
```
