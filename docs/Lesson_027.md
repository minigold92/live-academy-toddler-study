# Lesson 027 — 감각동사 Part 1 — `look / sound / smell / taste / feel + 형용사`

> 📺 **원본 영상**: [감각동사 - PART 1 (기본개념) - look, sound, smell, taste, feel](https://www.youtube.com/watch?v=vPpsH30SmZs)
> ⏱ 11분 25초 · 👁 335K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **감각동사는 be동사를 대체** → `This is expensive` → `This looks expensive` (비싸 보인다)
> 2. **감각동사 5가지**: `look` (보이다), `sound` (들리다/~한 것 같다), `smell` (냄새가 나다), `taste` (맛이 나다), `feel` (느껴지다/기분이 ~하다)
> 3. **1차원적 감각뿐 아니라 감각을 통해 받은 인상도 표현 가능**
> 4. **시제 적용 가능** → `it tasted strange` (과거), `it feels heavy` (현재)

---

## ✨ 상황 1 — 보이는 것에 대한 인상 (look)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **This looks really expensive.** | **이거 되게 비싸 보인다.** | **← 오늘 #1 (look + 형용사)** |
| 2 | 🆕 **This looks cheap.** | **이거 싸 보인다. (생긴 게 싸 보인다)** | **← 오늘 #1** |
| 3 | 🆕 **It looked really serious.** | **되게 심각해 보였어요. (분위기가 심각해 보였어)** | **← 오늘 #1 (과거)** |

---

## ✨ 상황 2 — 들리는 것 / 느낌으로 받은 인상 (sound)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **This sounds a little strange.** | **이거 소리가 좀 이상한데. / 뭔가 좀 이상한데.** | **← 오늘 #2 (sound + 형용사)** |
| 2 | 🆕 **Hey, that sounds a little strange.** | **야, 그거 뭔가 좀 이상한데.** | **← 오늘 #2 (들은 얘기에 대한 인상)** |

---

## ✨ 상황 3 — 촉감·기분 (feel)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **This feels really heavy.** | **이거 정말 무겁게 느껴진다.** | **← 오늘 #3 (feel + 형용사)** |
| 2 | 🆕 **This feels really smooth.** | **이거 엄청 부드럽다. (촉감)** | **← 오늘 #3** |
| 3 | 🆕 **This feels really nice.** | **느낌이 되게 좋다.** | **← 오늘 #3 (촉감/기분)** |
| 4 | 🆕 **It felt really awkward.** | **엄청 어색했어. / 기분이 묘했어.** | **← 오늘 #3 (과거)** |
| 5 | 🆕 **It felt so awkward and weird.** | **너무 어색하고 이상했어.** | **← 오늘 #3** |

---

## ✨ 상황 4 — 냄새·맛 (smell / taste)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It didn't smell bad,** | **냄새는 나쁘지 않았는데,** | **← 오늘 #4 (smell + 형용사, 과거 부정)** |
| 2 | 🆕 **but it tasted a little strange.** | **맛이 좀 이상했어.** | **← 오늘 #4 (taste + 형용사, 과거)** |

### 🧩 문맥으로 이어 말하기
> **I saw my brother cry for the first time and it felt so awkward and weird.**
> (내 형이/남동생이 우는 걸 처음 봤는데 너무 어색하고 묘했어.)

> **It didn't smell bad, but it tasted a little strange.**
> (냄새는 나쁘지 않았는데 맛이 좀 이상했어.)

---

## 🔑 오늘의 변환/응용 패턴 정리

```
[감각동사 = be동사 대체]
This is expensive.      →  This looks expensive.   (비싸 보인다)
It is heavy.            →  This feels really heavy. (무겁게 느껴진다)
It was awkward.         →  It felt really awkward.  (어색하게 느껴졌어)

[감각동사 5가지 + 형용사]
look   + 형용사  →  보이다, ~해 보이다
sound  + 형용사  →  들리다, ~한 것 같다 (들은 것에 대한 인상도 포함)
smell  + 형용사  →  냄새가 나다
taste  + 형용사  →  맛이 나다
feel   + 형용사  →  느껴지다, 기분이 ~하다

[인상 표현 — 표면적 감각뿐 아니라 느낌도 포함]
It looked really serious.          →  되게 심각해 보였어 (분위기)
That sounds a little strange.      →  뭔가 좀 이상한데 (들은 얘기)
It felt so awkward.                →  너무 어색했어 (느낀 감정)
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **expensive** | adj. | 비싼 | `↔ cheap` (싼) |
| **serious** | adj. | 심각한 | `It looked really serious.` |
| **smooth** | adj. | 부드러운 (촉감) | `This feels really smooth.` |
| **awkward** | adj. | 어색한, 묘한 | 발음: "어크워드". 어색함·묘함·이상함에 모두 사용 |
| **weird** | adj. | 이상한, 묘한 | `awkward and weird`로 묶어 쓰기도 함 |
| **for the first time** | 구 | 처음으로, 최초로 | `I saw ~ for the first time` |

---

## 🗣️ 발음 팁

1. **awkward** — "어크워드" 처럼 발음. `awk`의 `aw`는 "어" 소리. 낯선 발음이므로 반복 연습 필요.
2. **This feels really heavy** — `feels`의 `s`와 `really`의 `r`을 자연스럽게 이어 읽기. "디스 필즈 리얼리 헤비" 한 호흡으로.

---

## 📝 파생 연습 (5문제)

1. **변환**: `This is really cheap.` → 감각동사로 "싸 보인다" 표현으로.
2. **영작**: "그 음식 맛이 정말 좋았어요." (`taste` + 과거)
3. **영작**: "그 사람 얘기를 들었는데 뭔가 좀 이상하게 들렸어요." (`sound` + 과거)
4. **영작**: "처음으로 그 영화를 봤는데 너무 슬펐어요." (`for the first time` 응용)
5. **조합**: "냄새는 좋았는데 맛은 좀 이상했어요." (`smell` / `taste` 조합)

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 감각동사 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 최근에 먹은 음식 중 맛이나 냄새가 인상적이었던 게 있나요?
2) 최근에 보고 어떤 인상을 받은 것이 있나요? (물건, 사람, 장소 등)
3) 최근에 들은 얘기 중 이상하거나 놀라웠던 게 있나요?
4) 만졌거나 느낀 것 중 인상적인 게 있었나요?
5) 최근에 어색하거나 불편하게 느꼈던 상황이 있나요?

제 답변 5개를 받은 다음, 그것을 재료로 다음을 만들어 주세요:
- smell/taste + 형용사 문장 1개
- look + 형용사 문장 1개
- sound + 형용사 문장 1개
- feel + 형용사 문장 2개

각 영어 문장은 한국어 의역을 같이 붙여 주세요. 마지막에 5문장을 표로 정리해 주세요.
```
