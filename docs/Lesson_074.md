# Lesson 074 — 날씨 변화를 영어로 표현하기 (It's cool → It's gotten much cooler)

> 📺 **원본 영상**: [5분 영어로 💬 "확실히 더 시원해졌어."](https://www.youtube.com/watch?v=8sPG7Wf2dgU)
> ⏱ 4분 22초 · 👁 69K회

> 🎯 **오늘의 핵심 포인트**
> 1. 단순 현재 상태 → `It's cool` (시원하다)
> 2. 비교급으로 상태 비교 → `It's cooler than yesterday` (어제보다 더 시원하다)
> 3. 변화가 진행 중 → `It's getting cooler` (시원해지고 있다)
> 4. 과거부터 지금까지 변화 완료 → `It has gotten cooler` (시원해졌다)
> 5. "확실히" 강조 → `definitely` — 변화를 나타내는 말 앞에 위치

---

## ✨ 상황 1 — 날씨 변화를 단계적으로 표현하기

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | It's cool. | 시원하다. | 도입 (현재 상태) |
| 2 | 🆕 **It's cooler.** | **더 시원하다.** | **← 오늘 #1 (비교급)** |
| 3 | 🆕 **It's cooler than yesterday.** | **어제보다 시원하다.** | **← 오늘 #1** |
| 4 | 🆕 **It's much cooler than yesterday.** | **어제보다 훨씬 더 시원하다.** | **← 오늘 #1 (much로 강조)** |
| 5 | 🆕 **It's getting cooler.** | **시원해지고 있다.** | **← 오늘 #2 (변화 진행 중)** |
| 6 | 🆕 **It's getting much cooler.** | **훨씬 시원해지고 있다.** | **← 오늘 #2** |
| 7 | 🆕 **It has gotten cooler.** | **시원해졌다.** | **← 오늘 #3 (변화 완료, 현재완료)** |
| 8 | 🆕 **It has gotten much cooler.** | **훨씬 더 시원해졌다.** | **← 오늘 #3** |
| 9 | 🆕 **It's definitely gotten much cooler.** | **확실히 훨씬 더 시원해졌어.** | **← 오늘 #4 (definitely 강조)** |

### 🧩 문맥으로 이어 말하기
> **Oh yeah, it's definitely gotten much cooler. I can feel it.**
> (맞아, 확실히 훨씬 더 시원해졌어. 느껴져.)

---

## 🔑 오늘의 변환 패턴 정리

```
[현재 상태]
It's cool.                          → 시원하다

[비교급]
It's cooler.                        → 더 시원하다
It's cooler than yesterday.         → 어제보다 시원하다
It's much cooler than yesterday.    → 어제보다 훨씬 더 시원하다

[변화 진행 중 — be getting + 비교급]
It's getting cooler.                → 시원해지고 있다
It's getting much cooler.           → 훨씬 시원해지고 있다

[변화 완료 — have gotten + 비교급]
It has gotten cooler.               → 시원해졌다  (과거 → 현재까지 변화)
It has gotten much cooler.          → 훨씬 더 시원해졌다

[강조 — definitely 위치]
It's definitely gotten much cooler. → 확실히 훨씬 더 시원해졌어
         ↑ definitely는 '변화'를 가리키는 gotten 앞에
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **cooler** | adj. | 더 시원한 | cool의 비교급 |
| **be getting + 비교급** | 구조 | ~해지고 있다 | 변화가 진행 중 |
| **have gotten + 비교급** | 구조 | ~해졌다 | 과거→현재까지 변화 완료 (현재완료) |
| **definitely** | adv. | 확실히 | 변화를 가리키는 말 바로 앞에 위치 |
| **much** | adv. | 훨씬 (비교급 강조) | `much cooler` = 훨씬 더 시원한 |

---

## 🗣️ 발음 팁

1. **It's gotten cooler** — `It's`의 t를 약하게. "잇스 가른 쿨러"처럼 연음.
2. **definitely** — "데피닛리"로 빠르게. 중간 음절 약화.
3. **It's definitely gotten much cooler** — 전체를 한 호흡으로 이어서.

---

## 📝 파생 연습 (5문제)

1. **영작**: "어제보다 훨씬 더 덥다." (hot 사용)
2. **변환**: `It's warm.` → "따뜻해지고 있다"로
3. **변환**: `It's cold.` → "훨씬 더 추워졌다"로 (현재완료)
4. **영작**: "확실히 훨씬 더 추워졌어. 느껴져." (cold, definitely)
5. **구분**: `It's getting cooler`와 `It has gotten cooler`의 차이를 우리말로 설명하면?

<details>
<summary>정답 보기</summary>

1. `It's much hotter than yesterday.`
2. `It's getting warmer.`
3. `It has gotten much colder.`
4. `It's definitely gotten much colder. I can feel it.`
5. `getting cooler` = 지금 변화가 진행 중 (시원해지는 중) / `gotten cooler` = 과거부터 지금까지 변화가 완료됨 (시원해진 상태)
</details>

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 날씨 변화 표현 — 현재 상태 / 비교 / 변화 진행 / 변화 완료
- 오늘의 핵심:
  1. 현재 상태: It's cool.
  2. 비교급: It's cooler than yesterday. / It's much cooler than yesterday.
  3. 변화 진행: It's getting cooler. (be getting + 비교급)
  4. 변화 완료: It has gotten cooler. (have gotten + 비교급, 현재완료)
  5. 강조: It's definitely gotten much cooler. (definitely는 gotten 앞에)
- 핵심 예문:
  · It's much cooler than yesterday. (어제보다 훨씬 더 시원하다)
  · It's getting much cooler. (훨씬 시원해지고 있다)
  · It has gotten cooler. (시원해졌다)
  · It's definitely gotten much cooler. I can feel it. (확실히 훨씬 더 시원해졌어. 느껴져)
- 핵심 어휘: cooler(비교급), be getting + 비교급, have gotten + 비교급, definitely, much
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ / ❌).
3) 총 10문제. 처음 5개는 레슨 표현 그대로, 나머지 5개는 응용.
4) 마지막에 자주 틀린 패턴 복습 노트.

[레슨 핵심 패턴]
- 비교급: It's cooler than yesterday. / It's much cooler than yesterday.
- 변화 진행: It's getting cooler. / It's getting much cooler.
- 변화 완료: It has gotten cooler. / It has gotten much cooler.
- 강조: It's definitely gotten much cooler.
- 적용 어휘: cool, warm, cold, hot, humid(습한), dry(건조한)

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 길을 걷다가 날씨 얘기를 하고 있습니다.

규칙:
- 첫 인사는 당신이 먼저. 날씨에 대한 자연스러운 말로 시작.
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 어색한 부분은 부드럽게 교정해 주세요.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 유도해 주세요:
  · It's much cooler than yesterday.
  · It's getting cooler.
  · It has gotten cooler.
  · It's definitely gotten much cooler.

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 날씨 변화 패턴을 날씨 이외의 상황에도 적용해 보고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 요즘 날씨가 어떻게 변하고 있나요?
2) 최근에 실력이 늘고 있는 것이 있나요? (운동, 영어, 요리 등)
3) 반대로 최근에 뭔가 나빠지고 있는 것이 있나요?

제 답변을 받은 다음, 아래를 만들어 주세요:
- be getting + 비교급 문장 2개
- have gotten + 비교급 문장 2개 (날씨 이외 주제)
- definitely를 넣은 강조 문장 1개

각 문장에 한국어 의역을 붙여 주세요. 마지막에 5문장 표로 정리해 주세요.
```
