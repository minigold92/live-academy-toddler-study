# Lesson 071 — **looks/sounds/feels familiar**: 낯이 익다 / 귀에 익다 / 잘 안다

> 📺 **원본 영상**: [낯익다, 낯설다 [ 영어 회화 패턴 ] → 반복 듣고 따라 말하기 [ familiar 연습영상 ]](https://www.youtube.com/watch?v=gxZTTgEnkcE)
> ⏱ 17분 55초 · 👁 268K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **낯이 익다** = **looks familiar** (감각동사 사용 — `is familiar` 보다 자연스러움)
> 2. **어디서 본 것 같다** = **I think I've seen him/her/it somewhere**
> 3. **귀에 익다** = **sounds familiar** / **어디서 들어본 것 같다** = **I think I've heard it somewhere**
> 4. **장소가 낯익다** = **this place looks/feels familiar** / **전에 와본 것 같다** = **I think I've been here before**
> 5. **경험으로 잘 안다** = **be familiar with ~** (감각이 아닌 경험·지식)
> 6. **아직 낯설다** = **I'm not very familiar with ~ yet**

> 💡 강사의 한마디:
> "강의를 그냥 듣는 것에서만 그치면 절대로 여러분의 것으로 만들 수 없어요. 이해한 것을 토대로 반드시 직접 생각해내는 내용을 적용시켜서 예문을 많이 만들어봐야 돼요."

## ✨ 상황 1 — 사람이 낯이 익다 (looks familiar)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **He looks kind of familiar.** | **저 남자 좀 낯이 익은데.** | **← 오늘 #1** |
| 2 | 🆕 **I think I've seen him somewhere.** | **어디서 본 것 같아.** | **← 오늘 #2 (현재완료 경험)** |
| 3 | 🆕 **She looks really familiar.** | **저 여자 낯이 굉장히 익은데.** | **← 오늘 #3** |
| 4 | 🆕 **I think I've seen her somewhere.** | **어디서 본 것 같아.** | **← 오늘 #4** |
| 5 | 🆕 **This looks kind of familiar.** | **이거 뭔가 좀 낯이 익은데.** | **← 오늘 #5 (물건도 가능)** |
| 6 | 🆕 **I think I've seen it somewhere.** | **어디서 본 것 같아.** | **← 오늘 #6** |

## ✨ 상황 2 — 소리·말이 귀에 익다 (sounds familiar)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **This song sounds really familiar.** | **이 노래 굉장히 귀에 익은데.** | **← 오늘 #7 (sounds)** |
| 2 | 🆕 **I think I've heard it somewhere.** | **어디서 들어본 것 같아.** | **← 오늘 #8** |
| 3 | 🆕 **That sounds really familiar.** | **그거 익숙한 얘기인데. (어디서 들어본 것 같아.)** | **← 오늘 #9 (남의 말에 대해서도)** |
| 4 | The song sounds kind of familiar. I think I've heard it somewhere. | 이 노래 좀 익숙한데, 어디서 들어본 것 같아. | 도입 |

---

## ✨ 상황 3 — 장소가 낯익다 (looks/feels familiar)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **This place looks kind of familiar.** | **여기 뭔가 낯이 익은데.** | **← 오늘 #10 (looks)** |
| 2 | 🆕 **I think I've been here before.** | **전에 와본 적이 있는 것 같아.** | **← 오늘 #11 (been here before)** |
| 3 | 🆕 **This place feels really familiar.** | **여기 굉장히 익숙한 느낌인데.** | **← 오늘 #12 (feels — 전반적 감각)** |

---

## ✨ 상황 4 — 경험으로 잘 안다 / 아직 낯설다 (be familiar with)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I grew up there, so I'm very familiar with that area.** | **거기서 자라서 그 지역을 아주 잘 알아요.** | **← 오늘 #13 (be familiar with)** |
| 2 | I've used it for a very long time, so I'm very familiar with it. | 오랫동안 사용해서 그것을 아주 잘 알아요. | 도입 |
| 3 | 🆕 **I just moved here, so I'm not very familiar with this area yet.** | **이사 온 지 얼마 안 돼서 아직 이 동네가 낯설어요.** | **← 오늘 #14** |
| 4 | 🆕 **I just bought it, so I'm not very familiar with it yet.** | **산 지 얼마 안 돼서 아직 잘 모르겠어요.** | **← 오늘 #15** |

### 🧩 문맥으로 이어 말하기
> **He looks kind of familiar. I think I've seen him somewhere.**
> (저 남자 좀 낯이 익는데. 어디서 본 것 같아.)

> **I just moved here, so I'm not very familiar with this area yet.**
> (이사 온 지 얼마 안 돼서 아직 이 동네가 낯설어요.)

---

## 🔑 오늘의 패턴 정리

```
[감각동사 + familiar — 낯익다/귀에 익다]
보고 낯이 익다  : looks familiar   (사람/물건/장소)
듣고 귀에 익다  : sounds familiar  (음악/말)
있어 보니 익숙  : feels familiar   (장소, 전반적 감각)
※ is/are familiar 도 틀린 건 아니지만 감각동사가 더 자연스러움

[어디서 본/들은 것 같다]
보다  : I think I've seen  him/her/it  somewhere.
듣다  : I think I've heard it          somewhere.
가다  : I think I've been here          before.     ← somewhere 아님!

[be familiar with — 경험으로 잘 안다/모른다]
잘 안다  : I'm (very) familiar with that area.
낯설다   : I'm not (very) familiar with this area yet.
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **familiar** | adj. | 낯익은, 익숙한 | `looks/sounds/feels familiar` |
| **kind of** | adv. | 좀, 약간 | `looks kind of familiar` |
| **somewhere** | adv. | 어딘가에서 | `seen/heard it somewhere` |
| **before** | adv. | 전에 | `I've been here before` (somewhere 아님) |
| **grew up** | v. | 자랐다 (grow up 과거) | `I grew up there` |
| **be familiar with** | 표현 | ~를 잘 알다, ~에 익숙하다 | 경험·지식 기반 |
| **I just moved here** | 표현 | 이사 온 지 얼마 안 됐어요 | `just` = 방금/얼마 안 된 |

---

## 🗣️ 발음 팁

1. **I'm not very familiar with this area yet** — 후반부 연음이 어려움. `with`를 기준으로 나눠 연습: `I'm not very familiar with` + `this area yet`.
2. **I think I've seen him somewhere** — `I've seen`을 "아이브 씬"처럼 빠르게.
3. **I grew up there** — `grew up`은 "그류업"처럼 연음.

---

## 📝 파생 연습 (5문제)

1. **영작**: "저 여자 굉장히 낯이 익은데. 어디서 본 것 같아." (she / somewhere)

5. **영작**: "이 앱 산 지 얼마 안 돼서 아직 잘 모르겠어요." (I just bought it / not very familiar)

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 6가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: looks/sounds/feels familiar — 낯이 익다 / 귀에 익다 / 잘 안다
- 오늘의 핵심:
  1. 낯이 익다 = looks familiar (감각동사, is familiar보다 자연스러움)
  2. 어디서 본 것 같다 = I think I've seen him/her/it somewhere
  3. 귀에 익다 = sounds familiar / 어디서 들어봤다 = I think I've heard it somewhere
  4. 장소 낯익다 = this place looks/feels familiar / 전에 와봤다 = I've been here before
  5. 경험으로 잘 안다 = be familiar with ~ (감각 아닌 지식/경험)
  6. 아직 낯설다 = I'm not very familiar with ~ yet
- 핵심 예문:
  · He looks kind of familiar. I think I've seen him somewhere.
  · She looks really familiar. I think I've seen her somewhere.
  · This song sounds really familiar. I think I've heard it somewhere.
  · That sounds really familiar. I think I've heard it somewhere.
  · This place feels really familiar. I think I've been here before.
  · I grew up there, so I'm very familiar with that area.
  · I just moved here, so I'm not very familiar with this area yet.
  · I just bought it, so I'm not very familiar with it yet.
- 핵심 어휘: familiar, kind of, somewhere, before, grew up, be familiar with, I just moved/bought
- 발음 팁: I'm not very familiar with this area yet — with 기준으로 나눠 연습
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)


### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 familiar 표현을 제 실제 경험으로 연습하고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 최근에 낯이 익다고 느낀 사람이나 물건 (어디서 봤는지)
2) 귀에 익은 노래나 말을 들은 경험
3) 가본 적 있는 것 같은 장소에 간 경험
4) 오래 경험해서 잘 아는 것 (장소/물건)
5) 아직 낯선 것 (이사/구매 등)

답변을 받은 뒤:
- looks/sounds/feels familiar + I think I've seen/heard/been 문장 각 1개
- I'm very familiar with ~ 문장 1개
- I'm not very familiar with ~ yet 문장 1개

각 문장에 한국어 의역과 발음 팁(필요시)을 붙여 표로 정리해 주세요.
```
