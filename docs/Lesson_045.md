# Lesson 045 — 고민 표현하기: "I'm not sure if I should..." / "I'm trying to decide..."

> 📺 **원본 영상**: [[ 기초영어회화 ] '고민'에 대해서 영어로 표현하기](https://www.youtube.com/watch?v=bNqR7zgoc04)
> ⏱ 13분 47초 · 👁 400K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. 고민을 표현할 때 **"고민이다"에 해당하는 단일 동사는 없다** — 돌려서 표현
> 2. **`I'm not sure if I should ~`** — 할지 안 할지 잘 모르겠어
> 3. **`I'm trying to decide if I should ~`** — 결정하려고 노력 중이야 (고민 중이야)
> 4. **`I'm still trying to decide ~`** — 아직도 고민 중이야 (오래 지속된 고민)
> 5. `if` (Yes/No) 뿐 아니라 **의문사**(what, which) 간접 의문문도 연결 가능

---

## ✨ 상황 1 — 할지 말지 고민: if + should

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | I should go to bed early. | 일찍 자는 게 좋겠어. | 도입 (should의 권장 뉘앙스) |
| 2 | Should I go to bed early? | 일찍 자는 게 좋을까? | 도입 (의문문) |
| 3 | 🆕 **if I should go to bed early** | **일찍 자는 것이 좋을지** | **← 오늘 #1 (if 간접 의문절)** |
| 4 | 🆕 **I'm not sure if I should go to bed early.** | **일찍 자는 것이 좋을지 잘 모르겠어.** | **← 오늘 #1 핵심 패턴** |
| 5 | 🆕 **I'm not sure if I should tell him about this.** | **걔한테 이거에 대해서 얘기를 해줄지 고민이야.** | **← 오늘 #1 응용** |

---

## ✨ 상황 2 — 둘 중 어느 쪽 할지 고민: if ~ or ~

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I'm not sure if I should tell him about this now or wait.** | **걔한테 지금 얘기를 할지 좀 기다릴지 고민이야.** | **← 오늘 #2 (if ~ or ~)** |
| 2 | 🆕 **I'm not sure if I should buy a new smartphone now or wait until a new model comes out.** | **지금 새 스마트폰을 살지 아니면 새 모델이 나올 때까지 기다릴지 모르겠어.** | **← 오늘 #2 응용** |

---

## ✨ 상황 3 — "결정하려고 노력 중": I'm trying to decide

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I'm trying to decide if I should buy a new smartphone now or wait until a new model comes out.** | **지금 살지 아니면 기다릴지 결정하려고 노력 중이야.** | **← 오늘 #3** |
| 2 | 🆕 **I'm still trying to decide if I should ~** | **아직도 ~할지 고민 중이야.** | **← 오늘 #3 (still 강조)** |

---

## ✨ 상황 4 — 의문사(what, which)와 연결

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | What should I tell him? | 걔한테 뭐라고 얘기를 할까? | 도입 |
| 2 | 🆕 **what I should tell him** | **그에게 무슨 말을 해줄지** | **← 오늘 #4 (what 간접절)** |
| 3 | 🆕 **I'm not sure what I should tell him.** | **걔한테 뭐라고 얘기를 해줄지 고민이야.** | **← 오늘 #4** |
| 4 | 🆕 **I'm not sure which one I should buy.** | **어느 거를 사는 게 좋을지 고민이야.** | **← 오늘 #4 (which one)** |
| 5 | 🆕 **I'm trying to decide which one I should buy.** | **어느 걸 살지 결정하려고 노력 중이야.** | **← 오늘 #4** |
| 6 | 🆕 **I'm still trying to decide which one I should buy.** | **아직도 어느 걸 살지 고민 중이야.** | **← 오늘 #4 (still)** |

---

## 🔑 오늘의 변환 패턴 정리

```
[고민 표현 두 가지 핵심 패턴]
I'm not sure        + if I should ~            (할지 안 할지 잘 모르겠어)
I'm trying to decide + if I should ~           (결정하려고 고민 중이야)
I'm still trying to decide + if I should ~     (아직도 고민 중이야)

[옵션 추가]
if ~ or ~          : if I should do this now or wait
                     (이렇게 할지 저렇게 할지)

[의문사 연결]
I'm not sure + what I should tell him          (뭐라고 할지 모르겠어)
I'm not sure + which one I should buy          (어느 걸 살지 모르겠어)
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **should** | 조동사 | ~하는 게 좋겠다 (권장) | 의무(have to)보다 부드러운 권장의 뉘앙스 |
| **I'm not sure** | v.구 | 잘 모르겠다, 확실하지 않다 | `I don't know well`이 아님 |
| **I'm trying to decide** | v.구 | 결정하려고 노력 중이다 | 고민 중이다의 직설적 표현 |
| **still** | adv. | 아직도 | `I'm still trying to decide` — 오래된 고민 강조 |
| **which one** | 표현 | 어느 것, 어느 거 | `what`은 무엇을, `which one`은 주어진 선택지 중 어느 것 |
| **until** | prep./conj. | ~할 때까지 | `wait until a new model comes out` |
| **comes out** | v.구 | 출시되다, 나오다 | `a new model comes out` |

---

## 📝 파생 연습 (5문제)

1. **영작**: "지금 이걸 살지 잘 모르겠어." (`I'm not sure if I should` 사용)
2. **영작**: "뭐라고 말해야 할지 고민이야." (`I'm not sure what I should` 사용)
3. **영작**: "어느 거를 골라야 할지 아직도 결정을 못 하겠어." (`I'm still trying to decide which one` 사용)
4. **응용**: "지금 여행을 갈지 아니면 조금 더 기다릴지 모르겠어." (`if ~ or ~` 패턴)
5. **선택**: `I'm not sure if I should ~` vs `I'm trying to decide if I should ~` — 어감 차이를 한국어로 설명해보세요.

<details>
<summary>정답 보기</summary>

1. `I'm not sure if I should buy this now.`
2. `I'm not sure what I should say.`
3. `I'm still trying to decide which one I should choose.`
4. `I'm not sure if I should go on a trip now or wait a little longer.`
5. `I'm not sure` = 막연하게 잘 모르겠다 / `I'm trying to decide` = 결정하려고 적극적으로 고민 중이다 (좀 더 직설적)
</details>

## 🤖 AI와 함께 공부하기

ChatGPT / Gemini / Claude 등 어떤 AI에든 **아래 블록 전체를 그대로 복사해서 붙여넣으면** 그 모드로 학습 세션이 시작됩니다.

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 고민 표현하기 — I'm not sure if I should / I'm trying to decide
- 오늘의 핵심:
  1. 고민이다 = I'm not sure / I'm trying to decide (단일 동사 없음)
  2. I'm not sure if I should ~ (할지 안 할지 잘 모르겠어)
  3. I'm trying to decide if I should ~ (결정하려고 노력 중이야)
  4. I'm still trying to decide ~ (아직도 고민 중이야)
  5. 의문사(what, which one) 연결도 가능
- 핵심 예문:
  · I'm not sure if I should tell him about this. (걔한테 얘기할지 고민이야)
  · I'm not sure if I should buy a new smartphone now or wait until a new model comes out. (지금 살지 기다릴지 모르겠어)
  · I'm trying to decide which one I should buy. (어느 걸 살지 결정하려고 노력 중이야)
  · I'm not sure what I should tell him. (뭐라고 얘기해줄지 모르겠어)
  · I'm still trying to decide which one I should buy. (아직도 어느 걸 살지 고민 중)
- 핵심 어휘: should(권장), I'm not sure, I'm trying to decide, still, which one, until, comes out
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ / ❌). 잘못된 부분은 짧게 설명.
3) 총 10문제. 처음 5문제는 레슨 표현, 나머지 5문제는 응용.
4) 마지막에 자주 틀린 패턴 복습 노트 정리.

[레슨 핵심 패턴]
- I'm not sure if I should + 동사원형 (~할지 모르겠어)
- I'm not sure if I should ~ or + 동사원형 (이렇게 할지 저렇게 할지)
- I'm trying to decide if/what/which one I should ~
- I'm still trying to decide ~
- 활용 표현: tell him about this, buy a new smartphone, wait until ~, which one to ~

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 카페에서 만나 서로의 고민 이야기를 나누고 있습니다.

규칙:
- 첫 인사는 당신이 먼저. ("Hey, you look like you're thinking hard about something." 같은 자연스러운 시작)
- 매 턴마다 영어 1~2문장 + (괄호 안에 한국어 의역).
- 어색한 부분은 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 유도:
  · I'm not sure if I should ~
  · I'm trying to decide if I should ~ or ~
  · I'm still trying to decide which one I should ~
  · I'm not sure what I should ~

10턴 후 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 고민 표현 패턴을 제 실제 고민으로 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 요즘 할지 말지 고민하는 것 (여행, 구매, 결정 등)
2) 두 가지 선택지 중에서 고민 중인 것
3) 누군가에게 얘기해야 할지 말지 고민인 것
4) 아직도 오랫동안 결정 못 한 것
5) 뭐라고 말해야 할지 모르는 상황

제 답변 5개를 받은 다음, 오늘 배운 패턴으로 문장을 만들어 주세요:
- I'm not sure if I should ~ (2문장)
- I'm trying to decide if ~ or ~ (1문장)
- I'm still trying to decide which one I should ~ (1문장)
- I'm not sure what I should ~ (1문장)

각 문장은 한국어 의역과 함께 표로 정리해 주세요.
```
