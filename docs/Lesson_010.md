# Lesson 010 — 문장 연결법: **because** (이유) · **and** (나열) · **so** (결과)

> 📺 **원본 영상**: [[LESSON#10] - 영어 문장 연결하기 [PART 1]- and / so / because - [라이브 아카데미 토들러]](https://www.youtube.com/watch?v=uPWsw5GIbtI)
> ⏱ 7분 05초 · 👁 457K회

> 🎯 **오늘의 핵심 포인트**
> 1. **because** = ~때문에 (이유를 설명하는 것이 주목적일 때): 결과 먼저, 이유 나중
> 2. **and** = 그리고 (일어난 순서대로 나열)
> 3. **so** = 그래서 (상황 → 내 반응·대처)
> 4. 같은 한국어("차가 막혀서 늦었어")도 의도에 따라 세 가지 다르게 표현
> 5. **there was a lot of traffic** = 차가 많이 막혔어요 (덩어리째 외우기)

---

## ✨ 상황 — 차 막혀서 지각, 세 가지 방식으로 말하기

### 방법 1: 이유 설명이 목적 → because (결과 먼저!)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I was late** | **저는 늦었어요.** | **← 결과 먼저** |
| 2 | 🆕 **because there was a lot of traffic.** | **차가 많이 막혔기 때문이에요.** | **← 오늘 #1 (이유: because)** |
| 3 | 🆕 **I was late because there was a lot of traffic.** | **차가 많이 막혀서 늦었어요.** | **← 전체 문장** |

### 방법 2: 일어난 순서대로 나열 → and

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **There was a lot of traffic** | **차가 많이 막혔어요.** | **← 먼저 일어난 일** |
| 2 | 🆕 **and I was late.** | **그리고 늦었어요.** | **← 오늘 #2 (나열: and)** |
| 3 | 🆕 **There was a lot of traffic and I was late.** | **차가 많이 막혔고, 늦었어요.** | **← 전체 문장** |

### 방법 3: 상황 → 나의 반응 → so

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | There was a lot of traffic. | 차가 많이 막혔어요. | 상황 |
| 2 | 🆕 **So I got off.** | **그래서 (차에서) 내렸어요.** | **← 오늘 #3 (반응: so + get off 과거)** |
| 3 | 🆕 **And I took the subway.** | **그리고 지하철을 탔어요.** | **← 오늘 #4 (이어지는 행동: and)** |
| 4 | 🆕 **There was a lot of traffic. So I got off. And I took the subway.** | **차가 많이 막혔어요. 그래서 내려서 지하철을 탔어요.** | **← 전체 문장** |

---

## 🔑 오늘의 패턴 정리

```
[세 가지 연결 방식 비교]
because  :  결과 먼저, 이유 설명이 목적
            I was late because there was a lot of traffic.

and      :  일어난 순서대로 나열
            There was a lot of traffic and I was late.

so       :  상황 → 나의 반응·대처
            There was a lot of traffic. So I got off. And I took the subway.

[핵심 표현 — 덩어리째 외우기]
there was a lot of traffic    차가 많이 막혔어요
I was late                    저는 늦었어요
I got off                     (교통수단에서) 내렸어요
I took the subway             지하철을 탔어요
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **because** | conj. | ~때문에 | 이유 설명이 주목적일 때 |
| **there was** | expr. | ~이 있었다 | `there` = '거기'가 아님. `there was + 명사` |
| **a lot of traffic** | expr. | 많은 교통량, 차가 많이 막힘 | `traffic` = 교통량 (셀 수 없는 명사) |
| **get off / got off** | v. | (교통수단에서) 내리다 | 과거: `got off` |
| **take the subway / took the subway** | expr. | 지하철을 타다 | 과거: `took` |
| **so** | conj. | 그래서 (결과·반응) | 상황 → 나의 대처 연결 |

## 📝 파생 연습 (4문제)

1. **선택**: "배고파서 밥을 먹었어요." — 이유 설명이 목적이라면 어떻게? (because)
2. **영작 (so)**: "비가 왔어요. 그래서 집에 있었어요."
3. **영작 (because)**: "늦었어요. 차가 많이 막혔기 때문이에요."
4. **조합**: "차가 많이 막혔어요. 그래서 내려서 지하철을 탔어요." 영어로.

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 because / and / so 패턴을 제 일상으로 옮겨 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 최근에 어떤 일이 생겨서 늦거나 계획이 바뀐 경험
2) 그 일의 원인 (이유)
3) 그 일에 대해 어떻게 반응했는지 (대처 방법)

제 답변을 받아 다음을 만들어 주세요:
- because 사용 문장: [결과] because [이유]
- and 사용 문장: [이유] and [결과] (순서 나열)
- so 사용 문장: [이유]. So [반응/대처].
- 각 문장은 한국어 의역 포함, 마지막에 표로 정리
```
