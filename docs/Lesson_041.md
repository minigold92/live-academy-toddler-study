# Lesson 041 — **left** — "뭐뭐가 남아 있다/없다" 표현 (have/there is + left)

> 📺 **원본 영상**: [left - 시간이 조금 남았다 / 돈이 많이 안남았다 [ 기초영어회화 ]](https://www.youtube.com/watch?v=V3yn8eUAHYk)
> ⏱ 7분 37초 · 👁 224K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **`I have some time left`** — 나 시간 좀 남았어 (have + 명사 + left)
> 2. **`I don't have any time left`** — 나 시간이 하나도 안 남았어
> 3. **`I have a lot of time left`** — 시간이 많이 남았어
> 4. **`There are/aren't any tickets left`** — 표가 남았다/하나도 안 남았다 (there is/are + 명사 + left)
> 5. `some` / `any` / `a lot of` + 명사 + **left** — 어떤 양이 남아 있는 상태

---

## ✨ 상황 1 — 나/우리에게 뭐뭐가 남아 있을 때 (have + left)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | I have some time. / I don't have any time. | 나 시간 좀 있어. / 나 시간 없어. | 복습 (some, any) |
| 2 | 🆕 **I have some time left.** | **나 시간 좀 남았어.** | **← 오늘 #1 (have + left)** |
| 3 | 🆕 **I don't have any time left.** | **나 시간이 하나도 안 남았어.** | **← 오늘 #2 (부정 + any)** |
| 4 | 🆕 **I have a lot of time left.** | **시간이 많이 남았어.** | **← 오늘 #3 (a lot of)** |
| 5 | 🆕 **I don't have a lot of time left.** | **시간이 많이 안 남았어. / 시간이 별로 안 남았어.** | **← 오늘 #3 부정** |
| 6 | 🆕 **I have some money left.** | **나 돈 좀 남았어.** | **← 오늘 응용 (money)** |
| 7 | 🆕 **I don't have any money left.** | **돈이 하나도 안 남았어.** | **← 오늘 응용 부정** |

---

## ✨ 상황 2 — 뭐뭐가 (일반적으로) 남아 있을 때 (there is/are + left)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **There aren't any tickets left.** | **표가 하나도 안 남았어요.** | **← 오늘 #4 (셀 수 있는 복수 → aren't)** |
| 2 | 🆕 **There are some tickets left.** | **표가 좀 남았어요.** | **← 오늘 #4 긍정** |
| 3 | 🆕 **There aren't many people left there.** | **거기에 사람이 많이 안 남았어요.** | **← 오늘 #5 (many = 복수)** |
| 4 | 🆕 **There weren't many people left there.** | **거기에 사람이 많이 안 남아 있었어요.** | **← 오늘 #5 과거형** |

---

## 🔑 오늘의 변환 패턴 정리

```
[have + 명사 + left — 나한테 남아 있다]
I have some time left.              시간 좀 남았어.
I don't have any time left.         시간 하나도 안 남았어.
I have a lot of time left.          시간 많이 남았어.
I don't have a lot of time left.    시간 많이 안 남았어.
(time → money, food, energy 등으로 교체 가능)

[there is/are + 명사 + left — (일반적으로) 남아 있다]
There are some tickets left.        표가 좀 남았어요.
There aren't any tickets left.      표가 하나도 안 남았어요.
There aren't many people left.      사람이 많이 안 남았어요.
There isn't a lot of time left.     시간이 별로 안 남았어요.

[시제 교체]
There aren't → There weren't  (현재 → 과거)
I don't have → I didn't have  (현재 → 과거)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **left** | adj./adv. | 남아 있는 (leave의 분사형) | 문장 끝에 붙여 "남아 있다" 의미 |
| **ticket** | n. | 표, 티켓 | 셀 수 있음 → tickets (복수) |

---

## 📝 파생 연습 (5문제)

1. **영작**: "나 배터리가 좀 남았어요." (`battery`)
2. **변환 (부정)**: "나 돈이 하나도 안 남았어요."
3. **영작**: "좌석이 몇 개 남았어요." (`seats`, there are some)
4. **변환 (과거)**: "어제 시간이 별로 안 남아 있었어요."
5. **응용**: "음식이 많이 남았어요." (food, there is a lot of)

```

2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]

```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 "뭐뭐가 남아 있다/없다 (left)" 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 지금 나한테 얼마나 시간이 남아 있는지
2) 오늘 지갑/계좌에 돈이 얼마나 남아 있는지
3) 최근에 티켓/자리 등이 다 팔렸던 경험
4) 냉장고에 뭔가 남아 있는지 없는지
5) 배터리나 에너지가 얼마나 남아 있는지

제 답변 5개를 받은 다음, 그것을 재료로 다음을 만들어 주세요:
- I have/don't have + (some/any/a lot of) + 명사 + left
- There is/are/(aren't) + (some/any/many) + 명사 + left

각 영어 문장은 한국어 의역을 같이 붙여 주세요. 마지막에 제가 따라 말할 수 있도록 5문장을 표로 정리해 주세요.
```
