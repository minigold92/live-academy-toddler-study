# Lesson 031 — 관계절 Part 1 — 대상이 먼저, 부연 설명이 나중 (`that` 관계대명사)

> 📺 **원본 영상**: [관계절 - PART 1 - 기본개념과 기본 구성 이해하기](https://www.youtube.com/watch?v=xY8-tOJu0GQ)
> ⏱ 10분 47초 · 👁 301K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. **한국어 vs 영어 어순** — 한국어는 [부연설명 + 대상], 영어는 [대상 + 부연설명]
> 2. **관계대명사 `that`** — 대상과 부연 설명을 이어주는 연결고리
> 3. **관계절의 목적어 생략** — 앞에 나온 대상이 이미 목적어 역할 → 뒤에서 중복 금지
> 4. **완성 문장으로 만들기** — `This is the + 대상 + that + 부연설명`
> 5. **`for the last two years`** = 지난 2년 동안 (현재완료 진행과 함께)

> 💡 강사의 한마디:
> "오늘 제가 보여드린 이 세 가지 예문만 이해를 완벽히 하고 외웠다고 해서 거기서 그쳐서는 안 돼요. 반드시 여러분 주변에 있는 것들을 다 이용해서 문장을 직접 많이 만드는 연습을 하셔야 돼요."

## ✨ 상황 1 — 대상 지목 연습 (완성 문장 전 단계)

| | 한국어 | 영어 (대상 먼저) | 구분 |
|---|---|---|---|
| 1 | 내가 지난 2년 동안 써온 **스마트폰** | 🆕 **The smartphone that I've been using for the last two years** | **← 오늘 #1/#2** |
| 2 | 친구가 나한테 선물로 준 **펜** | 🆕 **A pen that a friend gave me as a gift** | **← 오늘 #1/#2** |
| 3 | 내가 강의를 할 때 쓰는 **모자** | 🆕 **The hat that I wear when I teach** | **← 오늘 #1/#2** |

## ✨ 상황 2 — 완성 문장으로 만들기 (`This is ~`)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **This is the smartphone that I've been using for the last two years.** | **이게 제가 지난 2년 동안 쭉 써온 스마트폰이에요.** | **← 오늘 #4** |
| 2 | 🆕 **This is a pen that a friend gave me as a gift.** | **이거는 친구가 저한테 선물로 준 펜이에요.** | **← 오늘 #4** |
| 3 | 🆕 **This is the hat that I wear when I teach.** | **이거는 제가 강의를 할 때 쓰는 모자예요.** | **← 오늘 #4** |

## 🔑 오늘의 변환/응용 패턴 정리

```
[한국어 vs 영어 어순 비교]
한국어:  [부연설명] + [대상]       내가 지난 2년 동안 써온 스마트폰
영어:    [대상] + that + [부연설명]  The smartphone that I've been using for the last two years

[관계절 구조]
The smartphone  that  I've been using for the last two years
  ↑ 대상        ↑연결  ↑ 부연 설명 (목적어 생략 주의!)

★ 목적어 생략 규칙:
앞에 나온 대상이 이미 목적어 역할을 하기 때문에,
뒤 부연 설명 안에서는 그 목적어를 다시 쓰면 안 됨.
✗ The smartphone that I've been using the smartphone for the last two years
✅ The smartphone that I've been using for the last two years

[완성 문장]
This is + 대상 + that + 부연설명
= 이게 바로 ~한/했던 [대상]이에요.

[시제 선택]
현재 사용 중:       The hat that I wear when I teach (현재)
과거에 줬음:        A pen that a friend gave me as a gift (과거)
과거~현재 계속:     The smartphone that I've been using for the last two years (현재완료진행)
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **that** | 관계대명사 | ~한/했던 (대상과 부연설명 연결) | 사람에는 `who`도 쓰지만, `that`도 가능 |
| **for the last two years** | 구 | 지난 2년 동안 | 현재완료(진행)와 함께 씀 |
| **as a gift** | 구 | 선물로 | `gave me as a gift` = 선물로 줬어 |
| **wear** | v. | (모자/옷 등을) 쓰다, 입다 | 쓰다/입다 = 같은 동사 `wear` |
| **teach** | v. | 가르치다, 강의하다 | `when I teach` = 내가 강의를 할 때 |

---

## 🗣️ 발음 팁

1. **that I've been using** — `that`이 약화돼 "댓"처럼 짧게. `I've been`을 "아이브 빈"으로 이어 읽기. 전체를 한 호흡으로 이어서 연습.
2. **for the last two years** — `for the`를 "퍼더" 처럼 연음. "퍼더 래스트 투 이어즈" 한 덩어리로.

---

## 📝 파생 연습 (5문제)

1. **변환**: "내가 음악을 들을 때 쓰는 이어폰" → 영어로 대상 지목 표현 만들기.
2. **완성 문장**: "이게 내가 매일 쓰는 노트북이에요." (`This is the laptop that ~`)
3. **완성 문장**: "이거는 엄마가 나한테 사준 가방이에요." (`gave me as a gift` 응용)
4. **변환**: "내가 공부할 때 앉는 의자" → 영어로 대상 지목 표현 만들기.
5. **오류 찾기**: `This is the book that I read the book last year.` — 무엇이 잘못됐나?

<details>
<summary>정답 보기</summary>

1. `The earphones that I use when I listen to music`
2. `This is the laptop that I use every day.`
3. `This is the bag that my mom bought me.` (또는 `gave me as a gift`)
4. `The chair that I sit in when I study`
5. `the book`이 중복. → `This is the book that I read last year.`

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 관계절 Part 1 — 대상 먼저, that으로 연결, 부연 설명 나중
- 오늘의 핵심:
  1. 한국어 [부연설명+대상] vs 영어 [대상+that+부연설명] 어순
  2. 관계대명사 that = 대상과 부연설명을 이어주는 연결고리
  3. 부연설명 안에서 목적어 반드시 생략 (앞 대상이 목적어 역할)
  4. This is + 대상 + that + 부연설명 (완성 문장)
  5. for the last two years = 지난 2년 동안 (현재완료진행과 함께)
- 핵심 예문 (대상 지목):
  · The smartphone that I've been using for the last two years
  · A pen that a friend gave me as a gift
  · The hat that I wear when I teach
- 핵심 예문 (완성 문장):
  · This is the smartphone that I've been using for the last two years. (이게 지난 2년 동안 써온 스마트폰이에요.)
  · This is a pen that a friend gave me as a gift. (이거는 친구가 선물로 준 펜이에요.)
  · This is the hat that I wear when I teach. (이거는 강의할 때 쓰는 모자예요.)
- 핵심 어휘: that (관계대명사), for the last two years, as a gift, wear, teach
- 발음 팁: that I've been using 연음, for the last 연음
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ 부분정답 / ❌).
3) 같은 패턴으로 총 10문제.
4) 마지막에 자주 틀린 패턴 복습 노트.

처음 5문제는 "대상 지목" 형태(완성 문장 아님), 나머지 5문제는 "This is ~ that ~" 완성 문장으로.

[레슨 핵심 패턴]
- 대상 지목: 대상 + that + 부연설명 (예: The smartphone that I've been using for the last two years)
- 완성 문장: This is + 대상 + that + 부연설명 (예: This is the hat that I wear when I teach.)
- 목적어 생략 규칙: 관계절 안에서 앞 대상과 중복되는 목적어는 반드시 생략
- 시제: 현재, 과거, 현재완료진행 모두 적용 가능

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 카페에서 만나서 영어로 잡담하는 중입니다. 저는 주변 물건들을 가리키며 소개하고 싶어요.

규칙:
- 당신이 먼저 주변 물건에 대해 물어봐 주세요. (예: "Oh, is that a new phone?")
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 제가 답하면 어색한 부분은 자연스럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 화제를 유도해 주세요:
  · This is the + 대상 + that I've been using for the last ~
  · This is the + 대상 + that a friend gave me as a gift
  · This is the + 대상 + that I use/wear when I ~

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 관계절 패턴을, 제 주변 물건들로 연습하고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 지금 쓰고 있는 스마트폰은 얼마나 됐나요?
2) 지금 쓰는 가방이나 지갑은 어디서 났나요? (구매/선물)
3) 공부할 때나 일할 때 항상 쓰는 물건이 뭐예요?
4) 특별한 날이나 상황에만 쓰는 물건이 있나요?
5) 누군가에게 선물로 받은 물건 중 지금도 쓰는 게 있나요?

제 답변 5개를 받은 다음, 그것을 재료로 다음을 만들어 주세요:
- 대상 지목 표현 (완성 문장 전) 3개
- This is ~ that ~ 완성 문장 2개

각 영어 문장은 한국어 의역을 같이 붙여 주세요. 마지막에 5문장을 표로 정리해 주세요.

그리고 제가 댓글로 쓸 수 있는 관계절 예문 연습을 위해, 제 물건과 관련된 관계절 문장 3개를 추가로 만들어 주세요.
```
