# Lesson 020 — **boring vs. bored / confusing vs. confused / exciting vs. excited**: -ing형과 -ed형 구분하기 (Part 1)

> 📺 **원본 영상**: [[LESSON#19 - Part 1] -  과거분사형 / 동명사형 - bored/boring](https://www.youtube.com/watch?v=tgAnPN_mf-k)
> ⏱ 15분 01초 · 👁 460K회

> 🎯 **오늘의 핵심 포인트**
> 1. **`I'm difficult` → 틀림!** 어떤 것/상황을 묘사하는 형용사는 사람 주어에 쓰면 의미가 달라진다
> 2. **`-ing형`** → 어떤 것/상황을 묘사: `It's boring` (그게 지루해), `It's exciting` (그게 흥미진진해)
> 3. **`-ed형`** → 내 감정을 묘사: `I'm bored` (나 심심해), `I'm confused` (나 잘 모르겠어/혼란스러워)
> 4. **`I'm excited`** → 너무 신난다 / 너무 기대돼 (흥분이 아님!)
> 5. **나한테 어렵다** → `It's hard for me` / `It's too hard for me`

> 💡 강사의 한마디:
> "사전에 나오는 뜻풀이를 그대로 회화로 옮겨서 쓸 생각을 하면 안 돼요. 실제 쓰임을 통해 감을 익히는 게 중요해요."

---

## ✨ 상황 1 — 나한테 어렵다 (I'm hard ❌)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | I'm difficult. | (이렇게 하면 "내가 어려운 사람"이 됨) | ❌ 이렇게 하면 안 됨 |
| 2 | 🆕 **It's hard for me.** | **나한테 너무 어려워요.** | **← 오늘 #1 (for me로 나한테 적용)** |
| 3 | 🆕 **It's too hard for me.** | **나한테 너무 힘들어요.** | **← 오늘 #1** |
| 4 | 🆕 **It's too difficult for me.** | **나한테 너무 어려워요.** | **← 오늘 #1** |

---

## ✨ 상황 2 — boring vs. bored (지루하다)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **The movie was boring.** | **영화 재미없었어. (그 영화가 지루했어.)** | **← 오늘 #2 (-ing: 영화가 지루하게 만듦)** |
| 2 | 🆕 **It was boring.** | **재미없었어. 지루했어.** | **← 오늘 #2** |
| 3 | 🆕 **I'm bored.** | **나 심심해.** | **← 오늘 #3 (-ed: 내 감정)** |

---

## ✨ 상황 3 — confusing vs. confused (헷갈리다)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **It's confusing.** | **이거 헷갈려.** | **← 오늘 #2 (-ing: 그것이 헷갈리게 만듦)** |
| 2 | 🆕 **This is so confusing.** | **이거 너무 헷갈려.** | **← 오늘 #2** |
| 3 | 🆕 **That's really confusing.** | **그거 진짜 헷갈린다.** | **← 오늘 #2** |
| 4 | 🆕 **I'm confused.** | **나 잘 모르겠어. (혼란스러워.)** | **← 오늘 #3 (-ed: 내 감정)** |
| 5 | 🆕 **I'm so confused.** | **뭐가 뭔지 잘 모르겠어.** | **← 오늘 #3** |

---

## ✨ 상황 4 — exciting vs. excited (신나다 / 기대되다)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I'm really excited.** | **너무 신난다. / 너무 기대돼.** | **← 오늘 #4 (-ed: 내 감정, "흥분"이 아님!)** |
| 2 | 🆕 **I'm so excited.** | **너무 신난다. / 너무 기대돼.** | **← 오늘 #4** |
| 3 | 🆕 **I'm so excited about the trip.** | **여행이 너무 기대돼요.** | **← 오늘 #4 (about + 대상)** |
| 4 | 🆕 **I'm so excited about the trip next month.** | **다음 달 여행이 너무 기대돼요.** | **← 오늘 #4** |
| 5 | 🆕 **It's exciting.** | **그거 진짜 흥미진진해. / 너무 신나는 거야.** | **← 오늘 #2 (-ing: 상황 묘사, "기대"의 의미는 없어짐)** |

---

## ✨ 상황 5 — interesting vs. interested (재미있다 / 신기하다 / 관심 있다)

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **I'm very interested in this.** | **이거에 관심이 많아요.** | **← 오늘 #3 (-ed: 내 감정, in + 대상)** |
| 2 | 🆕 **That's really interesting.** | **그거 되게 재밌다. / 되게 신기하다.** | **← 오늘 #2 (-ing: 상황·것을 묘사, 호기심 자극하는 재미)** |
| 3 | 🆕 **That's very interesting.** | **그거 되게 흥미롭다. / 신기하다.** | **← 오늘 #2** |

---

## 🔑 오늘의 핵심 대조

```
[-ing형 = 어떤 것/상황을 묘사 "~하게 만드는"]
It's boring.          그게 지루해. (지루하게 만드는)
It's confusing.       이거 헷갈려. (헷갈리게 만드는)
It's exciting.        그거 흥미진진해. (흥분하게 만드는)
It's interesting.     그거 재밌어/신기해. (호기심을 자극하는)

[-ed형 = 내 감정을 묘사 "내가 ~된 상태"]
I'm bored.            나 심심해.
I'm confused.         나 잘 모르겠어. / 혼란스러워.
I'm excited.          너무 신난다. / 너무 기대돼.  ← "흥분"이 아님!
I'm interested in ~.  ~에 관심이 있어.

[나한테 어렵다]
I'm hard/difficult.  ❌ (내가 어려운 사람이 됨)
It's hard for me.    ✅ (나한테 어렵다)

[How was 질문]
How was the movie?       영화 어땠어?
→ It was boring.         재미없었어.
→ It was so exciting.    너무 흥미진진했어.
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **boring** | adj. | 지루하게 만드는, 재미없는 | `I'm boring` = 내가 지루한 사람 (주의!) |
| **bored** | adj. | 심심한 (내 감정) | `I'm bored` = 나 심심해 |
| **confusing** | adj. | 헷갈리게 만드는 | `it's confusing` |
| **confused** | adj. | 혼란스러운 (내 감정) | `I'm confused` = 잘 모르겠어 |
| **exciting** | adj. | 흥미진진한, 신나게 만드는 | `it's exciting` (기대의 의미는 없음) |
| **excited** | adj. | 신이 난, 기대되는 (내 감정) | `I'm excited` = 신난다/기대돼 |
| **interesting** | adj. | 흥미로운, 신기한 (호기심 자극) | `fun`보다 깊이 있는 재미 |
| **interested** | adj. | 관심이 있는 | `I'm interested in ~` |

## 📝 파생 연습 (5문제)

1. **빈칸 채우기**: "영화 어땠어? — 재미없었어." → `How was the movie? — It was _____.`
2. **빈칸 채우기**: "나 심심해." → `I'm _____.`
3. **골라 쓰기**: `boring / bored` 중 하나 — "이 책은 너무 ______다."
4. **영작**: "다음 주 콘서트가 너무 기대돼요." (`I'm excited about ~` 사용)
5. **영작**: "이거 진짜 헷갈리는데, 잘 모르겠어." (두 문장으로: `It's ~ / I'm ~`)

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 5가지를 1줄씩으로 다시 설명해 주세요.
2) 제가 가장 헷갈릴 만한 포인트 1가지를 골라 추가 예문 3개로 보충 설명해 주세요.
3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 발음 팁이 있다면 한국어로 안내해 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: -ing형 vs -ed형 형용사 구분 (boring/bored, confusing/confused, exciting/excited, interesting/interested)
- 오늘의 핵심:
  1. I'm difficult/hard는 틀림 → It's hard for me로 표현
  2. -ing형: 어떤 것/상황을 묘사 (It's boring = 그게 지루해)
  3. -ed형: 내 감정을 묘사 (I'm bored = 나 심심해)
  4. I'm excited = 신난다/기대돼 ("흥분"이 아님!)
  5. interesting = 호기심 자극하는 재미, 신기하다 (fun과 다름)
- 핵심 예문:
  · It's hard for me. / It's too hard for me. (나한테 너무 어려워요)
  · The movie was boring. / It was boring. (영화 재미없었어)
  · I'm bored. (나 심심해)
  · This is so confusing. (이거 너무 헷갈려) / I'm so confused. (잘 모르겠어)
  · I'm so excited about the trip. (여행이 너무 기대돼요)
  · It's exciting. (흥미진진해) vs I'm excited. (신난다/기대돼)
  · That's really interesting. (되게 재밌다/신기하다)
  · I'm very interested in this. (이거에 관심이 많아요)
- 핵심 어휘: boring/bored, confusing/confused, exciting/excited, interesting/interested
```

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 제가 영어로 답하면 채점 (✅ / ⚠️ / ❌).
3) 총 10문제.
4) 마지막에 자주 틀린 패턴 복습 노트 정리.

[레슨 핵심 패턴]
- It's + -ing형 (상황 묘사): boring, confusing, exciting, interesting
- I'm + -ed형 (내 감정): bored, confused, excited, interested in ~
- It's hard for me. / It's too difficult for me. (나한테 어렵다)
- How was ~? → It was boring/exciting/interesting.
- I'm excited about ~ (~ 기대돼요)
- I'm interested in ~ (~에 관심 있어요)

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 최근에 본 영화, 읽은 책, 새로운 경험에 대해 이야기하고 있습니다.

규칙:
- 첫 인사는 당신이 먼저. ("Have you watched anything good lately?" 같은 질문)
- 매 턴마다 영어 문장 1~2개 + (괄호 안에 한국어 의역).
- 어색한 부분은 부드럽게 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 화제를 유도해 주세요:
  · How was the movie?
  · It was boring / It was so exciting
  · I'm so excited about ~
  · This is really confusing
  · I'm very interested in ~
  · That's really interesting

10턴쯤 대화한 뒤 "오늘 잘 쓴 표현 / 더 연습할 표현"을 한국어로 짧게 정리해 주세요.

자, 인사부터 시작해 주세요.
```

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 패턴을, 제 일상 상황으로 옮겨서 영어 문장을 만들고 싶어요.

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:
1) 최근에 본 영화, 드라마, 책 — 어땠어요?
2) 요즘 심심하거나 지루한 상황이 있나요?
3) 헷갈리거나 잘 모르겠는 것이 있나요? (영어, 상황 등)
4) 요즘 기대되는 것이 있나요?
5) 관심이 많은 분야나 취미가 있나요?

제 답변을 받은 다음, 오늘 배운 -ing/-ed 패턴으로 각각 영어 문장을 만들어 주세요.
한국어 의역과 함께 마지막에 5문장을 표로 정리해 주세요.
```
