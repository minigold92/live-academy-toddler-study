# Lesson 070 — 소유격 회화: **mine / whose / someone else's**

> 📺 **원본 영상**: [[ 기초영어회화 ] 소유격 연습](https://www.youtube.com/watch?v=O1YcgSj1abk)
> ⏱ 9분 24초 · 👁 95K회

> 🎯 **오늘의 핵심 포인트**
> 1. **Did you forget something?** = 뭐 잊었어요? (돌아온 사람에게)
> 2. **I think I left my mask here.** = 제 마스크를 여기에 두고 간 것 같아요 (`left` = leave 과거형)
> 3. **Is this it?** = 이거예요? (앞서 언급한 것을 `it`으로 지칭)
> 4. **mine** = 내 것 (소유대명사 — `my mask`를 간결하게 대체)
> 5. **Whose is that?** = 그건 누구 거예요? (`whose` = who의 소유격 의문사)
> 6. **someone else's** = 다른 사람 것 (`I guess it's someone else's.`)

---

## ✨ 상황 1 — 두고 간 마스크 찾기 대화

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | 🆕 **Did you forget something?** | **뭐 잊었어요? (뭐 두고 갔어요?)** | **← 오늘 #1** |
| 2 | 🆕 **Yeah, I think I left my mask here.** | **네, 제 마스크를 여기에 두고 간 것 같아요.** | **← 오늘 #2 (left = leave 과거)** |
| 3 | 🆕 **Wait a minute, is this it?** | **잠깐만요, 이거예요?** | **← 오늘 #3 (it = 앞서 말한 마스크)** |
| 4 | 🆕 **No, mine is pink.** | **아니요, 제 거는 핑크색이에요.** | **← 오늘 #4 (mine = 내 것)** |
| 5 | 🆕 **Whose is that?** | **그건 누구 거예요?** | **← 오늘 #5 (whose)** |
| 6 | 🆕 **I guess it's someone else's.** | **다른 사람 건가 봐요.** | **← 오늘 #6 (someone else's)** |

### 🧩 문맥으로 이어 말하기 (전체 대화)
> **A: Did you forget something?**
> **B: Yeah, I think I left my mask here.**
> **A: Oh wait, is this it?**
> **B: No, mine is pink. Whose is that?**
> **A: I guess it's someone else's.**

---

## 🔑 오늘의 소유격 패턴 정리

```
[소유격 → 소유대명사]
my mask      →  mine     (내 마스크 → 내 거)
your mask    →  yours    (네 마스크 → 네 거)
his/her mask →  his/hers (그/그녀 마스크 → 그/그녀 거)
Larry's mask →  Larry's  (Larry의 마스크 → Larry 거)

[~의 것 표현]
This is Larry's mask.  →  This is Larry's.  (소유격 + mask 생략 가능)
I guess it's someone else's mask.  →  I guess it's someone else's.

[whose = 누구의 것]
Whose is that?    (그건 누구 거예요?)
※ 철자: W-H-O-S-E / 발음: "후즈"

[is this it?]
it = 앞서 언급된 물건을 가리킴
Is this it? = Is this your mask? (더 간결한 표현)
```

---

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **left** | v. | 두고 갔다 (leave 과거형) | `I left my mask here` |
| **mine** | 소유대명사 | 내 것 | `my mask` → `mine` |
| **whose** | 의문사 | 누구의 것 | 철자 `whose`, 발음 "후즈" |
| **someone else's** | 표현 | 다른 사람 것 | `someone else` + 소유격 `'s` |
| **I guess** | 표현 | ~인가 봐요, ~인 것 같아요 | 추측을 부드럽게 표현 |
| **wait a minute** | 표현 | 잠깐만요 | `oh wait`도 동일하게 사용 |

---

## 🗣️ 발음 팁

1. **I think I left** — `left`의 끝 `t`는 약화. "아이 씽크 아이 레프" 정도.
2. **mine is pink** — 빠르게 말하면 "마인이즈 핑크"처럼 연음.
3. **someone else's** — "썸원 엘시즈"처럼. `else's`의 `s` 두 개를 구별해서.

---

## 📝 파생 연습 (5문제)

1. **영작**: "제 지갑을 여기에 두고 간 것 같아요." (wallet)
2. **영작**: "아니요, 제 거는 검은색이에요." (mine / black)
3. **영작**: "그건 누구 거예요?" (whose)
4. **영작**: "다른 사람 건가 봐요." (I guess / someone else's)
5. **변환**: `my bag` → 소유대명사로, `Larry's bag` → `Larry's` 단독으로.

### 모드 1 — 튜터 모드 (개념 점검 + 막히는 부분 질문)

```
당신은 제 영어 회화 튜터입니다. 아래 레슨 자료를 읽고, 저와 한국어로 대화하면서 다음 순서로 진행해 주세요.

1) 레슨의 핵심 포인트 6가지를 1줄씩으로 다시 설명해 주세요.

3) 마지막에 "지금 가장 궁금하거나 막히는 부분이 뭐예요?"라고 물어봐 주세요.

규칙:
- 영어 예문은 반드시 한국어 의미와 같이 보여 주세요.
- 제가 영작을 시도하면 즉시 문법/뉘앙스 첨삭해 주세요.

[레슨 자료]
- 주제: 소유격 회화 — mine / whose / someone else's
- 오늘의 핵심:
  1. Did you forget something? = 뭐 잊었어요?
  2. I think I left my mask here. (left = leave 과거형)
  3. Is this it? = 이거예요? (it = 앞서 언급한 물건)
  4. mine = 내 것 (my mask의 소유대명사)
  5. Whose is that? = 그건 누구 거예요? (whose 철자 주의)
  6. I guess it's someone else's. = 다른 사람 건가 봐요.
- 핵심 예문 (전체 대화):

  · Wait a minute, is this it?
  · No, mine is pink. Whose is that?

- 핵심 어휘: left(leave 과거), mine, whose, someone else's, I guess, wait a minute

### 모드 2 — 퀴즈/드릴 모드 (한→영 작문 반복 연습)

```
당신은 제 영어 회화 드릴 코치입니다. 아래 레슨 자료에 기반해서, 한국어 → 영어 작문 퀴즈를 진행해 주세요.

진행 방식:
1) 한 번에 한국어 문장 1개만 제시.
2) 채점 (✅ / ⚠️ / ❌) 후 짧게 설명.
3) 총 10문제, 마지막에 복습 노트.

[레슨 핵심 패턴]
- 깜빡함 확인: Did you forget something?
- 두고 옴: I think I left my + 물건 + here.
- 확인: Is this it? / Is this your ~?
- 소유대명사: mine / yours / his / hers
- 누구 것: Whose is that/this?
- 다른 사람 것: I guess it's someone else's.

자, 첫 문제 주세요.
```

### 모드 3 — 회화 롤플레이 (오늘 표현으로 진짜 대화)

```
당신은 제 미국인 친구입니다. 우리는 카페에서 만났다가 헤어지는 상황입니다.

규칙:
- 대화 상황: 내가 뭔가를 두고 갔다가 돌아온 상황으로 시작.
- 매 턴마다 영어 1~2문장 + (괄호 한국어 의역).
- 어색한 부분은 "Oh, you mean ~ ?" 방식으로 교정.
- 대화 속에서 아래 표현이 자연스럽게 등장하도록 유도:
  · Did you forget something?

  · Is this it? / Wait a minute
  · No, mine is ~
  · Whose is that?
  · I guess it's someone else's.

10턴 후 정리해 주세요.

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```
저는 한국어 기초 영어 학습자입니다. 오늘 배운 소유격 표현을 제 일상 상황으로 연습하고 싶어요.

먼저 저에게 한국어로 다음을 물어봐 주세요:
1) 최근에 어딘가에 물건을 두고 온 경험이 있는지 (무엇을)
2) 누군가의 물건을 발견했을 때 어떻게 물어봤는지
3) 내 물건이 아니라는 걸 설명할 때 (색이나 모양)

답변을 받은 뒤:
- I think I left my ~ 문장
- Is this it? / Is this your ~? 문장
- No, mine is ~ 문장
- Whose is that? + I guess it's someone else's. 문장

각 문장에 한국어 의역을 붙여 표로 정리해 주세요.
```
