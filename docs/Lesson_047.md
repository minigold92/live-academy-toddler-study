# Lesson 047 — "원래 이런 거야?" — It's supposed to be like this / It's always like this

> 📺 **원본 영상**: [[ 기초영어회화 ] "이거 원래 이래?" , "응. 이거 원래 이래." - PART 1](https://www.youtube.com/watch?v=yK_MuPCqc5w)
> ⏱ 16분 27초 · 👁 777K회

> 🎯 **오늘의 핵심 포인트**
>
> 1. "원래 이래" (정상이다, 의도된 것) → **`It's supposed to be like this`**
> 2. "원래 이래" (항상 그렇다, 성향/상황) → **`It's always like this`**
> 3. 의문형: **`Is it supposed to be like this?`** / **`Is it always like this?`**
> 4. 사람 성향에는 `supposed to` 대신 **`always`** 사용 (`He's always like that`)
> 5. 과거부터 쭉 그랬다고 강조: **`It's always been like this`** (현재완료)

---

## ✨ 상황 1 — 숙제 검사: 외식 / 주말 계획 대화

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | Hey, do you wanna go out for dinner today? | 우리 오늘 저녁 나가서 먹을까? | 복습 (숙제 확인) |
| 2 | That sounds great! Do you have something in mind? | 좋아! 뭐 생각해 둔 거 있어? | 복습 |
| 3 | I don't really have anything in mind. What do you want? | 딱히 생각해 둔 건 없는데. 넌 뭐 먹고 싶어? | 복습 |
| 4 | How about pizza? | 피자 어때? | 복습 |
| 5 | Hey, do you wanna go somewhere this weekend? | 우리 주말에 어디 좀 갈까? | 복습 |
| 6 | I have a few places in mind. | 생각해 둔 곳이 몇 군데 있는데. | 복습 |
| 7 | Where did you have in mind? | 생각해 둔 데가 어딘데? | 복습 |

---

## ✨ 상황 2 — "원래 이래" (정상, 의도된 것): be supposed to

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | it's like this | 이것은 이렇다 | 도입 (기초 구조) |
| 2 | 🆕 **It's supposed to be like this.** | **이거 원래 이런 거야.** | **← 오늘 #1 (정상이다)** |
| 3 | 🆕 **It's supposed to be like that.** | **저거 원래 저런 거야.** | **← 오늘 #1** |
| 4 | 🆕 **Is it supposed to be like this?** | **이거 원래 이런 거야? (의문형)** | **← 오늘 #1 의문형** |
| 5 | Is this supposed to be like this? | 이거 원래 이런 거야? (this 주어) | 도입 (주어 변형) |

---

## ✨ 상황 3 — "원래 이래" (항상 그렇다, 성향/상황): always

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | Why is it so hot this summer? | 이번 여름에는 왜 이렇게 더워? | 도입 (대화 예시) |
| 2 | 🆕 **It's always like this in the summer.** | **여름에는 원래 이렇잖아.** | **← 오늘 #2 (항상 그렇다)** |
| 3 | 🆕 **It's always like this.** | **원래 이래, 항상 이래.** | **← 오늘 #2 핵심 패턴** |
| 4 | 🆕 **It's always been like this.** | **이거 항상 이랬어, 원래부터 이랬어.** | **← 오늘 #2 (현재완료, 과거부터 쭉)** |

---

## ✨ 상황 4 — 사람의 성향: "쟤 원래 저래?"

| | 영어 | 한국어 | 구분 |
|---|---|---|---|
| 1 | it feels like we're living in a tropical country these days. | 요즘은 열대 국가에서 살고 있는 기분이야. | 도입 (feels like) |
| 2 | it's like we're living in a tropical country these days. | 요즘은 열대 국가에서 살고 있는 것 같아. | 도입 (it's like) |
| 3 | 🆕 **Is he always like that?** | **쟤는 항상 저러니? / 쟤 원래 저래?** | **← 오늘 #3 (사람 성향)** |
| 4 | 🆕 **He's always like that.** | **쟤는 항상 저래. / 쟤 원래 저래.** | **← 오늘 #3** |

## 🔑 오늘의 변환 패턴 정리

```
["원래 이래" 두 가지 패턴]
① 정상이다, 의도된 것:
   It's supposed to be like this.    (이거 원래 이런 거야)
   Is it supposed to be like this?   (이거 원래 이런 거야? — 의문형)

② 항상 그렇다, 성향/상황:
   It's always like this.            (원래 이래, 항상 이래)
   It's always like this in the summer.  (여름에는 원래 이래)
   It's always been like this.       (원래부터 항상 이랬어 — 현재완료)

[사람 성향은 always만]
   Is he always like that?    (쟤 항상 저러니? / 쟤 원래 저래?)
   He's always like that.     (쟤 항상 저래)
   ※ be supposed to는 사람 성향에 쓰지 않음

[구조 분석]
   it is like this  +  be supposed to  →  it is supposed to be like this
   (is가 be supposed to 안으로, 원래 is는 to 뒤 be로)
```

## 📖 핵심 어휘 (오늘 처음 나온 것만)

| 단어/표현 | 품사 | 뜻 | 메모 |
|---|---|---|---|
| **be supposed to** | v.구 | 원래 ~하게 되어 있다, ~가 정상이다 | 기준/규정/의도에 따른 정상 상태 |
| **It's always like this** | 표현 | 원래 이래, 항상 이래 | 성향·반복 상황 |
| **It's always been like this** | 표현 | 원래부터 항상 이랬어 | 현재완료로 과거부터 지속 강조 |
| **have something in mind** | v.구 | 생각해 둔 것이 있다 | `Do you have something in mind?` |
| **go out for dinner** | v.구 | 저녁 먹으러 나가다, 외식하다 | `go out for breakfast/lunch/dinner` |
| **a few places** | 표현 | 몇 군데, 몇 곳 | `I have a few places in mind` |
| **tropical** | adj. | 열대의 | `a tropical country` |

## 📝 파생 연습 (5문제)

1. **영작**: "이게 원래 이렇게 작동하는 거야?" (`be supposed to` 사용)
2. **선택**: 사람이 항상 늦는 성향에 대해 "쟤 원래 이렇게 늦어?" — `supposed to` vs `always` 중 어느 쪽?
3. **영작**: "여기 원래 이렇게 추워. 겨울에는 항상 이래."
4. **변환**: `It's always like this.` → 과거부터 쭉 그랬다는 걸 강조하는 형태로.
5. **영작**: "쟤 항상 저러니? / 어, 쟤 원래 저래."

<details>
<summary>정답 보기</summary>

1. `Is it supposed to work like this?`
2. `always` — `Is he always like this?` (성향에는 `supposed to` 부적절)
3. `It's always like this here. It's always like this in the winter.`
4. `It's always been like this.`
5. `Is she always like that?` / `Yeah, she's always like that.`
</details>

### 모드 4 — 응용 확장 (내 일상 단어로 같은 패턴 만들기)

```

먼저 저에게 한국어로 다음 5가지를 물어봐 주세요:

- Is it supposed to be like this? 의문형 (1문장)

을 만들어 주세요. 각 문장은 한국어 의역과 함께 표로 정리해 주세요.
```
